# <a name="doclastprint-function"></a>Función DOCLASTPRINT

Devuelve la fecha y la hora en que se imprimió por última vez un documento, en forma de valor de serie.
  
## <a name="syntax"></a>Sintaxis

DOCLASTPRINT ()
  
## <a name="remarks"></a>Comentarios

Utilice la función FORMAT para dar formato al valor.

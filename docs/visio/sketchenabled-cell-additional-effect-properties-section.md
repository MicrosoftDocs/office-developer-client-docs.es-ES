# <a name="sketchenabled-cell-additional-effect-properties-section"></a>Celda SketchEnabled (sección Propiedades de efecto adicional)

Determina si se muestra un efecto de boceto en la forma o no, como un valor Boolean. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **SketchEnabled** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | SketchEnabled  <br/> |
   
Para obtener una referencia desde un programa a la celda **SketchEnabled** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visSketchEnabled** <br/> |

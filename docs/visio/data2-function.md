# <a name="data2-function"></a>Función DATA2

Devuelve el texto del cuadro Datos  **2** del cuadro de  diálogo Nombre de forma de una forma (en la ficha Programador, haga clic en Nombre de forma **)** como una cadena. 
  
## <a name="syntax"></a>Sintaxis

DATA2 ()
  
## <a name="remarks"></a>Comentarios

Si el cuadro está vacío, la función devuelve una cadena vacía.

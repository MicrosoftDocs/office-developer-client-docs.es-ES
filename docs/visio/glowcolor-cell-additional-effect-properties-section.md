# <a name="glowcolor-cell-additional-effect-properties-section"></a>Celda GlowColor (sección Propiedades del efecto adicional)

Determina el color utilizado para el trazo del resplandor externo aplicado a una forma, como un valor RGB o tema.
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **GlowColor** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | GlowColor  <br/> |
   
Para obtener una referencia a la celda **GlowColor** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visGlowColor** <br/> |

# <a name="glowcolor-cell-additional-effect-properties-section"></a>Celda GlowColor (sección Propiedades de efecto adicional)

Determina el color usado para el trazo del resplandor externo aplicado a una forma, como un valor RGB o de tema.
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la **celda GlowColor** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | GlowColor  <br/> |
   
Para obtener una referencia desde un programa a **la celda GlowColor** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visGlowColor** <br/> |

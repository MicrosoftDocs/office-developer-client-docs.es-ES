# <a name="sin-function-visioshapesheet"></a>SIN Function (VisioShapeSheet)

Devuelve el seno de un ángulo.
    
 
  
## <a name="syntax"></a>Sintaxis

SENO (** *ángulo* **) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Obligatorio/opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _ángulo_ <br/> |Obligatorio  <br/> |**Numeric** <br/> |Ángulo del que obtener el seno.  <br/> |
   
### <a name="return-value"></a>Valor devuelto

Number
  
## <a name="remarks"></a>Comentarios

La función SIN utiliza radianes a menos que especifique una unidad de medida diferente en _ángulo_.
  
## <a name="example"></a>Ejemplo

SIN(45 grados) 
  
Devuelve 0,7071.

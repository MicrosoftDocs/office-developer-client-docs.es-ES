# <a name="sin-function-visioshapesheet"></a>Función SIN (VisioShapeSheet)

Devuelve el seno de un ángulo. 
  
## <a name="syntax"></a>Sintaxis

SIN (* * *Angle* * *) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Necesario/Opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _respecto_ <br/> |Obligatorio  <br/> |**Numeric** <br/> |Ángulo del que obtener el seno.  <br/> |
   
### <a name="return-value"></a>Valor devuelto

Número
  
## <a name="remarks"></a>Comentarios

La función SIN usa radianes a menos que especifique una unidad de medida diferente en el _ángulo_.
  
## <a name="example"></a>Ejemplo

SIN(45 grados) 
  
Devuelve 0,7071.

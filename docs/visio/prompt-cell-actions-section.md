# <a name="prompt-cell-actions-section"></a>Celda Prompt (sección Acciones)

Esta celda dejó de utilizarse a partir de Visio 2002.
  
## <a name="remarks"></a>Comentarios

En versiones anteriores de Visio, esta celda especificaba un mensaje descriptivo que aparecía en la barra de estado cuando se seleccionaba el comando contextual correspondiente.

# <a name="bevellightingangle-cell-bevel-properties-section"></a>Celda BevelLightingAngle (sección Propiedades de bisel)

Determina el ángulo de rayo en relación con el bisel en grados. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **BevelLightingAngle** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | BevelLightingAngle  <br/> |
   
Para obtener una referencia desde un programa a la celda **BevelLightingAngle** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowBevelProperties** <br/> |
| Índice de celda:  <br/> |**visBevelLightingAngle** <br/> |

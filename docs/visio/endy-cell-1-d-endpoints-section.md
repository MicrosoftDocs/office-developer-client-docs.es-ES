# <a name="endy-cell-1-d-endpoints-section"></a>Celda EndY (Sección de extremos 1D)

Representa la coordenada *y* del extremo de la forma 1D en relación con el origen de su forma principal. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda EndY por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | Single  <br/> |
   
Para obtener una referencia desde un programa a la celda EndY por su índice
, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowXForm1D** <br/> |
| Índice de celda:  <br/> |**vis1DEndY** <br/> |

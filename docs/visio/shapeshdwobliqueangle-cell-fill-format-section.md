# <a name="shapeshdwobliqueangle-cell-fill-format-section"></a>Celda ShapeShdwObliqueAngle (Sección de formato de relleno)

Especifica el ángulo de dirección oblicua de la sombra de una forma.
  
## <a name="remarks"></a>Comentarios

Un valor de cero (0) en esta celda indica que la dirección del ángulo es vertical hacia arriba y que se mide moviéndose en el sentido de las agujas del reloj.
  
Este valor corresponde al valor del parámetro **Dirección** del cuadro de diálogo **Sombra** (en la ficha **Inicio**, en el grupo **Forma**, haga clic en **Sombra** y, a continuación, en **Opciones de sombra**).
  
Para obtener una referencia a la celda ShapeShdwObliqueAngle por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | ShapeShdwObliqueAngle  <br/> |
   
Para obtener una referencia desde un programa a la celda ShapeShdwObliqueAngle por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowFill** <br/> |
| Índice de celda:  <br/> |**visFillShdwObliqueAngle** <br/> |

# <a name="quickstylefillmatrix-cell-quick-style-section"></a>Celda QuickStyleFillMatrix (sección Estilo rápido)

Determina el estilo de relleno estilo rápido que la forma hereda del tema activo, como un entero de 0-6. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleFillMatrix** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la propiedad **CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleFillMatrix  <br/> |
   
Para obtener una referencia desde un programa a la celda **QuickStyleFillMatrix** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleFillMatrix** <br/> |

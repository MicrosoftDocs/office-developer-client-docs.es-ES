# <a name="quickstyleeffectsmatrix-cell-quick-style-section"></a>Celda QuickStyleEffectsMatrix (sección estilos rápidos)

Determina los efectos de estilo rápido que la forma hereda del tema activo, como un entero del 0-6. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleEffectsMatrix** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleEffectsMatrix  <br/> |
   
Para obtener una referencia desde un programa a la celda **QuickStyleEffectsMatrix** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleEffectsMatrix** <br/> |

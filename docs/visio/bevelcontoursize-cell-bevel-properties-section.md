# <a name="bevelcontoursize-cell-bevel-properties-section"></a>Celda BevelContourSize (sección Propiedades de bisel)

Determina el tamaño del contorno del bisel en puntos. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **BevelContourSize** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | BevelContourSize  <br/> |
   
Para obtener una referencia desde un programa a la celda **BevelContourSize** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowBevelProperties** <br/> |
| Índice de celda:  <br/> |**visBevelContourSize** <br/> |

# <a name="reflectiondist-cell-additional-effect-properties-section"></a>Celda ReflectionDist (sección Propiedades de efecto adicional)

Determina la distancia que se desplaza un reflejo desde una forma, en puntos de 0,0 a 100,0. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la **celda ReflectionDist** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la propiedad **CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | ReflectionDist  <br/> |
   
Para obtener una referencia desde un programa a **la celda ReflectionDist** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visReflectionDist** <br/> |

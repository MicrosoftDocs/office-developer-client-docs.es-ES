# <a name="text-fields-section"></a>Sección de campos de texto

Presenta funciones y fórmulas insertadas en el texto de una forma con el cuadro de diálogo **Campo**. 
  
## <a name="remarks"></a>Comentarios

La única forma de agregar esta sección es insertar un campo en una forma mediante el cuadro de diálogo **Campo** (ficha **Insertar**). 
  
Cada campo de texto de un bloque de texto tiene una fila correspondiente en la ventana ShapeSheet. El nombre de la fila representa el número de caracteres que contiene el campo.

## <a name="remarks"></a><span data-ttu-id="7cc77-105">Observaciones</span><span class="sxs-lookup"><span data-stu-id="7cc77-105">Remarks</span></span>

<span data-ttu-id="7cc77-p101">La asignación de capas se muestra como un índice de la lista de capas de la página. El índice de las capas se corresponde con el orden de las capas en el cuadro de diálogo **Propiedades de las capas** (en la ficha **Inicio**, en el grupo **Edición**, haga clic en **Capas** y, a continuación, en **Propiedades de las capas **). The first name in the dialog box is layer 0, the second is layer 1, and so forth.</span></span>

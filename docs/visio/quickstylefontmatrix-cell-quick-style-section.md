# <a name="quickstylefontmatrix-cell-quick-style-section"></a>Celda QuickStyleFontMatrix (sección Estilos rápidos)

Determina el estilo de la fuente para cada estilo rápido, como un número entero de 1 a 6.
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleFontMatrix** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleFontMatrix  <br/> |
   
Para obtener una referencia a la celda **QuickStyleFontMatrix** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleFontMatrix** <br/> |

# <a name="custommenusfile-element-documentsettingstype-complextype-visio-xml"></a>Elemento CustomMenusFile (complexType DocumentSettings_Type) ("XML" de Visio)

Contiene el nombre del archivo de interfaz de usuario de Microsoft Visio (. VSU) que define los menús y los aceleradores personalizados para un documento.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[CustomMenusFile_Type](custommenusfile_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15. xsd  <br/> |
|**Elementos de documento** <br/> |Document. XML  <br/> |
   
## <a name="definition"></a>Definición

```XML
< xs:element name="CustomMenusFile" type="CustomMenusFile_Type" minOccurs="0" maxOccurs="1" >
</xs:element>
```

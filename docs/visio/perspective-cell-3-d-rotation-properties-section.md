# <a name="perspective-cell-3-d-rotation-properties-section"></a>Celda Perspective (sección de propiedades de giro 3D)

Determina el ángulo de perspectiva para una rotación en perspectiva, en grados (de 0 a 359,9).
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **Perspective** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |Perspective  <br/> |
   
Para obtener una referencia desde un programa a la celda **Perspective** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRow3DRotationProperties** <br/> |
|Índice de celda:  <br/> |**visPerspective** <br/> |

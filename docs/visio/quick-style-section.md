# <a name="quick-style-section"></a>Sección de estilo rápido

Muestra los valores actuales de formato de estilo rápido aplicados a la forma, incluidos el color y el estilo de línea, el estilo y color de relleno, el color de sombra, los efectos, la fuente y el tipo de estilo. 
  
## <a name="remarks"></a>Comentarios

Puede crear estilos rápidos para seleccionar la forma y, a continuación, seleccionar un nuevo estilo rápido (en la ficha **Inicio** , en el grupo **estilos de forma** , haga clic en **más**y, a continuación, seleccione otro estilo rápido de la galería).

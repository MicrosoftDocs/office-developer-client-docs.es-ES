# <a name="type-function"></a>Función TYPE

Devuelve el tipo interno de un objeto como un número. 
  
## <a name="syntax"></a>Sintaxis

TYPE ()
  
### <a name="return-value"></a>Valor devuelto

Número
  
## <a name="remarks"></a>Comentarios

La tabla siguiente enumera los tipos internos de objeto como números.

|Externo  <br/> |78  <br/> |

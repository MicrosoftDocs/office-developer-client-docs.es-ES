# <a name="type-function"></a>Función TYPE

Devuelve el tipo interno de un objeto como un número. 
  
## <a name="syntax"></a>Sintaxis

TYPE ()
  
### <a name="return-value"></a>Valor devuelto

Número
  
## <a name="remarks"></a>Comentarios

La tabla siguiente enumera los tipos internos de objeto como números.

|Page  <br/> |70  <br/> |
|Group  <br/> |71  <br/> |
|Forma  <br/> |72  <br/> |
|Style  <br/> |74  <br/> |
|Introducción  <br/> |77  <br/> |
|Foreign  <br/> |78  <br/> |
|Document  <br/> |79  <br/> |

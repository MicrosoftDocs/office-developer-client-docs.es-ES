# <a name="stencilgroup-element-windowtype-complextype-visio-xml"></a>Elemento de StencilGroup (Window_Type complexType) ('XML de Visio')

Especifica el grupo de windows de galería de símbolos combinadas de que la ventana es un miembro.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[StencilGroup_Type](stencilgroup_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Elementos de documento** <br/> |Windows.Xml  <br/> |
   
## <a name="definition"></a>Definición

```XML
< xs:element name="StencilGroup" type="StencilGroup_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

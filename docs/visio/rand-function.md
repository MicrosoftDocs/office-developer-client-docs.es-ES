# <a name="rand-function"></a>Función RAND

Devuelve un número aleatorio de punto flotante desde 0,0 a 1,0. Devuelve un n? mero diferente cada vez que se evalúa la función, que es una vez por minuto según el reloj del sistema. 
  
## <a name="syntax"></a>Sintaxis

RAND( )
  
### <a name="return-value"></a>Valor devuelto

Float
  
## <a name="remarks"></a>Comentarios

Esta función se puede usar para crear efectos de animación al asignar valores aleatorios a las propiedades de las formas.
  
## <a name="example"></a>Ejemplo

RAND( ) 
  
Devuelve un valor decimal, por ejemplo 0,3503.

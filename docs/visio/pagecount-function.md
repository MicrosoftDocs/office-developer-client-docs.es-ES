# <a name="pagecount-function"></a>Función PAGECOUNT

Devuelve el número de páginas de primer plano en un documento. 
  
## <a name="syntax"></a>Sintaxis

PAGECOUNT ()
  
### <a name="return-value"></a>Valor devuelto

Number

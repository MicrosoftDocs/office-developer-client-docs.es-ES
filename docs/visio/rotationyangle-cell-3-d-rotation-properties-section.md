# <a name="rotationyangle-cell-3-d-rotation-properties-section"></a>Celda RotationYAngle (sección de propiedades de giro 3D)

Determina el ángulo de rotación a lo largo del eje Y, en grados (0,0-359,9).
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **RotationYAngle** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |RotationYAngle  <br/> |
   
Para obtener una referencia desde un programa a la celda **RotationYAngle** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRow3DRotationProperties** <br/> |
|Índice de celda:  <br/> |**visRotationYAngle** <br/> |

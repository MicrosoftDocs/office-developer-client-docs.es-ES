# <a name="quickstylelinematrix-cell-quick-style-section"></a>Celda QuickStyleLineMatrix (sección Estilos rápidos)

Determina el estilo de línea de estilos rápidos que hereda de la forma, como un número entero de 0 a 6. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleLineMatrix** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleLineMatrix  <br/> |
   
Para obtener una referencia a la celda **QuickStyleLineMatrix** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleLineMatrix** <br/> |

# <a name="row-element-scratch-section-visio-xml"></a>Elemento Row (sección de borrador) ('XML de Visio')

Un área de trabajo para escribir y probar fórmulas a las que hacen referencia otras celdas.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[ScratchRow_Type](scratchrow_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Elementos de documento** <br/> |Document.XML, masters.xml, maestra # .xml, pages.xml, página # .xml  <br/> |
   
## <a name="definition"></a>Definición

```XML
< xs:element name="Row" type="ScratchRow_Type" >
</xs:element >
```

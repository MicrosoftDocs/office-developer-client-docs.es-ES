# <a name="reflectiontrans-cell-additional-effect-properties-section"></a>Celda ReflectionTrans (Sección de propiedades de efecto adicional)

Determina la transparencia del reflejo, como un porcentaje del 0 al 100%. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **ReflectionTrans** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** o desde un programa mediante la propiedad **CellsU,** utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | ReflectionTrans  <br/> |
   
Para obtener una referencia desde un programa a **la celda ReflectionTrans** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visReflectionTrans** <br/> |

# <a name="glowcolortrans-cell-additional-effect-properties-section"></a>Celda GlowColorTrans (sección Propiedades de efecto adicional)

Determina el nivel de transparencia del color usado para el trazo del resplandor de la forma, como porcentaje. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la **celda GlowColorTrans** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | GlowColorTrans  <br/> |
   
Para obtener una referencia desde un programa a **la celda GlowColorTrans** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visGlowColorTrans** <br/> |

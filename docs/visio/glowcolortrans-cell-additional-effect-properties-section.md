# <a name="glowcolortrans-cell-additional-effect-properties-section"></a>Celda GlowColorTrans (sección Propiedades del efecto adicional)

Determina el nivel de transparencia del color utilizado para el trazo de resplandor de la forma, como un porcentaje. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **GlowColorTrans** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | GlowColorTrans  <br/> |
   
Para obtener una referencia a la celda **GlowColorTrans** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visGlowColorTrans** <br/> |

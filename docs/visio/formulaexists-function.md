# <a name="formulaexists-function"></a>Función FORMULAEXISTS

Indica si la celda a la que se hace referencia contiene una fórmula. 
  
## <a name="syntax"></a>Sintaxis

FORMULAEXISTS (* * *referenciaDeCelda* * *) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Necesario/Opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _referenciaDeCelda_ <br/> |Obligatorio  <br/> |**String** <br/> |La celda en la que se debe comprobar si existe una fórmula.  <br/> |
   
## <a name="remarks"></a>Comentarios

La función FORMULAEXISTS devuelve 1 si la celda contiene una fórmula; Si no contiene una fórmula, devuelve cero (0).

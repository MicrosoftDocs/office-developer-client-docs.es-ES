# <a name="formulaexists-function"></a>Función FORMULAEXISTS

Indica si la celda referida contiene una fórmula. 
  
## <a name="syntax"></a>Sintaxis

FORMULAEXISTS (** *cellref* **) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Obligatorio/opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _CellRef_ <br/> |Obligatorio  <br/> |**String** <br/> |La celda en la que se debe comprobar si existe una fórmula.  <br/> |
   
## <a name="remarks"></a>Observaciones

La función FORMULAEXISTS devuelve 1 si la celda contiene una fórmula; Si no contiene una fórmula, devuelve cero (0).

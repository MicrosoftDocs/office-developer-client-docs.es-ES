# <a name="icon-element-mastershortcuttype-complextype-visio-xml"></a>Elemento Icon (complexType MasterShortcut_Type) ("XML" de Visio)

Especifica un icono binario codificado MIME (Extensiones multipropósito de correo Internet) (en formato. ico) para un elemento MasterShortcut de un documento.

</xs:element >

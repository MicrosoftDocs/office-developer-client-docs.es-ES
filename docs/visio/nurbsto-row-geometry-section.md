# <a name="nurbsto-row-geometry-section"></a>Fila NURBSTo (Sección de Geometría)

Contiene las coordenadas  *x*  - e  *y,*  posición del segundo al último nodo, posición del último grosor, posición del primer nodo, posición del primer grosor y fórmula para una spline B racional no uniforme (NURBS). 
  
Las filas NURBSTo contienen las celdas siguientes.
  
|**Cell**|**Descripción**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |Coordenada  *x*  del último punto de control de una NURBS.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |Coordenada  *y*  del último punto de control de una NURBS.  <br/> |
|[A](a-cell-geometry-section.md) <br/> |Penúltimo nodo de una NURBS.  <br/> |
|[B](b-cell-geometry-section.md) <br/> |Último grosor de una NURBS.  <br/> |
|[C](c-cell-geometry-section.md) <br/> |Primer nodo de una NURBS.  <br/> |
|[D](d-cell-geometry-section.md) <br/> |Primer grosor de una NURBS.  <br/> |
|[E](e-cell-geometry-section.md) <br/> |Fórmula de NURBS.  <br/> |
   
## <a name="remarks"></a>Comentarios

NURBS es una forma habitual de representar curvas matemáticamente. Puede crear una NURBS con la herramienta **Forma libre**.

# <a name="pi-function"></a>Función PI

Devuelve la constante matemática *PI* , que es aproximadamente 3.1415926535898. 
  
## <a name="syntax"></a>Sintaxis

PI( )
  
### <a name="return-value"></a>Valor devuelto

Número
  
## <a name="example"></a>Ejemplo

(14 in/2) ^ 2 \* PI () 
  
Devuelve 153,9380 pulgadas, el área de un círculo cuyo diámetro es de 14 pulgadas.

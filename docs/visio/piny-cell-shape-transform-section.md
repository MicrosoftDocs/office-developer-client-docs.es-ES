# <a name="piny-cell-shape-transform-section"></a>Celda PinY (sección Transformación de forma)

Representa la *y* -coordenadas del eje de la forma (centro de rotación) en relación con el origen de su forma principal. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda PinY por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | PinY  <br/> |
   
Para obtener una referencia desde un programa a la celda PinY por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowXFormOut** <br/> |
| Índice de celda:  <br/> |**visXFormPinY** <br/> |

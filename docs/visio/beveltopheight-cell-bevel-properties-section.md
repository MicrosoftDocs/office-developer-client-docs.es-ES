# <a name="beveltopheight-cell-bevel-properties-section"></a>Celda BevelTopHeight (sección Propiedades de bisel)

Determina el alto del bisel superior de una forma en puntos. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **BevelTopHeight** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | BevelTopHeight  <br/> |
   
Para obtener una referencia desde un programa a la celda **BevelTopHeight** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowBevelProperties** <br/> |
| Índice de celda:  <br/> |**visBevelTopHeight** <br/> |

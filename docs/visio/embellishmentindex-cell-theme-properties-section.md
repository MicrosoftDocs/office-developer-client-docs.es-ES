# <a name="embellishmentindex-cell-theme-properties-section"></a>Celda EmbellishmentIndex (sección Propiedades del tema)

Cambia el aspecto (adorno) de las llamadas, los contenedores, las escalas de tiempo y las formas del gráfico de organización.
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **EmbellishmentIndex** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la propiedad **CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | EmbellishmentIndex  <br/> |
   
Para obtener una referencia desde un programa a la celda **EmbellishmentIndex** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowThemeProperties** <br/> |
| Índice de celda:  <br/> |**visEmbellishmentIndex** <br/> |

# <a name="sheetref-function"></a>SHEETREF (función)

Devuelve una referencia a la hoja (forma) que se especifica en _sheetref_, o bien, si no hay ningún calificador de _sheetref_ , a la hoja actual. 
  
## <a name="version-information"></a>Información de versión

Versión agregada: Visio 2010 
  
## <a name="syntax"></a>Sintaxis

 *\<sheetref\>!*  SHEETREF() 
  
### <a name="return-value"></a>Valor devuelto

Referencia de ShapeSheet
  
## <a name="remarks"></a>Observaciones

Puede usar esta función en otras funciones que toman un símbolo (token) de hoja de referencia.

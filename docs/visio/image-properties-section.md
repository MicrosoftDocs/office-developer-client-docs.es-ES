# <a name="image-properties-section"></a>Sección Propiedades de imagen

Contiene los valores de gamma, brillo, contraste, desenfoque, nitidez y ruido para mapas de bits (una imagen que se compone de píxeles) y aparece en la ventana ShapeSheet sólo para dichos objetos.

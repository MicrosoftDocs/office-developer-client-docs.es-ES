# <a name="doclastsave-function"></a>Función DOCLASTSAVE

Devuelve la fecha y hora en que se guardó por última vez un documento, como un valor en serie.
  
## <a name="syntax"></a>Sintaxis

DOCLASTSAVE ()
  
## <a name="remarks"></a>Comentarios

Utilice la función FORMAT para dar formato al valor.

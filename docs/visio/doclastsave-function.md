# <a name="doclastsave-function"></a>Función DOCLASTSAVE

Devuelve la fecha y la hora de la última vez que se guardó un documento, como un valor de serie.
  
## <a name="syntax"></a>Sintaxis

DOCLASTSAVE ()
  
## <a name="remarks"></a>Comentarios

Utilice la función FORMAT para dar formato al valor.

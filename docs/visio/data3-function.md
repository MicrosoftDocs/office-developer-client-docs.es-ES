# <a name="data3-function"></a>Función DATA3

Devuelve el texto del cuadro de **datos 3** en el cuadro de diálogo **nombre de forma** de una forma (en la ficha **programador** , haga clic en nombre de **forma** ), en forma de cadena. 
  
## <a name="syntax"></a>Sintaxis

DATA3 ()
  
## <a name="remarks"></a>Comentarios

Si el cuadro está vacío, la función devuelve una cadena vacía.

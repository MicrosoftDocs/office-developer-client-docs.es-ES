# <a name="data3-function"></a>Función DATA3

Devuelve el texto del cuadro Datos  **3** del cuadro de  diálogo Nombre de forma de una forma (en la ficha Programador, haga clic en Nombre de forma **)** como una cadena. 
  
## <a name="syntax"></a>Sintaxis

DATA3 ()
  
## <a name="remarks"></a>Comentarios

Si el cuadro está vacío, la función devuelve una cadena vacía.

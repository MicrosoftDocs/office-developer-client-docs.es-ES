# <a name="data3-function"></a>Función DATA3

Devuelve el texto en el cuadro **dato 3** en el cuadro de diálogo **Nombre de la forma** de una forma (en la ficha **Programador** , haga clic en **Nombre de la forma** ), como una cadena. 
  
## <a name="syntax"></a>Sintaxis

DATA3 ()
  
## <a name="remarks"></a>Observaciones

Si el cuadro está vacío, la función devuelve una cadena vacía.

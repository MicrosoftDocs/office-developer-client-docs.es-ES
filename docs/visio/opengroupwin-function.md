# <a name="opengroupwin-function"></a>Función OPENGROUPWIN

Abre el grupo en la ventana de grupo. Si el objeto no es un grupo, esta función no tiene ningún efecto.
  
## <a name="syntax"></a>Sintaxis

OPENGROUPWIN( )
  
## <a name="remarks"></a>Observaciones

En versiones anteriores de Visio, esta función se denominaba _OPENGROUPWIN. La versión 4.0 de Visio y posteriores aceptan cualquiera de las dos denominaciones.

# <a name="is1d-function"></a>Función IS1D

Devuelve TRUE si la forma es 1D (unidimensional) y FALSE si la forma es 2D (bidimensional).
  
## <a name="version-information"></a>Información de versión

Versión añadida: Visio 2010
 
  
## <a name="syntax"></a>Sintaxis

IS1D ()
  
### <a name="return-value"></a>Valor devuelto

 **Boolean**
  
## <a name="remarks"></a>Observaciones

Si la hoja (forma) es una página o un documento, la función IS1D devuelve #REF!.

# <a name="enablefillprops-cell-style-properties-section"></a>Celda EnableFillProps (Sección de propiedades de estilo)

Determina si un estilo incluye propiedades de relleno.
  
|**Value**|**Descripción**|
|:-----|:-----|
|TRUE  <br/> |Incluir propiedades de relleno.  <br/> |
|FALSE  <br/> |Excluir propiedades de relleno.  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda EnableFillProps por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |EnableFillProps  <br/> |
   
Para obtener una referencia desde un programa a la celda EnableFillProps por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRowStyle** <br/> |
|Índice de celda:  <br/> |**visStyleIncludesFill** <br/> |

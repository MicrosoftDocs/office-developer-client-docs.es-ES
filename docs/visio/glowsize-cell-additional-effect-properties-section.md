# <a name="glowsize-cell-additional-effect-properties-section"></a>Celda GlowSize (sección Propiedades de efecto adicional)

Determina el tamaño del resplandor externo de una forma en puntos. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la **celda GlowSize** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | GlowSize  <br/> |
   
Para obtener una referencia desde un programa a **la celda GlowSize** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice de celda:  <br/> |**visGlowSize** <br/> |

# <a name="fonttoid-function"></a>Función FONTTOID

Devuelve el identificador (Id.) de la fuente especificada.
  
> [!NOTE]
> Se recomienda usar la función de la [fuente](font-function.md) , que es similar a la función **FONTTOID** . En Microsoft Visio 2013, se introdujo la función de la **fuente** que va a reemplazar **FONTTOID**. 
  
## <a name="syntax"></a>Sintaxis

FONTTOID (** *font_name* **) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Obligatorio/opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |Obligatorio  <br/> |**String** <br/> |Nombre de la fuente.  <br/> |

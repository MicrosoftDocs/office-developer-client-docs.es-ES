# <a name="bevelbottomwidth-cell-bevel-properties-section"></a>Celda BevelBottomWidth (sección de propiedades de bisel)

Determina el ancho del bisel inferior en puntos. 
  
## <a name="remarks"></a>Notas

Para obtener una referencia a la celda **BevelBottomWidth** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | BevelBottomWidth  <br/> |
   
Para obtener una referencia a la celda **BevelBottomWidth** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowBevelProperties** <br/> |
| Índice de celda:  <br/> |**visBevelBottomWidth** <br/> |

# <a name="ln-function"></a>Función LN

Devuelve el logaritmo natural (neperiano) de un número. El número debe ser positivo o LN devuelve el valor de error #NUM.
  
## <a name="syntax"></a>Sintaxis

LN(** *number* ** ) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Necesario/Opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obligatorio  <br/> |**Numérico** <br/> | El número cuyo logaritmo natural desea encontrar.  <br/> |
   
### <a name="return-value"></a>Valor devuelto

Numérico
  
## <a name="example"></a>Ejemplo

LN(42) 
  
Devuelve 3,7377.

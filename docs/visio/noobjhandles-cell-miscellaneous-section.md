# <a name="noobjhandles-cell-miscellaneous-section"></a>Celda NoObjHandles (sección Varios)

Habilita y deshabilita la presentación de los controladores de selección para la forma seleccionada.
  
|**Valor**|**Descripción**|
|:-----|:-----|
| TRUE  <br/> | Los controladores de selección no se muestran cuando se selecciona una forma.  <br/> |
| FALSE  <br/> | Los controladores de selección se muestran cuando se selecciona una forma.  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda NoObjHandles por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | NoObjHandles  <br/> |
   
Para obtener una referencia a la celda NoObjHandles por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowMisc** <br/> |
| Índice de celda:  <br/> |**visNoObjHandles** <br/> |

# <a name="doccreation-function"></a>Función DOCCREATION

Devuelve la fecha y hora de que creación de un documento, como un valor de serie.
  
## <a name="syntax"></a>Sintaxis

DOCCREATION ()
  
## <a name="remarks"></a>Observaciones

Utilice la función FORMAT para dar formato al valor.

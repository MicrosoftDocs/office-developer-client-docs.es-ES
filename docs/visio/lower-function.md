# <a name="lower-function"></a>Función LOWER

Devuelve una cadena convertida a minúsculas.
  
## <a name="syntax"></a>Sintaxis

INFERIOR (** *expresión* **) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Obligatorio/opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Obligatorio  <br/> |**Varies** <br/> | Cadena, referencia de celda o expresión; el resultado se convierte en una cadena cuyos caracteres se convierten en minúsculas.  <br/> |
   
### <a name="return-value"></a>Valor devuelto

String
  
## <a name="remarks"></a>Observaciones

La conversión de mayúsculas y minúsculas depende de la configuración regional especificada en la configuración actual del usuario. 
  
## <a name="example"></a>Ejemplo

LOWER("MAYÚSCULAS y minúsculas") 
  
Devuelve "mayúsculas y minúsculas".

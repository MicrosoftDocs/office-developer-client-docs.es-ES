## <a name="remarks"></a><span data-ttu-id="c617b-116">Comentarios</span><span class="sxs-lookup"><span data-stu-id="c617b-116">Remarks</span></span>

<span data-ttu-id="c617b-117">Los valores de **la fila RelMoveTo** son equivalentes a los valores de una fila [MoveTo](moveto-row-geometry-section.md) que se multiplican por el ancho y el alto de la forma.</span><span class="sxs-lookup"><span data-stu-id="c617b-117">Values in the **RelMoveTo** row are equivalent to values in a [MoveTo](moveto-row-geometry-section.md) row that are multiplied by the width and height of the shape.</span></span> <span data-ttu-id="c617b-118">Por ejemplo: una fila **RelMoveTo** donde el valor de la celda **X** es "0" y el valor de la celda Y es "0,5" podría reemplazarse con la fila **MoveTo** donde el valor de la celda **X** es la fórmula "Width *0"* y la celda **Y** es la fórmula "Height 0.5".</span><span class="sxs-lookup"><span data-stu-id="c617b-118">For example: a **RelMoveTo** row where the value of the **X** cell is "0" and the value of the **Y** cell is "0.5" could be replaced with **MoveTo** row where the value of the **X** cell is the formula "Width *0" and the **Y** cell is the formula "Height* 0.5."</span></span> 
  
<span data-ttu-id="c617b-119">La **fila RelMoveTo** contiene las coordenadas  *x*  - e  *y*  del primer vértice de la forma si la fila MoveTo es la primera fila de la sección.</span><span class="sxs-lookup"><span data-stu-id="c617b-119">The **RelMoveTo** row contains the  *x*  - and  *y*  -coordinates of the first vertex for the shape if the MoveTo row is the first row in the section.</span></span> <span data-ttu-id="c617b-120">Normalmente, se trata del primer vértice que se colocó al dibujar la forma y no corresponde necesariamente con el punto inicial de una forma unidimensional (1D).</span><span class="sxs-lookup"><span data-stu-id="c617b-120">Usually this is the first vertex placed when the shape was drawn, and it does not necessarily correspond to the begin point of a 1-D shape.</span></span> 
  
<span data-ttu-id="c617b-121">Una **sección Geometry** debe comenzar por una fila **MoveTo** o **RelMoveTo,** pero también puede usar las filas **RelMoveTo** y **MoveTo** para representar un espacio en el trazado de una forma con relación al ancho y alto de la forma.</span><span class="sxs-lookup"><span data-stu-id="c617b-121">A **Geometry** section must begin with a **MoveTo** or a **RelMoveTo** row, but you can also use the **RelMoveTo** row and **MoveTo** row to represent a gap in the stroking of a shape's path relative to the shape's width and height.</span></span> <span data-ttu-id="c617b-122">Sin embargo, cuando la trayectoria se utiliza para definir el límite de una zona rellena, esta interrupción se interpreta como segmento de línea recta.</span><span class="sxs-lookup"><span data-stu-id="c617b-122">However, when the path is used to define the boundary of a filled region, this gap is interpreted as a straight line segment.</span></span> <span data-ttu-id="c617b-123">Para insertar dicho espacio, inserte una fila entre dos filas y cambie el tipo de fila a **RelMoveTo**.</span><span class="sxs-lookup"><span data-stu-id="c617b-123">To insert such a gap, insert a row between two rows and change the row type to **RelMoveTo**.</span></span> <span data-ttu-id="c617b-124">Si la **fila RelMoveTo** está entre dos filas, contiene las coordenadas  *x*  - e  *y del*  primer vértice de la línea después del salto en la ruta de acceso de la forma.</span><span class="sxs-lookup"><span data-stu-id="c617b-124">If the **RelMoveTo** row is between two rows, it contains the  *x*  - and  *y*  -coordinates of the first vertex of the line after the break in the shape's path.</span></span>

# <a name="stencilgrouppos-element-windowtype-complextype-visio-xml"></a>Elemento de StencilGroupPos (Window_Type complexType) ('XML de Visio')

Contiene un número entero que especifica la posición relativa de una galería de símbolos dentro de un grupo en una ventana.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[StencilGroupPos_Type](stencilgrouppos_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Elementos de documento** <br/> |Windows.Xml  <br/> |
   
## <a name="definition"></a>Definición

```XML
<xs:element name="StencilGroupPos" type="StencilGroupPos_Type" minOccurs="0" maxOccurs="1" >
</xs:element>
```

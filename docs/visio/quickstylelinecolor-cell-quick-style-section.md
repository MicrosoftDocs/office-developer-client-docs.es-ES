# <a name="quickstylelinecolor-cell-quick-style-section"></a>Celda QuickStyleLineColor (sección Estilos rápidos)

Determina el color del tema que usa la línea de una forma, como un número entero de 0 a 7.
  
|||
|:-----|:-----|
|Value  <br/> |Descripción  <br/> |
|0  <br/> |El color de línea de forma hereda el color del tema oscuro.  <br/> |
|1  <br/> |El color de línea de forma hereda el color del tema claro.  <br/> |
|2  <br/> |El color de línea de forma hereda el color del tema Énfasis 1  <br/> |
|3  <br/> |El color de línea de forma hereda el color del tema Énfasis 2  <br/> |
|4  <br/> |El color de línea de forma hereda el color del tema Énfasis 3  <br/> |
|5  <br/> |El color de línea de forma hereda el color del tema Énfasis 4  <br/> |
|6  <br/> |El color de línea de forma hereda el color del tema Énfasis 5  <br/> |
|7  <br/> |El color de línea de forma hereda el color del tema Énfasis 6  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleLineColor** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleLineColor  <br/> |
   
Para obtener una referencia a la celda **QuickStyleLineColor** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleLineColor** <br/> |

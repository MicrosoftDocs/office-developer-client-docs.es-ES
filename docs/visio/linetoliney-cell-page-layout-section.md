# <a name="linetoliney-cell-page-layout-section"></a>Celda LineToLineY (sección Diseño de página)

Determina el gálibo vertical entre todos los conectores de la página de dibujo.
  
## <a name="remarks"></a>Observaciones

También puede establecer el valor de esta celda en el cuadro de diálogo **Espaciado del diseño y el enrutamiento**. (En la ficha **Diseño**, haga clic en la flecha de **Configurar página**, haga clic en **Diseño y enrutamiento** y, a continuación, en **Espaciado**).
  
Para obtener una referencia a la celda LineToLineY por su nombre desde otra fórmula, o desde un programa mediante la propiedad **CellsU**, utilice: 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |LineToLineY  <br/> |
   
Para obtener una referencia desde un programa a la celda LineToLineY por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRowPageLayout** <br/> |
|Índice de celda:  <br/> |**visPLOLineToLineY** <br/> |

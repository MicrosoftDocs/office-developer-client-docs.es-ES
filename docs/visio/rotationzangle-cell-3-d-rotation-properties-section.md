# <a name="rotationzangle-cell-3-d-rotation-properties-section"></a>Celda RotationZAngle (sección Propiedades de rotación 3D)

Determina el ángulo de giro a lo largo del eje Z, en grados (0,0 - 359,9).
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la **celda RotationZAngle** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |RotationZAngle  <br/> |
   
Para obtener una referencia desde un programa a **la celda RotationZAngle** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRow3DRotationProperties** <br/> |
|Índice de celda:  <br/> |**visRotationZAngle** <br/> |

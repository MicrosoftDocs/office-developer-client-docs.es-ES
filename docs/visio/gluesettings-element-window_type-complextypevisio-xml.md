# <a name="gluesettings-element-window_type-complextype-visio-xml"></a>Elemento GlueSettings (Window_Type complexType) (XML de Visio)

Especifica los objetos a los que se pegan las formas cuando se habilita el pegado en el documento.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[GlueSettings_Type](gluesettings_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Elementos de documento** <br/> |windows.xml, document.xml  <br/> |
   
## <a name="definition"></a>Definición

```XML
< xs:element name="GlueSettings" type="GlueSettings_Type" minOccurs="0" maxOccurs="1" >
</xs:element>
```

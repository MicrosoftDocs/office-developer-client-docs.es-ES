# <a name="gradient-stop-row-fill-gradient-section"></a>Fila de detención de degradado (sección Degradado de relleno)

Contiene el color, la transparencia y la posición de un detente degradado para un degradado de relleno.
  
Una **fila Detención** de degradado contiene las celdas siguientes. 
  
|**Cell**|**Descripción**|
|:-----|:-----|
|**Color** <br/> |Valor de color de la detención de degradado. Este valor se puede expresar como el número de índice de un color en la paleta de documentos o mediante las funciones [RGB,](rgb-function-visioshapesheet.md) [THEMEVAL](themeval-function.md)o [HSL](hsl-function.md) (por ejemplo).  <br/> |
|**ColorTrans** <br/> |La cantidad de transparencia del color degradado, como porcentaje.  <br/> |
|**Position** <br/> |La posición del degradado se detiene a lo largo de la dirección del degradado de relleno, como un porcentaje desde el punto de origen del degradado hasta el borde del área del relleno.  <br/> |

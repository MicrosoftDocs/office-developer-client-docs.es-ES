# <a name="gradient-stop-row-fill-gradient-section"></a>Fila Gradient Stop (sección Degradado de relleno)

Contiene el color, la transparencia y la posición de un punto de degradado para un degradado de relleno.
  
Una fila **Detener degradado** contiene las celdas siguientes. 
  
|**Cell**|**Descripción**|
|:-----|:-----|
|**Color** <br/> |El valor de color del punto de degradado. Este valor se puede expresar como el número de índice de un color en la paleta de documento o usar las funciones [RGB](rgb-function-visioshapesheet.md), [THEMEVAL](themeval-function.md)o [HSL](hsl-function.md) (por ejemplo).  <br/> |
|**ColorTrans** <br/> |El grado de transparencia del color de degradado, como un porcentaje.  <br/> |
|**Position** <br/> |La posición del punto de degradado a lo largo de la dirección del relleno degradado, como un porcentaje desde el punto de origen del degradado hasta el borde del área del relleno de.  <br/> |

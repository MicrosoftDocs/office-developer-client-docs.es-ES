# <a name="bevelmaterialtype-cell-bevel-properties-section"></a>Celda BevelMaterialType (sección Propiedades de bisel)

Determina el tipo de material que está formado por el bisel. 
  
|**Descripción**|**Value**|
|:-----|:-----|
|0  <br/> |No hay especial (en inglés)  <br/> |
|1  <br/> |Mate  <br/> |
|2  <br/> |Mate cálido  <br/> |
|3  <br/> |Plástico  <br/> |
|4  <br/> |Metal  <br/> |
|5  <br/> |Borde oscuro  <br/> |
|6  <br/> |Borde suave  <br/> |
|7  <br/> |Sin relieve  <br/> |
|8  <br/> |Alambre  <br/> |
|9  <br/> |Polvo  <br/> |
|10  <br/> |Polvo translúcido  <br/> |
|11  <br/> |Claro  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **BevelMaterialType** por su nombre desde otra fórmula, por el valor del atributo **N** de un elemento de **celda** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | BevelMaterialType  <br/> |
   
Para obtener una referencia a la celda **BevelMaterialType** por su índice desde un programa, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowBevelProperties** <br/> |
| Índice de celda:  <br/> |**visBevelMaterialType** <br/> |

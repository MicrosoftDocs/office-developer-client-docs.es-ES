# <a name="drawingresizetype-cell-page-properties-section"></a>Celda DrawingResizeType (sección Propiedades de página)

Determina si la página de dibujo cambia de tamaño automáticamente para ajustarse al diagrama. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia desde otra fórmula a la celda DrawingResizeType por su nombre, o desde un programa mediante la propiedad **CellsU**, use lo siguiente. 
  
|||
|:-----|:-----|
|Nombre de celda:  <br/> |DrawingResizeType  <br/> |
   
Para obtener una referencia desde un programa a la celda DrawingResizeType por su índice, use la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
|Índice de sección:  <br/> |**visSectionObject** <br/> |
|Índice de fila:  <br/> |**visRowPage** <br/> |
|Índice de celda:  <br/> |**visPageDrawResizeType** <br/> |

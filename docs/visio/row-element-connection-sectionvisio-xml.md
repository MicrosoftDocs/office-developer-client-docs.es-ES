# <a name="row-element-connection-section-visio-xml"></a>Elemento Row (sección conexión) (XML de Visio)

Contiene las coordenadas x o y, la dirección horizontal y vertical y el tipo de un único punto de conexión de una forma.
  
## <a name="element-information"></a>Información del elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[ConnectionRow_Type](connectionrow_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Archivo de esquema** <br/> |VisioSchema15. xsd  <br/> |
|**Elementos de documento** <br/> |Master #. XML, página #. XML  <br/> |
   
## <a name="definition"></a>Definición

```XML
< xs:element name="Row" type="ConnectionRow_Type" >
</xs:element >
```

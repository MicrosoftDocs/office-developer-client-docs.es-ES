# <a name="quickstyleshadowcolor-cell-quick-style-section"></a>Celda QuickStyleShadowColor (sección estilos rápidos)

Determina el color del tema que usa la sombra de una forma, como un entero de 0 a 7.
  
|||
|:-----|:-----|
|Valor  <br/> |Descripción  <br/> |
|comprendi  <br/> |El color de sombra de la forma hereda del color del tema oscuro.  <br/> |
|1  <br/> |El color de sombra de la forma hereda del color del tema claro.  <br/> |
|segundo  <br/> |El color de sombra de la forma hereda del color del tema énfasis 1.  <br/> |
|3  <br/> |El color de sombra de la forma se hereda del color del tema énfasis 2.  <br/> |
|4   <br/> |El color de sombra de la forma hereda del color del tema énfasis 3.  <br/> |
|5   <br/> |El color de sombra de la forma hereda del color del tema énfasis 4.  <br/> |
|6   <br/> |El color de sombra de la forma hereda del color del tema énfasis 5.  <br/> |
|7   <br/> |El color de sombra de la forma se hereda del color del tema énfasis 6.  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleShadowColor** por su nombre desde otra fórmula, por valor del atributo **N** de un elemento **Cell** , o desde un programa mediante la propiedad **CellsU** , utilice: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleShadowColor  <br/> |
   
Para obtener una referencia desde un programa a la celda **QuickStyleShadowColor** por su índice, utilice la propiedad **CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleShadowColor** <br/> |

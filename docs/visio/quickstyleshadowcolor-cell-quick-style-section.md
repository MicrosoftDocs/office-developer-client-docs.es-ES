# <a name="quickstyleshadowcolor-cell-quick-style-section"></a>Celda QuickStyleShadowColor (sección Estilo rápido)

Determina el color del tema que usa la sombra de una forma, como un entero de 0 a 7.
  
|||
|:-----|:-----|
|Valor  <br/> |Descripción  <br/> |
|0  <br/> |El color de sombra de forma hereda del color del tema Oscuro.  <br/> |
|1  <br/> |El color de sombra de forma hereda del color del tema Luz.  <br/> |
|2  <br/> |El color de sombra de forma se hereda del color del tema Énfénf 1.  <br/> |
|3  <br/> |El color de sombra de forma hereda del color del tema Énfénf 2.  <br/> |
|4   <br/> |El color de sombra de forma hereda del color del tema Énfénf 3.  <br/> |
|5   <br/> |El color de sombra de forma hereda del color del tema Énfénf 4.  <br/> |
|6   <br/> |El color de sombra de forma hereda del color del tema Énfénf 5.  <br/> |
|7   <br/> |El color de sombra de forma hereda del color del tema Énfénf 6.  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **QuickStyleShadowColor** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | QuickStyleShadowColor  <br/> |
   
Para obtener una referencia desde un programa a la celda **QuickStyleShadowColor** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de celda:  <br/> |**visQuickStyleShadowColor** <br/> |

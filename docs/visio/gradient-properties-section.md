# <a name="gradient-properties-section"></a>Sección Propiedades de degradado

Muestra las propiedades de los efectos de color de línea o de relleno de degradado aplicados a la forma. En esta sección también incluye celdas que controlan si degradado de una forma gira con la forma o si la forma se da por supuesto un degradado aplicado cuando se agrupan con otras formas. 
  
## <a name="remarks"></a>Comentarios

Puede establecer la configuración de degradado en el cuadro de diálogo **Formato de forma** (**Home** > **Estilos de forma** > **relleno** > **Opciones de relleno** > **relleno** > **relleno degradado**).

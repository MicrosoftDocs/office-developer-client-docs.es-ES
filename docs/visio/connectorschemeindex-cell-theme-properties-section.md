# <a name="connectorschemeindex-cell-theme-properties-section"></a>Celda ConnectorSchemeIndex (sección Propiedades del tema)

Determina el esquema de conector de un tema que se aplica a la forma, como un entero. 
  
## <a name="remarks"></a>Comentarios

Para obtener una referencia a la celda **ConnectorSchemeIndex** por su nombre desde otra fórmula, por valor del atributo **N** de un **elemento Cell** o desde un programa mediante la **propiedad CellsU,** use: 
  
|||
|:-----|:-----|
| Nombre de celda:  <br/> | ConnectorSchemeIndex  <br/> |
   
Para obtener una referencia desde un programa a la celda **ConnectorSchemeIndex** por su índice, utilice la **propiedad CellsSRC** con los argumentos siguientes: 
  
|||
|:-----|:-----|
| Índice de sección:  <br/> |**visSectionObject** <br/> |
| Índice de fila:  <br/> |**visRowThemeProperties** <br/> |
| Índice de celda:  <br/> |**visConnectorSchemeIndex ** <br/> |

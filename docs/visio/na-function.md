# <a name="na-function"></a>Función NA

Devuelve el valor de error #NA!. 
  
La función NA se utiliza en una celda ShapeSheet en la que falta información para evitar que el valor predeterminado de la celda se utilice en los cálculos.
  
## <a name="syntax"></a>Sintaxis

NA( )
  
### <a name="return-value"></a>Valor devuelto

String
  
## <a name="remarks"></a>Observaciones

NA es el error que significa que no hay ningún valor disponible.

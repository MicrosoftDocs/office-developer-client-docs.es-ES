# <a name="sum-function"></a>Función SUM

Devuelve la suma de una lista de números.
  
## <a name="syntax"></a>Sintaxis

SUMA (** *número1* **, ** *número2* **,..., ** *[numberN]* **) 
  
### <a name="parameters"></a>Parámetros

|**Name**|**Obligatorio/opcional**|**Tipo de datos**|**Descripción**|
|:-----|:-----|:-----|:-----|
| _Número1_ <br/> |Obligatorio  <br/> |**Numeric** <br/> |El primer número.  <br/> |
| _número2_ <br/> |Obligatorio  <br/> |**Numeric** <br/> |El segundo número.  <br/> |
| _numberN_ <br/> |Opcional  <br/> |**Numeric** <br/> |El número n.  <br/> |
   
### <a name="return-value"></a>Valor devuelto

Numeric
  
## <a name="example"></a>Ejemplo

SUM(5,7,12)
  
Devuelve 24.

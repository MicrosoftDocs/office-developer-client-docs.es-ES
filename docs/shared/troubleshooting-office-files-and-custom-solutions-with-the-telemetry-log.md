<span data-ttu-id="202f7-p105">Cuando se carga, usa, cierra un archivo o solución de Office o bien cuando este genera un error en una de las aplicaciones seleccionadas de Office 2013, la aplicación añade un registro en el almacén de datos local (una base de datos del mismo equipo) que incluye información sobre el evento.

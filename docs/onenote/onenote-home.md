# <a name="onenote"></a>OneNote

Busque contenido sobre procedimientos, artículos técnicos, contenido de referencia y entradas de blog para desarrollar soluciones y personalizar las aplicaciones de cliente de escritorio de OneNote.
  
> [!NOTE]
> Las API que se describen en esta documentación están destinadas únicamente para soluciones de cliente de escritorio de OneNote Win32 en escenarios desconectados. Para escenarios conectados, use las API del servicio de OneNote recomendada. Para obtener más información, visite [dev.onenote.com](http://dev.onenote.com/).

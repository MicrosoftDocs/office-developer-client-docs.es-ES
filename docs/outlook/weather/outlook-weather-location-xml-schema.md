# <a name="outlook-weather-location-xml-schema"></a>Outlook Esquema XML de ubicación meteorológica

Esta sección contiene la referencia Outlook esquema de 2013 para el esquema XML Outlook ubicación meteorológica. Este esquema permite a los proveedores de servicios meteorológicos responder a Outlook solicitud de servicio web para un código de ubicación. El código de ubicación debe corresponder a la ubicación para la que un usuario solicita información meteorológica en la barra de meteorología.

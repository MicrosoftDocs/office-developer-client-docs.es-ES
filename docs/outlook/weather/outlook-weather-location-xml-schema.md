# <a name="outlook-weather-location-xml-schema"></a>Esquema XML de ubicación meteorológica de Outlook

Esta sección contiene la referencia de esquema de Outlook 2013 para el esquema de XML de ubicación de meteorología de Outlook. Este esquema permite a los proveedores de servicio de tiempo responder a una solicitud de servicio web de Outlook para un código de ubicación. El código de ubicación debe corresponder a la ubicación para la que un usuario está solicitando información meteorológica en la barra de meteorología.

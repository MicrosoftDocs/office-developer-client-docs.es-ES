# <a name="outlook-weather-location-xml-schema"></a>Esquema XML de ubicación meteorológica de Outlook

Esta sección contiene la referencia del esquema de Outlook 2013 para el esquema XML de la ubicación meteorológica de Outlook. Este esquema permite a los proveedores de servicios meteorológicos responder a una solicitud de servicio web de Outlook para un código de ubicación. El código de ubicación debe corresponder a la ubicación para la que un usuario solicita información meteorológica en la barra de meteorología.

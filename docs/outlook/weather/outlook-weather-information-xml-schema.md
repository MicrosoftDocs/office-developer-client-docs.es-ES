# <a name="outlook-weather-information-xml-schema"></a>Esquema XML de información meteorológica de Outlook

Esta sección contiene la referencia de esquema de Outlook 2013 para el esquema de XML de información de meteorología de Outlook. Este esquema permite a los proveedores de servicio de tiempo enviar información de previsión meteorológica para una ubicación especificada como una respuesta del servicio web con Outlook, por lo que Outlook puede mostrar la información en la barra de meteorología.

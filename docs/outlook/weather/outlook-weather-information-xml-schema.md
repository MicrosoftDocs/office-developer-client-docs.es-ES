# <a name="outlook-weather-information-xml-schema"></a>Outlook Esquema XML de información meteorológica

Esta sección contiene la referencia Outlook esquema de 2013 para el Outlook XML de información meteorológica. Este esquema permite a los proveedores de servicios meteorológicos enviar información de previsión meteorológica para una ubicación especificada como respuesta de servicio web a Outlook, de modo que Outlook pueda mostrar la información en la barra de meteorología.

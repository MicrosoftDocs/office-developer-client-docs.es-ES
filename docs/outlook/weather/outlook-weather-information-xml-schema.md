# <a name="outlook-weather-information-xml-schema"></a>Esquema XML de información meteorológica de Outlook

Esta sección contiene la referencia de esquema de Outlook 2013 para el esquema XML de información meteorológica de Outlook. Este esquema permite que los proveedores de servicios meteorológicos envíen información sobre pronósticos meteorológicos para una ubicación determinada como respuesta de servicio Web a Outlook, de modo que Outlook pueda mostrar la información en la barra de Meteorología.

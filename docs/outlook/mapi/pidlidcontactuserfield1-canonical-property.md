# <a name="pidlidcontactuserfield1-canonical-property"></a>Propiedad canónica PidLidContactUserField1

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el primer campo del contacto destinado a usos varios para el contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactUserField1  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x0000804F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene texto que no está relacionado con ninguna otra propiedad específica del contacto.

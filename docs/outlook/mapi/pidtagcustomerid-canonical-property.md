# <a name="pidtagcustomerid-canonical-property"></a>Propiedad canónica PidTagCustomerId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de identificador de cliente del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CUSTOMER_ID, PR_CUSTOMER_ID_A, PR_CUSTOMER_ID_W  <br/> |
|Identificador:  <br/> |0x3A4A  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

El contenido de estas propiedades se define por el usuario o la organización del usuario.

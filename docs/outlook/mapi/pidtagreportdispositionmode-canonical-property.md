# <a name="pidtagreportdispositionmode-canonical-property"></a>Propiedad canónica PidTagReportDispositionMode

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica la eliminación del recibo de los mensajes que solicitan recibos. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_DISPOSITION_MODE, PR_REPORT_DISPOSITION_MODE_A, PR_REPORT_DISPOSITION_MODE_W  <br/> |
|Identificador:  <br/> |0x0081  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Los valores posibles para esta propiedad son "manual-action/MDN-sent-automatically" y "manual-action/MDN-sent-manually".

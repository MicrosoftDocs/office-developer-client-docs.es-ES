# <a name="pidtagreportdispositionmode-canonical-property"></a>Propiedad canónica PidTagReportDispositionMode

  
  
**Hace referencia a**: Outlook 
  
Indica la disposición de la recepción para los mensajes que solicitar confirmación. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_DISPOSITION_MODE, PR_REPORT_DISPOSITION_MODE_A, PR_REPORT_DISPOSITION_MODE_W  <br/> |
|Identificador:  <br/> |0x0081  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobres MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Los valores posibles para esta propiedad son "manual-acción/MDN-enviado-automáticamente" y "manual-acción/MDN-enviado-manualmente".

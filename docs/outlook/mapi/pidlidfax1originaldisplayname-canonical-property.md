# <a name="pidlidfax1originaldisplayname-canonical-property"></a>Propiedad canónico PidLidFax1OriginalDisplayName

  
  
**Se aplica a**: Outlook 
  
Especifica el nombre para mostrar original de dirección de fax del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1OriginalDisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080B4  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

Esta propiedad, si está presente, debe establecerse en el mismo valor que la propiedad **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).

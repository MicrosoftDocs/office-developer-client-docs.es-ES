# <a name="pidtagreplyrequested-canonical-property"></a>Propiedad canónica PidTagReplyRequested

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si un remitente de mensaje solicita una respuesta de un destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPLY_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C17  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

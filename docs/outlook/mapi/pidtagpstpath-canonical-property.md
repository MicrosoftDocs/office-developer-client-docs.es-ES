# <a name="pidtagpstpath-canonical-property"></a>Propiedad canónica PidTagPstPath

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene la ubicación del objeto en el almacén de carpeta personal para crear o configurar.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PST_PATH, PR_PST_PATH_A, PR_PST_PATH_W  <br/> |
|Identificador:  <br/> |0x6700  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Tabla de almacenamiento personal (.pst) interno  <br/> |

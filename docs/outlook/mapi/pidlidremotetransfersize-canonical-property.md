# <a name="pidlidremotetransfersize-canonical-property"></a>Propiedad canónica PidLidRemoteTransferSize

  
  
**Hace referencia a**: Outlook 
  
Indica el tamaño de transferencia, en bytes, de un elemento remoto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidRemoteXferSize  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Remote  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008F05  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Mensaje remoto  <br/> |

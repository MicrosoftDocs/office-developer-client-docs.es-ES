# <a name="pidtagstoreentryid-canonical-property"></a>Propiedad canónica PidTagStoreEntryId

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador único de entrada del almacén de mensajes donde reside un objeto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_STORE_ENTRYID  <br/> |
|Identificador:  <br/> |0x0FFB  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Propiedades de Id.  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se utiliza para abrir un almacén de mensajes con el método [IMAPISession::OpenMsgStore](imapisession-openmsgstore.md) . También se usa para abrir cualquier objeto cuyo propietario es por el almacén de mensajes. 
  
Para un almacén de mensajes, esta propiedad es idéntica a la propiedad de **entrada del objeto** ([PidTagEntryId](pidtagentryid-canonical-property.md)) del almacén. Una aplicación cliente puede comparar las dos propiedades con el método [IMAPISession::CompareEntryIDs](imapisession-compareentryids.md) .

# <a name="pidlidlogflags-canonical-property"></a>Propiedad canónica PidLidLogFlags

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene metadatos sobre el diario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLogFlags  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Log  <br/> |
|Id. largo (LID):  <br/> |0x0000870C  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diario  <br/> |
   
## <a name="remarks"></a>Comentarios

El campo de bits que contiene metadatos sobre el diario debe ser cero o "0x40000000".

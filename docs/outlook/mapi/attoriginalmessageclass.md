# <a name="attoriginalmessageclass"></a>attOriginalMessageClass

**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Una clase de mensaje se almacena como una cadena. La cadena codificada suele contener el nombre especificado por MAPI de la clase de mensaje.

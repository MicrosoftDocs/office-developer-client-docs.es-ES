# <a name="attoriginalmessageclass"></a>attOriginalMessageClass

**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Una clase de mensaje se almacena como una cadena. Normalmente, la cadena codificada contiene el nombre especificado de MAPI de la clase de mensaje.

# <a name="pidlidtaskglobalid-canonical-property"></a>Propiedad canónica PidLidTaskGlobalId

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Busca una tarea existente, mediante el uso de un GUID, tras la recepción de una respuesta de tarea o una actualización de tarea.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskGlobalObjId  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008519  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se deja sin establecer para las tareas sin asignar.

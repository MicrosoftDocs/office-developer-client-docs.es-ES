# <a name="pidlidtaskglobalid-canonical-property"></a>Propiedad canónica PidLidTaskGlobalId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Busca una tarea existente, mediante un GUID, al recibir una respuesta de tarea o actualización de tarea.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskGlobalObjId  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008519  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Tarea  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se deja sin establecer para las tareas no asignadas.

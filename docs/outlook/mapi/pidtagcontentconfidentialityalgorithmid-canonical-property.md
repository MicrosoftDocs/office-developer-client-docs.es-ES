# <a name="pidtagcontentconfidentialityalgorithmid-canonical-property"></a>Propiedad canónica PidTagContentConfidentialityAlgorithmId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador para el algoritmo que se usa para confirmar la confidencialidad del contenido del mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTENT_CONFIDENTIALITY_ALGORITHM_ID  <br/> |
|Identificador:  <br/> |0x0006  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Exchange  <br/> |

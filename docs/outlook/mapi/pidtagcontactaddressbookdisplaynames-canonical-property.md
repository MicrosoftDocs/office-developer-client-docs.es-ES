# <a name="pidtagcontactaddressbookdisplaynames-canonical-property"></a>Propiedad canónica PidTagContactAddressBookDisplayNames

  
  
**Hace referencia a**: Outlook 
  
Contiene una lista de nombres para mostrar para los usuarios de la libreta de direcciones de contactos de mensajería.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_DISPLAY_NAMES, PR_CONTAB_DISPLAY_NAMES_A, PR_CONTAB_DISPLAY_NAMES_W  <br/> |
|Identificador:  <br/> |0x6624  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario y la organización del usuario.

# <a name="pidtagcomputernetworkname-canonical-property"></a>Propiedad canónica PidTagComputerNetworkName

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre de la red que sirve para transmitir el mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_COMPUTER_NETWORK_NAME, PR_COMPUTER_NETWORK_NAME_A, PR_COMPUTER_NETWORK_NAME_W  <br/> |
|Identificador:  <br/> |0x3A49  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |

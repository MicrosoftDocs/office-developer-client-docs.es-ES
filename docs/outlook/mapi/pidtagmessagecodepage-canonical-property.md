# <a name="pidtagmessagecodepage-canonical-property"></a>Propiedad canónica PidTagMessageCodepage

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene la página de código que se usa para el mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MESSAGE_CODEPAGE  <br/> |
|Identificador:  <br/> |0x3FFD  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Common  <br/> |
   
## <a name="remarks"></a>Comentarios

Si esta propiedad se establece en cero (0), se usa la página de código de objeto de carpeta.

# <a name="pidlidfax1displayname-canonical-property"></a>Propiedad canónica PidLidFax1DisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre para mostrar de la dirección de fax comercial de un contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1DisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080B0  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

# <a name="pidtagreportdispositiontoemailaddresses-canonical-property"></a>Propiedad canónica PidTagReportDispositionToEmailAddresses

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica las direcciones de correo electrónico de los destinatarios del informe para los mensajes que solicitan confirmaciones.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_A, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_W  <br/> |
|Identificador:  <br/> |0x0084  <br/> |
|Tipo de datos:  <br/> |PT_MV_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |

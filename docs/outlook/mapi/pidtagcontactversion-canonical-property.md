# <a name="pidtagcontactversion-canonical-property"></a>Propiedad canónica PidTagContactVersion

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la versión de información de contacto de un destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTACT_VERSION  <br/> |
|Identificador:  <br/> |0x3A52  <br/> |
|Tipo de datos:  <br/> |PT_CLSID  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es una de las propiedades que proporciona la identificación y el acceso a información acerca de un destinatario. Estas propiedades son definidas por el destinatario y la organización del destinatario.

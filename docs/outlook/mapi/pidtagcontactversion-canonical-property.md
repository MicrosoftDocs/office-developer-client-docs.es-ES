# <a name="pidtagcontactversion-canonical-property"></a>Propiedad canónica PidTagContactVersion

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la versión de la información de contacto de un destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTACT_VERSION  <br/> |
|Identificador:  <br/> |0x3A52  <br/> |
|Tipo de datos:  <br/> |PT_CLSID  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es una de las propiedades que proporciona información de identificación y acceso a un destinatario. Estas propiedades las define el destinatario y la organización del destinatario.

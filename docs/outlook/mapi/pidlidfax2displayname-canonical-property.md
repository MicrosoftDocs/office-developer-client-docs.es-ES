# <a name="pidlidfax2displayname-canonical-property"></a>Propiedad canónica PidLidFax2DisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre para mostrar de la dirección de fax particular del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax2DisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080C0  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |

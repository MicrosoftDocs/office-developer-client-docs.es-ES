# <a name="pidlidfax2displayname-canonical-property"></a>Propiedad canónico PidLidFax2DisplayName

  
  
**Se aplica a**: Outlook 
  
Especifica el nombre para mostrar de la dirección de fax del domicilio particular del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax2DisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080C0  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

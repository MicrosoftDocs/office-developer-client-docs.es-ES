# <a name="pidtagabproviders-canonical-property"></a>Propiedad canónica PidTagAbProviders

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de identificadores de los proveedores de la libreta de direcciones en el perfil actual. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_AB_PROVIDERS  <br/> |
|Identificador:  <br/> |0x3D01  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones  <br/> |
   
## <a name="remarks"></a>Comentarios

No utilice esta propiedad. Está reservado para su uso por MAPI.

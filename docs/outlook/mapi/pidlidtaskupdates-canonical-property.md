# <a name="pidlidtaskupdates-canonical-property"></a>Propiedad canónica PidLidTaskUpdates

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Indica si se ha solicitado el encargado de la tarea para enviar una actualización de tarea cuando cambia la tarea asignada.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskUpdates  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000811B  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Task  <br/> |

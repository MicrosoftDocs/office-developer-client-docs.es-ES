# <a name="pidlidtaskupdates-canonical-property"></a>Propiedad canónica PidLidTaskUpdates

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si se ha solicitado al usuario al que se asigna una tarea que envíe una actualización de tarea cuando cambie la tarea asignada.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskUpdates  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x0000811B  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Task  <br/> |

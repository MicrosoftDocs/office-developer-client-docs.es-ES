# <a name="pidtagcontactaddressbookstoresupportmasks-canonical-property"></a>Propiedad canónica PidTagContactAddressBookStoreSupportMasks

  
  
**Hace referencia a**: Outlook 
  
Contiene los indicadores que indica características compatibles de la tienda.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_STORE_SUPPORT_MASKS  <br/> |
|Identificador:  <br/> |0x6621  <br/> |
|Tipo de datos:  <br/> |PT_MV_LONG  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se obtiene de los almacenes que contiene las carpetas de contactos. El proveedor de la libreta de direcciones de contacto usa para evaluar la adecuación de características compatibles de la tienda. Es una propiedad en una sección de perfil de la libreta de direcciones de contacto.

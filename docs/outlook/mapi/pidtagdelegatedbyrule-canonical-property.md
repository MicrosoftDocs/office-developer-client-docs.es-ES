# <a name="pidtagdelegatedbyrule-canonical-property"></a>Propiedad canónica PidTagDelegatedByRule

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si una regla delega el mensaje de una carpeta.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_DELEGATED_BY_RULE  <br/> |
|Identificador:  <br/> |0x3FE3  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Estado MAPI  <br/> |

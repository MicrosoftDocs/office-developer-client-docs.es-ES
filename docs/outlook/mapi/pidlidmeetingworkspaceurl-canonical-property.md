# <a name="pidlidmeetingworkspaceurl-canonical-property"></a>Propiedad canónica PidLidMeetingWorkspaceUrl

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la dirección URL del área de trabajo de reunión asociada a un objeto calendar.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidMWSURL  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Id. largo (LID):  <br/> |0x00008209  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad no es necesaria.

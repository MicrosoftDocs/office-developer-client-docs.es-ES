# <a name="pidtagabdefaultdir-canonical-property"></a>Propiedad canónica PidTagAbDefaultDir

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador de entrada del contenedor de libretas de direcciones para mostrar en primer lugar. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_AB_DEFAULT_DIR  <br/> |
|Identificador:  <br/> |0x3D06  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones  <br/> |
   
## <a name="remarks"></a>Comentarios

No utilice esta propiedad. Está reservado para su uso por parte de MAPI.

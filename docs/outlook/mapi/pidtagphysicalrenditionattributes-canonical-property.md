# <a name="pidtagphysicalrenditionattributes-canonical-property"></a>Propiedad canónica PidTagPhysicalRenditionAttributes

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador de objeto ASN.1 que se usa para representar datos adjuntos de mensajes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PHYSICAL_RENDITION_ATTRIBUTES  <br/> |
|Identificador:  <br/> |0x0C10  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

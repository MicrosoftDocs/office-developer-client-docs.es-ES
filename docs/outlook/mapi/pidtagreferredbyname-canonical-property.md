# <a name="pidtagreferredbyname-canonical-property"></a>Propiedad canónica PidTagReferredByName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre de la referencia del usuario de correo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REFERRED_BY_NAME, PR_REFERRED_BY_NAME_A, PR_REFERRED_BY_NAME_W  <br/> |
|Identificador:  <br/> |0x3A47  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |

# <a name="pidlidclassification-canonical-property"></a>Propiedad canónica PidLidClassification

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Indica el nombre de la clasificación de mensajes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidClassification  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000085B6  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |

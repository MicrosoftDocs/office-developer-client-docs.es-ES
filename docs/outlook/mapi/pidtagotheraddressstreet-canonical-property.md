# <a name="pidtagotheraddressstreet-canonical-property"></a>Propiedad canónica PidTagOtherAddressStreet

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la dirección de la otra calle del usuario de correo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_STREET, PR_OTHER_ADDRESS_STREET_A, PR_OTHER_ADDRESS_STREET_W  <br/> |
|Identificador:  <br/> |0x3A63  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

# <a name="table-positioning"></a>Posición de tabla

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
La posición actual dentro de una tabla siempre se indica mediante un cursor. Hay un cursor para cada vista de una tabla; su valor se establece por implementador de la tabla. Cuando un proveedor de servicio o de cliente con la tabla, realiza una llamada para cambiar la posición de la tabla, se restablece el valor del cursor. Puede cambiar la posición de una tabla con:
  
- Un marcador.
    
- Un valor fraccionario.
    
- Un filtro.

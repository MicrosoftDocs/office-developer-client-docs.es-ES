# <a name="pidtagcallbacktelephonenumber-canonical-property"></a>Propiedad canónica PidTagCallbackTelephoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un número de teléfono que el destinatario del mensaje puede usar para llegar al remitente. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CALLBACK_TELEPHONE_NUMBER, PR_CALLBACK_TELEPHONE_NUMBER_A, PR_CALLBACK_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A02  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son ejemplos de las propiedades que proporcionan información de identificación y acceso sobre un destinatario. El destinatario y la organización del destinatario las definen.

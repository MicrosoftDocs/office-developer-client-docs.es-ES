# <a name="pidtagprofiletype-canonical-property"></a>Propiedad canónica PidTagProfileType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el tipo de almacén de información.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PROFILE_TYPE  <br/> |
|Identificador:  <br/> |0x660A  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Configuración de perfil MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Los valores posibles de esta propiedad son:
  
- 1: usuario principal
    
- 2 - Delegado
    
- 3- Tienda pública
    
- 4- Suscripción

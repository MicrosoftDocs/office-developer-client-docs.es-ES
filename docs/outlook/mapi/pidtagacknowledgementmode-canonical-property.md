# <a name="pidtagacknowledgementmode-canonical-property"></a>Propiedad canónica PidTagAcknowledgementMode

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador del modo de confirmación del mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ACKNOWLEDGEMENT_MODE  <br/> |
|Identificador:  <br/> |0 x 0001  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad puede tener exactamente uno de los siguientes valores:
  
|**Valor**|**Descripción**|
|:-----|:-----|
|0  <br/> |Confirmación de manual.  <br/> |
|1  <br/> |Confirmación automática.  <br/> |

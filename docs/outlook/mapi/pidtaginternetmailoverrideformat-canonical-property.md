# <a name="pidtaginternetmailoverrideformat-canonical-property"></a>Propiedad canónica PidTagInternetMailOverrideFormat

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica el método de codificación y la inclusión HTML para datos adjuntos que deben establecerse en el correo saliente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INETMAIL_OVERRIDE_FORMAT  <br/> |
|Identificador:  <br/> |0x5902  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Comentarios

# <a name="pidlidfax1emailtype-canonical-property"></a>Propiedad canónico PidLidFax1EmailType

  
  
**Se aplica a**: Outlook 
  
Especifica el tipo de correo electrónico de la dirección de fax del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1EmailType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080B7  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

# <a name="pidlidfax1emailtype-canonical-property"></a>Propiedad canónica PidLidFax1EmailType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el tipo de correo electrónico de la dirección de fax de la empresa del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1EmailType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080B7  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |

# <a name="pidlidemail2addresstype-canonical-property"></a>Propiedad canónica PidLidEmail2AddressType

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Especifica el tipo de dirección de la segunda dirección de correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail2AddrType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008092  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Si está presente, el valor de esta propiedad debe ser un tipo de dirección válida.

# <a name="pidlidclassificationdescription-canonical-property"></a>Propiedad canónica PidLidClassificationDescription

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Describe lo que la clasificación de mensajes pretende lograr.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidClassDesc  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000085B7  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |

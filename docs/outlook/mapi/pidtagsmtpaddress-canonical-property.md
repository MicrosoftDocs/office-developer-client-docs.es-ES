# <a name="pidtagsmtpaddress-canonical-property"></a>Propiedad canónica PidTagSmtpAddress

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la dirección SMTP del objeto de libreta de direcciones.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SMTP_ADDRESS, PR_SMTP_ADDRESS_A, PR_SMTP_ADDRESS_W  <br/> |
|Identificador:  <br/> |0x39FE  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |

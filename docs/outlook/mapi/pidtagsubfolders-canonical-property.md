# <a name="pidtagsubfolders-canonical-property"></a>Propiedad canónica PidTagSubfolders

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si una carpeta contiene las subcarpetas.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SUBFOLDERS  <br/> |
|Identificador:  <br/> |0x360A  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contenedor MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Almacenes de mensaje deben proporcionar esta propiedad para todas las carpetas.

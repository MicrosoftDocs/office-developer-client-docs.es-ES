# <a name="pidtagformclassid-canonical-property"></a>Propiedad canónica PidTagFormClassId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador único global (GUID) de vinculación e incrustación de objetos de 128 bits (OLE) de un formulario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FORM_CLSID  <br/> |
|Identificador:  <br/> |0x3302  <br/> |
|Tipo de datos:  <br/> |PT_CLSID  <br/> |
|Área:  <br/> |Common MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

La estructura [MAPIUID](mapiuid.md) contiene la definición del identificador único.

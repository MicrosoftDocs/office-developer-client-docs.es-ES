# <a name="pidtagautoforwardcomment-canonical-property"></a>Propiedad canónica PidTagAutoForwardComment

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un comentario agregado por el agente de reenvío automático.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_AUTO_FORWARD_COMMENT, PR_AUTO_FORWARD_COMMENT_A, PR_AUTO_FORWARD_COMMENT_W  <br/> |
|Identificador:  <br/> |0x0004  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Informe de errores general  <br/> |

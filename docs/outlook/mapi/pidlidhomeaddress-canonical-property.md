# <a name="pidlidhomeaddress-canonical-property"></a>Propiedad canónica PidLidHomeAddress

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la dirección completa de la dirección principal del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidHomeAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x0000801A  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe ser una combinación de otras propiedades de dirección física y se basa en la configuración regional del cliente.

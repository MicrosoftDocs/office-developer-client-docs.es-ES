# <a name="pidtagassistant-canonical-property"></a>Propiedad canónica PidTagAssistant

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre del asistente administrativo del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ASSISTANT, PR_ASSISTANT_A, PR_ASSISTANT_W  <br/> |
|Identificador:  <br/> |0x3A30  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Los define el destinatario y su organización.

# <a name="pidnamekeywords-canonical-property"></a>Propiedad canónico PidNameKeywords

  
  
**Se aplica a**: Outlook 
  
Contiene palabras clave o categorías para el objeto de mensaje.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |None  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |Palabras clave  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |
   
## <a name="remarks"></a>Notas

Un valor de cadena múltiple que especifica las categorías para un objeto de mensaje, la longitud de cada cadena dentro de esta cadena de varios valores de propiedad, debe ser menor que 256.

# <a name="pidlidyomicompanyname-canonical-property"></a>Propiedad canónica PidLidYomiCompanyName

  
  
**Hace referencia a**: Outlook 
  
Especifica la pronunciación del nombre de la compañía del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidYomiCompanyName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000802E  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |

# <a name="pidlidfax1richtextformat-canonical-property"></a>Propiedad canónico PidLidFax1RichTextFormat

  
  
**Se aplica a**: Outlook 
  
Especifica el formato de texto enriquecido de dirección de fax del trabajo del contacto.

|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1RTF  <br/> |

|Identificador de tipo Long (LID):  <br/> |0x000080B6  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contacto  <br/> |

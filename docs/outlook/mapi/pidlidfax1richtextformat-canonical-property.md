# <a name="pidlidfax1richtextformat-canonical-property"></a>Propiedad canónica PidLidFax1RichTextFormat

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el formato de texto enriquecido de la dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1RTF  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080B6  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contacto  <br/> |

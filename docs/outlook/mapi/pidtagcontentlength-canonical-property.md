# <a name="pidtagcontentlength-canonical-property"></a>Propiedad canónica PidTagContentLength

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una longitud de mensaje, en bytes, pasada a una aplicación cliente o proveedor de servicios para determinar si se puede entregar un mensaje de esa longitud. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTENT_LENGTH  <br/> |
|Identificador:  <br/> |0x0009  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |

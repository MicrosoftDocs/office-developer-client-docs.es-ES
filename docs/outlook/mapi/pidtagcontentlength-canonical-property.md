# <a name="pidtagcontentlength-canonical-property"></a>Propiedad canónica PidTagContentLength

  
  
**Hace referencia a**: Outlook 
  
Contiene una longitud de mensaje, en bytes, que se pasan a una aplicación de cliente o un proveedor de servicios para determinar si se puede entregar un mensaje de esa longitud. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTENT_LENGTH  <br/> |
|Identificador:  <br/> |0 x 0009  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |

# <a name="pidtaglanguages-canonical-property"></a>Propiedad canónico PidTagLanguages

  
  
**Se aplica a**: Outlook 
  
Contiene una lista de ASCII de los idiomas que se incorporan en un mensaje. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_LANGUAGES, PR_LANGUAGES_A, PR_LANGUAGES_W  <br/> |
|Identificador:  <br/> |0x002F  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |
   
## <a name="remarks"></a>Notas

Estas propiedades contienen una secuencia de códigos de país o región de dos caracteres que están separados por comas.

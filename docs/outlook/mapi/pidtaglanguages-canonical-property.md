# <a name="pidtaglanguages-canonical-property"></a>Propiedad canónica PidTagLanguages

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista ASCII de los idiomas que se incorporan en un mensaje. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_LANGUAGES, PR_LANGUAGES_A, PR_LANGUAGES_W  <br/> |
|Identificador:  <br/> |0x002F  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Mensajes generales  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades contienen una secuencia de códigos de país o región de dos caracteres separados por comas.

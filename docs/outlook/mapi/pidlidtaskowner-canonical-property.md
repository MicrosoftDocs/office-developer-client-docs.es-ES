# <a name="pidlidtaskowner-canonical-property"></a>Propiedad canónica PidLidTaskOwner

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre del propietario de la tarea.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskOwner  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000811F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Task  <br/> |

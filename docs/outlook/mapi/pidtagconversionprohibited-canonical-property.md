# <a name="pidtagconversionprohibited-canonical-property"></a>Propiedad canónica PidTagConversionProhibited

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si las conversiones de mensajes están prohibidas de forma predeterminada para el usuario de mensajería asociado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONVERSION_PROHIBITED  <br/> |
|Identificador:  <br/> |0x3A03  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Dirección  <br/> |

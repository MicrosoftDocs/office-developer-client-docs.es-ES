# <a name="pidnamelastsavedatetime-canonical-property"></a>Propiedad canónica PidNameLastSaveDateTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la hora en que se guardó el archivo por última vez.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |LastSaveDtm  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Común  <br/> |

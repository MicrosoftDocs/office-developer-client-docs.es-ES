# <a name="pidtagbirthday-canonical-property"></a>Propiedad canónica PidTagBirthday

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la fecha de cumpleaños del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_BIRTHDAY  <br/> |
|Identificador:  <br/> |0x3A42  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es una de las propiedades que proporciona la identificación y el acceso a información sobre un contacto. Estas propiedades son definidas por el usuario y la organización del usuario.

# <a name="pidtagbirthday-canonical-property"></a>Propiedad canónica PidTagBirthday

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el cumpleaños del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_BIRTHDAY  <br/> |
|Identificador:  <br/> |0x3A42  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es una de las propiedades que proporciona información de identificación y acceso sobre un contacto. El usuario y la organización del usuario definen estas propiedades.

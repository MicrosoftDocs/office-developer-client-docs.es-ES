# <a name="pidlidfax1entryid-canonical-property"></a>Propiedad canónica PidLidFax1EntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el EntryID de la dirección de fax de la empresa del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1EntryID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080B1  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contact  <br/> |

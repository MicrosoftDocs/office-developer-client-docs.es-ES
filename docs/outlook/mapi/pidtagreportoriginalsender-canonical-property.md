# <a name="pidtagreportoriginalsender-canonical-property"></a>Propiedad canónica PidTagReportOriginalSender

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica el remitente original del mensaje para los mensajes que solicitan confirmaciones.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_ORIGINAL_SENDER, PR_REPORT_ORIGINAL_SENDER_A, PR_REPORT_ORIGINAL_SENDER_W  <br/> |
|Identificador:  <br/> |0x0082  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |

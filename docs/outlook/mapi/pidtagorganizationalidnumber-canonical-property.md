# <a name="pidtagorganizationalidnumber-canonical-property"></a>Propiedad canónica PidTagOrganizationalIdNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un número de identificación de la organización para el contacto, como un número de id. de empleado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ORGANIZATIONAL_ID_NUMBER, PR_ORGANIZATIONAL_ID_NUMBER_A, PR_ORGANIZATIONAL_ID_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A10  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas son propiedades opcionales. Su uso lo determina el usuario de mensajería u organización.

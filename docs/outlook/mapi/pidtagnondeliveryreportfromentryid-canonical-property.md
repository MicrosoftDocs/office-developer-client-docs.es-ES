# <a name="pidtagnondeliveryreportfromentryid-canonical-property"></a>Propiedad canónica PidTagNonDeliveryReportFromEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Si establece en un informe de no entrega, denota el identificador de entrada de la con errores de envío-como usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_NDR_FROM_ENTRYID  <br/> |
|Identificador:  <br/> |0x6904  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |General de mensajería  <br/> |

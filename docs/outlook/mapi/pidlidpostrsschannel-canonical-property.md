# <a name="pidlidpostrsschannel-canonical-property"></a>Propiedad canónico PidLidPostRssChannel

  
  
**Se aplica a**: Outlook 
  
Contiene el contenido de la \<título\> campo desde el código XML de la Atom \<fuente\> o RSS \<canal\>.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidPostRssChannel  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_PostRss  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008904  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |RSS  <br/> |

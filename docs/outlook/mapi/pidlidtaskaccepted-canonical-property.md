# <a name="pidlidtaskaccepted-canonical-property"></a>Propiedad canónica PidLidTaskAccepted

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Indica si un usuario asignado a la tarea respondió a una solicitud de tarea.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskAccepted  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008108  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Comentarios

El cliente establece esta propiedad en FALSE para una nueva tarea o el cliente establece esta propiedad en TRUE cuando se acepta o rechaza una tarea. Si la propiedad se deja sin establecer, se supone un valor FALSE.

# <a name="pidlidautolog-canonical-property"></a>Propiedad canónica PidLidAutoLog

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica si se va a crear un diario para cada acción asociada a este contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidAutoLog  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x00008025  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contacto  <br/> |

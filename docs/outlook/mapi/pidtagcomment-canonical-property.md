# <a name="pidtagcomment-canonical-property"></a>Propiedad canónica PidTagComment

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un comentario sobre el propósito o el contenido de un objeto. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_COMMENT, PR_COMMENT_A, PR_COMMENT_W  <br/> |
|Identificador:  <br/> |0x3004  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Common  <br/> |
   
## <a name="remarks"></a>Comentarios

El contenido de la cadena es definido por el usuario de mensajería.

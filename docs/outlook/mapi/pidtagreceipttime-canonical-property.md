# <a name="pidtagreceipttime-canonical-property"></a>Propiedad canónica PidTagReceiptTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la fecha y la hora en que se genera un informe de entrega.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RECEIPT_TIME  <br/> |
|Identificador:  <br/> |0x002A  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe establecerse mediante el proveedor de almacén de mensajes que recibe el mensaje original y genera el informe.

# <a name="pidtagcontactaddressbookuid-canonical-property"></a>Propiedad canónica PidTagContactAddressBookUid

  
  
**Hace referencia a**: Outlook 
  
Contiene el identificador exclusivo que identifica esta sesión de la libreta de direcciones de contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_UID  <br/> |
|Identificador:  <br/> |0x6601  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Si ha configurado varias libretas de direcciones de contacto, esta propiedad se debe establecer para cada uno de ellos.

# <a name="pidlidnoteheight-canonical-property"></a>Propiedad canónica PidLidNoteHeight

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el alto de la ventana del mensaje visible en píxeles. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidNoteHeight  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Note  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008B03  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Nota adhesiva  <br/> |
   
## <a name="remarks"></a>Comentarios

Este valor debe ser mayor que cero.

# <a name="pidtagsupplementaryinfo-canonical-property"></a>Propiedad canónica PidTagSupplementaryInfo

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene información adicional para su uso en un informe.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SUPPLEMENTARY_INFO, PR_SUPPLEMENTARY_INFO_A, PR_SUPPLEMENTARY_INFO_W  <br/> |
|Identificador:  <br/> |0x0C1B  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades contienen información generada por el agente de transferencia de mensajes o el proveedor de transporte relacionados con el informe. Normalmente se usa para el texto del informe de entrega o no entrega que se originó con el sistema de mensajería subyacente.

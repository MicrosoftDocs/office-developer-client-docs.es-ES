# <a name="pidtagpagertelephonenumber-canonical-property"></a>Propiedad canónica PidTagPagerTelephoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono del paginador del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PAGER_TELEPHONE_NUMBER, PR_PAGER_TELEPHONE_NUMBER_A, PR_PAGER_TELEPHONE_NUMBER_W, PR_BEEPER_TELEPHONE_NUMBER, PR_BEEPER_TELEPHONE_NUMBER_A, PR_BEEPER_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A21  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Dirección  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. El destinatario y su organización los definen.

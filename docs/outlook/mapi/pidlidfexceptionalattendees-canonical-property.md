# <a name="pidlidfexceptionalattendees-canonical-property"></a>Propiedad canónica PidLidFExceptionalAttendees

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si esta propiedad es un objeto de calendario periódico con una o más excepciones, y al menos uno de los mensajes incrustados de excepción tiene al menos un RecipientRow.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFExceptionalAttendees  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000822B  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Un valor FALSE, o la ausencia de esta propiedad, indica que el objeto de calendario no tiene excepciones o ninguno de los mensajes incrustados de excepción tiene RecipientRows.

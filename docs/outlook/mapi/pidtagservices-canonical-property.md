# <a name="pidtagservices-canonical-property"></a>Propiedad canónica PidTagServices

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de identificadores de servicios de mensaje en el perfil actual.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SERVICES  <br/> |
|Identificador:  <br/> |0x3D0E  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Perfil MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

No utilice esta propiedad. Está reservado para su uso por MAPI.

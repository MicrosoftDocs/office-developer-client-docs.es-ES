# <a name="pidlidlogdocumentrouted-canonical-property"></a>Propiedad canónica PidLidLogDocumentRouted

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si el documento se envió a un destinatario de distribución durante el registro en diario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLogDocRouted  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Log  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008710  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Diario  <br/> |

<span data-ttu-id="b9ce2-107">Para copiar una o varias entradas en un contenedor modificable, llame al método [IABContainer::CopyEntries](iabcontainer-copyentries.md) del contenedor de destino.</span><span class="sxs-lookup"><span data-stu-id="b9ce2-107">To copy one or more entries into a modifiable container, call the destination container's [IABContainer::CopyEntries](iabcontainer-copyentries.md) method.</span></span> <span data-ttu-id="b9ce2-108">Debido a que puede llevar mucho tiempo copiar entradas de la libreta de direcciones, **CopyEntries** acepta cuatro parámetros de entrada: una matriz de identificadores de entrada para las entradas que se va a copiar, un identificador de ventana, un indicador de progreso y una máscara de bits de indicadores.</span><span class="sxs-lookup"><span data-stu-id="b9ce2-108">Because copying address book entries can be time-consuming, **CopyEntries** accepts four input parameters: an array of entry identifiers for the entries to be copied, a window handle, a progress indicator, and a bitmask of flags.</span></span> 
  
<span data-ttu-id="b9ce2-109">El indicador de progreso y de controlador de ventana usadas por el proveedor de la libreta de direcciones para mostrar el estado de la operación al usuario.</span><span class="sxs-lookup"><span data-stu-id="b9ce2-109">The window handle and progress indicator are used by the address book provider to show the status of the operation to the user.</span></span> <span data-ttu-id="b9ce2-110">Si desea mostrar el progreso, pasar un identificador de ventana para la ventana principal del indicador de progreso en el parámetro _ulUIParam_ y no se establece la marca AB_NO_DIALOG en el parámetro _ulFlags indicado_ .</span><span class="sxs-lookup"><span data-stu-id="b9ce2-110">If you want to display progress, pass a window handle for the parent window of the progress indicator in the  _ulUIParam_ parameter and do not set the AB_NO_DIALOG flag in the  _ulFlags_ parameter.</span></span> <span data-ttu-id="b9ce2-111">Si tiene su propia implementación de un indicador de progreso, pase un puntero a la implementación en el parámetro _lpProgress_ .</span><span class="sxs-lookup"><span data-stu-id="b9ce2-111">If you have your own implementation of a progress indicator, pass a pointer to the implementation in the  _lpProgress_ parameter.</span></span> <span data-ttu-id="b9ce2-112">En caso contrario, pase el valor NULL.</span><span class="sxs-lookup"><span data-stu-id="b9ce2-112">If not, pass NULL.</span></span> <span data-ttu-id="b9ce2-113">El proveedor de la libreta de direcciones utilizará la implementación de indicador de progreso MAPI.</span><span class="sxs-lookup"><span data-stu-id="b9ce2-113">The address book provider will use the MAPI progress indicator implementation.</span></span>

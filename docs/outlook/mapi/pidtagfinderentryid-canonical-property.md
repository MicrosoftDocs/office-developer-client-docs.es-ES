# <a name="pidtagfinderentryid-canonical-property"></a>Propiedad canónica PidTagFinderEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador de entrada de la carpeta donde se crean normalmente los resultados de búsqueda.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FINDER_ENTRYID  <br/> |
|Identificador:  <br/> |0x35E7  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Almacén de mensajes MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

El identificador de entrada contenido en esta propiedad tiene el mismo formato que la estructura de la [propiedad ENTRYID](entryid.md) .

# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Propiedad canónica PidLidOriginalStoreEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el identificador de entrada del almacén del delegado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidOrigStoreEid  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Id. largo (LID):  <br/> |0x00008237  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe establecerse en objetos de reunión creados o actualizados por un delegado.

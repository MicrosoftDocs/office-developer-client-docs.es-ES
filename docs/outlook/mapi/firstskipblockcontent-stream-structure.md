# <a name="firstskipblockcontent-stream-structure"></a>Estructura de la secuencia FirstSkipBlockContent

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
La estructura de la secuencia FirstSkipBlockContent es el contenido de la primera estructura [SkipBlock](skipblock-stream-structure.md) del elemento de datos SkipBlocks de una secuencia [FieldDefinition](fielddefinition-stream-structure.md) . La secuencia FirstSkipBlockContent es simplemente un solo elemento de datos, FieldName: 
  
- FieldName: [PackedUnicodeString](packedunicodestring-stream-structure.md), el nombre del campo.

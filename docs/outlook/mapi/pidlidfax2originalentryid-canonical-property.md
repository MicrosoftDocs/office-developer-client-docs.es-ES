# <a name="pidlidfax2originalentryid-canonical-property"></a>Propiedad canónica PidLidFax2OriginalEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la propiedad EntryID original de dirección de fax del domicilio particular del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax2OriginalEntryID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080C5  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si está presente, debe especificar la propiedad EntryId de uso único que corresponde a esta dirección de fax.

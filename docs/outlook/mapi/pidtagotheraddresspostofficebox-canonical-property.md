# <a name="pidtagotheraddresspostofficebox-canonical-property"></a>Propiedad canónico PidTagOtherAddressPostOfficeBox

  
  
**Se aplica a**: Outlook 
  
Contiene el apartado de correos de un contacto otra dirección.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_POST_OFFICE_BOX, PR_OTHER_ADDRESS_POST_OFFICE_BOX_A, PR_OTHER_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Identificador:  <br/> |0x3A64  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Notas

Estas propiedades son definidas por el usuario o la organización del usuario.

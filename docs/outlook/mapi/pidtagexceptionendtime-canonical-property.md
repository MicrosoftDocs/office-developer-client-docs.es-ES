# <a name="pidtagexceptionendtime-canonical-property"></a>Propiedad canónico PidTagExceptionEndTime

  
  
**Se aplica a**: Outlook 
  
Indica la fecha de finalización y la hora de la excepción en la zona horaria local de la máquina cuando se crea la excepción.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|Identificador:  <br/> |0x7FFC  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Mensaje definido por la clase que no sean transmisible  <br/> |
   
## <a name="remarks"></a>Notas

> [!NOTE]
> Esta propiedad es informativa y no se debe confiar en para información crítica.

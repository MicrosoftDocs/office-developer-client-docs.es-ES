# <a name="ipstoverride1overridepstdisablegrow"></a>IPSTOVERRIDE1::OverridePSTDisableGrow

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Desbloquea un archivo carpetas personales (.pst) para una sesión, reemplazando la directiva PSTDisableGrow.
  
```cpp
HRESULT OverridePSTDisableGrow();
```

## <a name="return-value"></a>Valor devuelto

S_OK
  
> La llamada a la función se ha realizado correctamente.

# <a name="cbspropattrarray"></a>CbSPropAttrArray

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|Archivo de encabezado:  <br/> |IMessage.h  <br/> |
|Estructura relacionado:  <br/> |**SPropAttrArray** <br/> |
   
```cpp

```

## <a name="parameters"></a>Parámetros

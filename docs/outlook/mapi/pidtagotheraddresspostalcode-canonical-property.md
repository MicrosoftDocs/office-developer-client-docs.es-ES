# <a name="pidtagotheraddresspostalcode-canonical-property"></a>Propiedad canónica PidTagOtherAddressPostalCode

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el código postal de la otra dirección postal del usuario de correo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_POSTAL_CODE, PR_OTHER_ADDRESS_POSTAL_CODE_A, PR_OTHER_ADDRESS_POSTAL_CODE_W  <br/> |
|Identificador:  <br/> |0x3A61  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Dirección  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

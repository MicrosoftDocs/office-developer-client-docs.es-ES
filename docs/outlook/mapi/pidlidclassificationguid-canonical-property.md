# <a name="pidlidclassificationguid-canonical-property"></a>Propiedad canónico PidLidClassificationGuid

  
  
**Se aplica a**: Outlook 
  
Especifica un identificador único global (GUID) de una clasificación de mensajes existente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidClassGuid  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000085B8  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |
   
## <a name="remarks"></a>Notas

Para obtener más información, vea [cómo implementar la clasificación de mensaje de Outlook 2007](http://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)

# <a name="pidtagsearchfolderlastused-canonical-property"></a>Propiedad canónica PidTagSearchFolderLastUsed

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Representa la última vez que se obtiene acceso a la carpeta.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_WB_SF_LAST_USED  <br/> |
|Identificador:  <br/> |0x6834  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Búsqueda  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe tener el formato como el número de minutos transcurridos desde la medianoche hora Universal coordinada (UTC) del 1 de enero de 1601.

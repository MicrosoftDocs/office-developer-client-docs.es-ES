# <a name="pidtagphysicalforwardingprohibited-canonical-property"></a>Propiedad canónica PidTagPhysicalForwardingProhibited

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si el remitente de un mensaje prohíbe el reenvío físico de mensajes a un destinatario específico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PHYSICAL_FORWARDING_PROHIBITED  <br/> |
|Identificador:  <br/> |0x0C0F  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

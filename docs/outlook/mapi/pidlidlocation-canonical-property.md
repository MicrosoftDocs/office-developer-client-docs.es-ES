# <a name="pidlidlocation-canonical-property"></a>Propiedad canónica PidLidLocation

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Representa la ubicación de una cita.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLocation  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008208  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Calendario  <br/> |

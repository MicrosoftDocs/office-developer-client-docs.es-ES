# <a name="pidtagspamjunksenders-canonical-property"></a>Propiedad canónica PidTagSpamJunkSenders

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista delimitada por punto y coma de direcciones de correo electrónico y dominios que conforman los remitentes bloqueados.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x041A  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Spam  <br/> |

# <a name="pidlidnotewidth-canonical-property"></a>Propiedad canónica PidLidNoteWidth

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el ancho de la ventana de mensaje visible en píxeles. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidNoteWidth  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Note  <br/> |
|Id. largo (LID):  <br/> |0x00008B02  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Nota pegajosa  <br/> |
   
## <a name="remarks"></a>Comentarios

Este valor debe ser mayor que cero.

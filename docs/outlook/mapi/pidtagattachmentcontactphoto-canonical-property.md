# <a name="pidtagattachmentcontactphoto-canonical-property"></a>Propiedad canónica PidTagAttachmentContactPhoto

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica la existencia de un dato adjunto de foto de un contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ATTACHMENT_CONTACTPHOTO  <br/> |
|Identificador:  <br/> |0x7FFF  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Datos adjuntos del mensaje  <br/> |
   
## <a name="remarks"></a>Comentarios

Valor de esta propiedad debe establecerse en TRUE, y sólo debería haber un dato adjunto en un determinado objeto de contacto.

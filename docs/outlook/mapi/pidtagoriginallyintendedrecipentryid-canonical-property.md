# <a name="pidtagoriginallyintendedrecipentryid-canonical-property"></a>Propiedad canónica PidTagOriginallyIntendedRecipEntryId

  
  
**Hace referencia a**: Outlook 
  
Contiene el identificador de entrada del destinatario de un mensaje reenviado automático originalmente previsto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ENTRYID  <br/> |
|Identificador:  <br/> |0x1012  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es una de las propiedades de dirección para el destinatario del mensaje originalmente. Se debe establecer el agente automática que reenvió el mensaje.
  
Esta propiedad corresponde al atributo X.400 informe por destinatario.

# <a name="pidtagkeyword-canonical-property"></a>Propiedad canónica PidTagKeyword

  
  
**Hace referencia a**: Outlook 
  
Contiene una palabra clave que identifica al destinatario al administrador del sistema del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Identificador:  <br/> |0x3A0B  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y su organización.

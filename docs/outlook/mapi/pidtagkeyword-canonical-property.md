# <a name="pidtagkeyword-canonical-property"></a>Propiedad canónica PidTagKeyword

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una palabra clave que identifica al destinatario en el administrador del sistema del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Identificador:  <br/> |0x3A0B  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Los define el destinatario y su organización.

# <a name="pidtagattachmentflags-canonical-property"></a>Propiedad canónica PidTagAttachmentFlags

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica un control especial para este objeto Attachment.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ATTACHMENT_FLAGS  <br/> |
|Identificador:  <br/> |0x7FFD  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Datos adjuntos del mensaje  <br/> |
   
## <a name="remarks"></a>Comentarios

Debe ser 0x00000000, a menos que lo invalide otro protocolo que extienda el Protocolo de objetos de mensaje y datos adjuntos como se indica en [[MS-OXCMSG]](https://msdn.microsoft.com/library/7fd7ec40-deec-4c06-9493-1bc06b349682%28Office.15%29.aspx)

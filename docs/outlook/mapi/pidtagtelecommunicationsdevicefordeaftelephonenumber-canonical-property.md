# <a name="pidtagtelecommunicationsdevicefordeaftelephonenumber-canonical-property"></a>Propiedad canónica PidTagTelecommunicationsDeviceForDeafTelephoneNumber

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el dispositivo de telecomunicaciones del usuario de mensajería para los usuarios con discapacidades auditivas () teléfono TTYTDD.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A4B  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información acerca de un usuario de mensajería. El contenido es definido por el usuario de mensajería y organización de mensajería del usuario.

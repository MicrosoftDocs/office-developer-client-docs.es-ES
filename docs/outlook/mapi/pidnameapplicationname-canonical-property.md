# <a name="pidnameapplicationname-canonical-property"></a>Propiedad canónica PidNameApplicationName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la aplicación que puede abrir el archivo adjunto al documento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |AppName  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Común  <br/> |

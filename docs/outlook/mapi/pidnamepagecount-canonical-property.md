# <a name="pidnamepagecount-canonical-property"></a>Propiedad canónica PidNamePageCount

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el recuento de páginas del archivo adjunto al documento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |PageCount  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Común  <br/> |

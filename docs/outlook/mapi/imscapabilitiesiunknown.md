# <a name="imscapabilities--iunknown"></a>IMSCapabilities : IUnknown

  
  
**Hace referencia a**: Outlook 
  
Proporciona información sobre lo que puede admitir un almacén.
  
|||
|:-----|:-----|
|Suministrado por:  <br/> |Proveedor de almacén de mensajes  <br/> |
|Identificador de interfaz:  <br/> |IID_IMSCapabilities  <br/> |
   
## <a name="vtable-order"></a>Orden vtable

|||
|:-----|:-----|
|**[GetCapabilities](imscapabilities-getcapabilities.md)** <br/> |Devuelve las capacidades de un almacén en función de la selección especificada.  <br/> |
   
## <a name="remarks"></a>Comentarios

 **IMSCapabilities** proporciona información sobre lo que puede admitir un almacén si el almacén es compatible con esta interfaz.

# <a name="pidlidworkaddressstreet-canonical-property"></a>Propiedad canónica PidLidWorkAddressStreet

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la calle de la dirección de correo del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressStreet  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008045  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |

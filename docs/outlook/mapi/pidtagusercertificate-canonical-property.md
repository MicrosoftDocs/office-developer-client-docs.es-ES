# <a name="pidtagusercertificate-canonical-property"></a>Propiedad canónica PidTagUserCertificate

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un certificado de autenticación ASN.1 para un usuario de mensajería. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_USER_CERTIFICATE  <br/> |
|Identificador:  <br/> |0x3A22  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Un certificado de autenticación es similar a una firma digital. Varias propiedades MAPI suministran certificados ASN.1.

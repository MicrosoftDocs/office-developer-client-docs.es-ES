# <a name="pidtagcartelephonenumber-canonical-property"></a>Propiedad canónica PidTagCarTelephoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CAR_TELEPHONE_NUMBER, PR_CAR_TELEPHONE_NUMBER_A, PR_CAR_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A1E  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso sobre un destinatario. El destinatario y la organización del destinatario las definen.

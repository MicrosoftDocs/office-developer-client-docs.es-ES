# <a name="pidtagfreebusyentryids-canonical-property"></a>Propiedad canónica PidTagFreeBusyEntryIds

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene los **EntryIDs** para el mensaje de información de delegado, el mensaje de disponibilidad del usuario que ha iniciado sesión y la carpeta cuyo **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) es igual a "FreeBusy Data".
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FREEBUSY_ENTRYIDS  <br/> |
|Identificador:  <br/> |0x36E4  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Contenedor MAPI  <br/> |

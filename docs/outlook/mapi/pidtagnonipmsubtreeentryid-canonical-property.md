# <a name="pidtagnonipmsubtreeentryid-canonical-property"></a>Propiedad canónica PidTagNonIpmSubtreeEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Muestra el identificador de entrada de **la carpeta NON_IPM_SUBTREE** en un Exchange de carpetas públicas. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_NON_IPM_SUBTREE_ENTRYID  <br/> |
|Identificador:  <br/> |0x6620  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Almacén de mensajes  <br/> |

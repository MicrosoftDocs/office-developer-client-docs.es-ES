# <a name="pidtagreplytemplateid-canonical-property"></a>Propiedad canónica PidTagReplyTemplateId

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador único global (GUID) para la plantilla de respuesta.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPLY_TEMPLATE_ID  <br/> |
|Identificador:  <br/> |0x65C2  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reglas  <br/> |

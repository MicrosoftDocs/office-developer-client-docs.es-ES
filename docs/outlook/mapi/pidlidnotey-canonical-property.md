# <a name="pidlidnotey-canonical-property"></a>Propiedad canónico PidLidNoteY

  
  
**Se aplica a**: Outlook 
  
Especifica la distancia, en píxeles, desde el borde superior de la pantalla donde una interfaz de usuario muestra una nota.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidNoteY  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Note  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008B05  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Nota rápida  <br/> |

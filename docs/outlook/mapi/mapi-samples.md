# <a name="mapi-samples"></a>Ejemplos de MAPI

**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Esta sección contiene ejemplos de código que muestran cómo conseguir determinadas tareas o implementar ciertos proveedores de servicios con MAPI de Outlook. MFCMAPI también se usa en esta referencia para mostrar el uso de varias API. Para ejecutar cualquiera de los ejemplos en los sistemas operativos Windows Vista, Windows Server 2008, Windows XP o Windows Server 2003, primero debe configurar el sistema de tiempo de ejecución MAPI. Para obtener más información, consulte [Información general de programación de MAPI](mapi-programming-overview.md).
  
## <a name="in-this-section"></a>En esta sección

- [MFCMAPI como un ejemplo de código](mfcmapi-as-a-code-sample.md): describe cómo descargar y usar MFCMAPI como una aplicación de ejemplo para esta referencia.
    
- [Muestras de proveedor de servicios MAPI](mapi-service-provider-samples.md): muestra ejemplos de proveedores de servicios MAPI.
    
- [Crear elementos de Outlook usando MAPI](creating-outlook-items-by-using-mapi.md): muestra ejemplos de código de cómo crear elementos de Outlook con MAPI.

# <a name="mapi-samples"></a>Ejemplos de MAPI

**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Esta sección contiene ejemplos de código que muestran cómo realizar ciertas tareas o implementar determinados proveedores de servicios de MAPI. MFCMAPI también se usa en esta referencia para demostrar el uso de varias API. Para ejecutar cualquiera de los ejemplos en Windows Vista, Windows Server 2008, Windows XP o sistemas operativos Windows Server 2003, debe configurar el sistema de tiempo de ejecución MAPI. Para obtener más información, vea [Información general de programación de MAPI](mapi-programming-overview.md).

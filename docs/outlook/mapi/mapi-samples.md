**Hace referencia a**: Outlook 
  
Esta sección contiene ejemplos de código que muestran cómo realizar ciertas tareas o implementar determinados proveedores de servicios de MAPI. MFCMAPI también se usa en esta referencia para demostrar el uso de varias API.

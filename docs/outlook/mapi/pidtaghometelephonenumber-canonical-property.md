# <a name="pidtaghometelephonenumber-canonical-property"></a>Propiedad canónica PidTagHomeTelephoneNumber

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono principal de la página principal del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_TELEPHONE_NUMBER, PR_HOME_TELEPHONE_NUMBER_A, PR_HOME_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A09  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan la identificación y el acceso de un destinatario. Se definen por el destinatario y su organización.

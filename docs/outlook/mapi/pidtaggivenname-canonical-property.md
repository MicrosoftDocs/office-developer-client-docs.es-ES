# <a name="pidtaggivenname-canonical-property"></a>Propiedad canónica PidTagGivenName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el primer nombre o el nombre dado del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_GIVEN_NAME, PR_GIVEN_NAME_A, PR_GIVEN_NAME_W  <br/> |
|Identificador:  <br/> |0x3A06  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso sobre un destinatario. El destinatario y su organización los definen.

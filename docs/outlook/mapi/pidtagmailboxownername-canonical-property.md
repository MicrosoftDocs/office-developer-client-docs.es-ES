# <a name="pidtagmailboxownername-canonical-property"></a>Propiedad canónica PidTagMailboxOwnerName

  
  
**Hace referencia a**: Outlook 
  
Muestra el nombre del propietario del buzón.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MAILBOX_OWNER_NAME, PR_MAILBOX_OWNER_NAME_A, PR_MAILBOX_OWNER_NAME_W  <br/> |
|Identificador:  <br/> |0x001F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_UNICODE  <br/> |
|Área:  <br/> |Almacén de mensajes  <br/> |

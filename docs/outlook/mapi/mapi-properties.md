# <a name="mapi-properties"></a>Propiedades MAPI

 
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Esta sección contiene propiedades canónicas que son compatibles con la API de mensajería (MAPI) de Microsoft Outlook. Cada propiedad canónica corresponde a una o varias propiedades MAPI relacionadas. Para obtener más información sobre las propiedades canónicas, vea [Propiedades canónicas MAPI](mapi-canonical-properties.md). Para obtener una explicación general de cómo funciona MAPI con las propiedades y una lista de las propiedades disponibles para cada objeto, consulte [Información general de las propiedades MAPI](mapi-property-overview.md). Para obtener información sobre las macros relacionadas con la propiedad, consulte la página de referencia de la estructura [SPropValue](spropvalue.md). Para obtener una lista completa de propiedades MAPI y sus asignaciones a los nombres de propiedad canónicos, consulte [Asignar nombres MAPI a los nombres de propiedad canónicos](mapping-mapi-names-to-canonical-property-names.md) y [Asignar nombres de propiedad canónicos a nombres MAPI](mapping-canonical-property-names-to-mapi-names.md). 
  
Nota: A diferencia de los miembros de matrices variantes OLE, todos los miembros de una matriz de propiedad multivalor MAPI son del mismo tipo.

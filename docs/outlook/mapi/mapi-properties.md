# <a name="mapi-properties"></a>Propiedades MAPI

 
  
**Hace referencia a**: Outlook 
  
Esta sección contiene las propiedades canónicas que se admiten por Microsoft Outlook Messaging API (MAPI). Cada propiedad canónico corresponde a una o varias propiedades relacionadas de MAPI. Para obtener más información sobre las propiedades canónicas, vea [Las propiedades canónico de MAPI](mapi-canonical-properties.md). Para obtener una explicación general de cómo funciona la MAPI con propiedades y una lista de las propiedades disponibles para cada objeto, vea [Información general sobre la propiedad de MAPI](mapi-property-overview.md). Para obtener información acerca de las macros relacionadas con la propiedad, vea la página de referencia de la estructura [SPropValue](spropvalue.md) . Para obtener una lista completa de las propiedades MAPI y sus asignaciones de nombres de propiedad canónico, vea [Asignación de nombres de MAPI para los nombres de propiedad canónico](mapping-mapi-names-to-canonical-property-names.md) y [Asignación de nombres de propiedad canónico a nombres de MAPI](mapping-canonical-property-names-to-mapi-names.md). 
  
Nota: A diferencia de los miembros de matrices variantes OLE, todos los miembros de una matriz de varios valores de propiedad MAPI es del mismo tipo.

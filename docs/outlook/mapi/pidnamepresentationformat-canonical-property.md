# <a name="pidnamepresentationformat-canonical-property"></a>Propiedad canónica PidNamePresentationFormat

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el formato de presentación del archivo que se adjunta al documento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguna  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |PresFormat  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Common  <br/> |

# <a name="pidtagphysicaldeliveryreportrequest-canonical-property"></a>Propiedad canónica PidTagPhysicalDeliveryReportRequest

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el modo de un informe que se entregará a un destinatario de mensaje determinado al finalizar la entrega o entrega de mensajes físicos por parte del sistema de control de mensajes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PHYSICAL_DELIVERY_REPORT_REQUEST  <br/> |
|Identificador:  <br/> |0x0C0C  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

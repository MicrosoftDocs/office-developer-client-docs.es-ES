# <a name="pidtagcontroltype-canonical-property"></a>Propiedad canónica PidTagControlType

  
  
**Hace referencia a**: Outlook 
  
Contiene un valor que indica un tipo de control para un control que se utiliza en un cuadro de diálogo. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTROL_TYPE  <br/> |
|Identificador:  <br/> |0x3F02  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Tabla MAPI para mostrar  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad puede tener exactamente uno de los siguientes valores:
  
DTCT_BUTTON 
  
> Un control de botón de cuadro de diálogo.
    
DTCT_CHECKBOX 
  
> Una casilla de verificación del cuadro de diálogo.
    
DTCT_COMBOBOX 
  
> Un cuadro de diálogo combinado.
    
DTCT_DDLBX 
  
> Un cuadro de lista desplegable del cuadro de diálogo.
    
DTCT_EDIT 
  
> Un cuadro de diálogo Editar texto.
    
DTCT_GROUPBOX 
  
> Un cuadro de diálogo grupo.
    
DTCT_LABEL 
  
> Una etiqueta del cuadro de diálogo.
    
DTCT_LBX 
  
> Un cuadro de lista del cuadro de diálogo.
    
DTCT_LISTBOX 
  
> Un cuadro de lista del cuadro de diálogo.
    
DTCT_MVDDLBX 
  
> Un cuadro de lista multivalor rellenado por una propiedad con varios valores de tipo string.
    
DTCT_PAGE 
  
> Una página con fichas del cuadro de diálogo.
    
DTCT_RADIOBUTTON 
  
> Un botón de opción del cuadro de diálogo.

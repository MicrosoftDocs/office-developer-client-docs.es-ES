# <a name="pidlidnotecolor-canonical-property"></a>Propiedad canónica PidLidNoteColor

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el color de fondo sugerido de la nota. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidNoteColor  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Note  <br/> |
|Id. largo (LID):  <br/> |0x00008B00  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Notas rápidas  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe ser una de las entradas de la tabla siguiente:
  
|**Valor**|**Color**|
|:-----|:-----|
|0x00000000  <br/> |Azul  <br/> |
|0x00000001  <br/> |Verde  <br/> |
|0x00000002  <br/> |Rosa  <br/> |
|0x00000003  <br/> |Amarillo  <br/> |
|0x00000004  <br/> |Blanco  <br/> |

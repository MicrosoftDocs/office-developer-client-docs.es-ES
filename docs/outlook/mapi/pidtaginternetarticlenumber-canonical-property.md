# <a name="pidtaginternetarticlenumber-canonical-property"></a>Propiedad canónica PidTagInternetArticleNumber

  
  
**Hace referencia a**: Outlook 
  
Un número asociado a un elemento en un almacén de mensajes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INTERNET_ARTICLE_NUMBER  <br/> |
|Identificador:  <br/> |0x0E23  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Comentarios

En los almacenes de mensajes IMAPI, esta propiedad corresponde a la UID IMAPI.

# <a name="pidnamexsharingremotetype-canonical-property"></a>Propiedad canónica PidNameXSharingRemoteType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Representa el valor de la **propiedad dispidSharingRemoteType** ([PidLidSharingRemoteType](pidlidsharingremotetype-canonical-property.md)).
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |X-Sharing-Remote-Type  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Compartir  <br/> |

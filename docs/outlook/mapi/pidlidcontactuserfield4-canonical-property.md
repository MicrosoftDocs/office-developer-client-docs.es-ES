# <a name="pidlidcontactuserfield4-canonical-property"></a>Propiedad canónico PidLidContactUserField4

  
  
**Se aplica a**: Outlook 
  
Especifica el cuarto campo en el contacto al que está destinado a diversos usos del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactUserField4  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008052  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

Esta propiedad contiene texto que no está relacionado con cualquier otra propiedad específicos de los contactos.

# <a name="pidlidoldlocation-canonical-property"></a>Propiedad canónica PidLidOldLocation

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica el valor original de la **propiedad dispidLocation** ([PidLidLocation](pidlidlocation-canonical-property.md)) antes de una actualización de reunión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidOldLocation  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Meeting  <br/> |
|Id. largo (LID):  <br/> |0x00000028  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad no es necesaria.

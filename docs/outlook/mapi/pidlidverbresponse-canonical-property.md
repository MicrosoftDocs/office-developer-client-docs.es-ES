# <a name="pidlidverbresponse-canonical-property"></a>Propiedad canónica PidLidVerbResponse

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la opción de votación que cuestionario seleccionado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidVerbResponse  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008524  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |
   
## <a name="remarks"></a>Comentarios

Normalmente, esta propiedad se establece en uno de los valores delimitados que están contenidos en la propiedad **dispidVerbStream** ([PidLidVerbStream](pidlidverbstream-canonical-property.md)) en el que se aprueba el demandado.

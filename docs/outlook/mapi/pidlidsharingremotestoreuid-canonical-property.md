# <a name="pidlidsharingremotestoreuid-canonical-property"></a>Propiedad canónica PidLidSharingRemoteStoreUid

  
  
**Hace referencia a**: Outlook 
  
Especifica la representación de cadena hexadecimal del valor de la propiedad **PR_STORE_ENTRYID** ([PidTagStoreEntryId](pidtagstoreentryid-canonical-property.md)) en la carpeta compartida. Esto es una propiedad de un mensaje para compartir.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidSharingRemoteStoreUid  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Sharing  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008A48  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Uso compartido  <br/> |

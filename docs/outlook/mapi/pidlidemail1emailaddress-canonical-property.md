# <a name="pidlidemail1emailaddress-canonical-property"></a>Propiedad canónica PidLidEmail1EmailAddress

  
  
**Hace referencia a**: Outlook 
  
Especifica la primera dirección de correo electrónico del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail1EmailAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008083  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad debe ser apropiado para el tipo de dirección especificado para esta dirección de correo electrónico.

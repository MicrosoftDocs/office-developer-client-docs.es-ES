# <a name="pidtagspamtrustedsenders-canonical-property"></a>Propiedad canónica PidTagSpamTrustedSenders

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista delimitada por punto y coma de direcciones de correo electrónico y dominios que se componen de remitentes de confianza.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x0418  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Correo no deseado  <br/> |

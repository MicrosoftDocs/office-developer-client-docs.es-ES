# <a name="pidtagspamtrustedsenders-canonical-property"></a>Propiedad canónico PidTagSpamTrustedSenders

  
  
**Se aplica a**: Outlook 
  
Contiene una lista delimitada por punto y coma de direcciones de correo electrónico y los dominios que se componen de remitentes de confianza.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0418  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Spam  <br/> |

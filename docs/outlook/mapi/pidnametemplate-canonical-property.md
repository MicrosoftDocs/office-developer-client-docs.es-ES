# <a name="pidnametemplate-canonical-property"></a>Propiedad canónica PidNameTemplate

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la plantilla del archivo que se adjunta al documento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguna  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |Template  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Common  <br/> |

# <a name="pidlidemail2emailaddress-canonical-property"></a>Propiedad canónico PidLidEmail2EmailAddress

  
  
**Se aplica a**: Outlook 
  
Especifica la segunda dirección de correo electrónico del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail2EmailAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008093  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

El valor de esta propiedad debe ser apropiado para el tipo de dirección especificado para esta dirección de correo electrónico.

# <a name="pidtagproofofdeliveryrequested-canonical-property"></a>Propiedad canónica PidTagProofOfDeliveryRequested

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Esta propiedad contiene TRUE si un remitente del mensaje solicita una prueba de entrega para un destinatario determinado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PROOF_OF_DELIVERY_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C12  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

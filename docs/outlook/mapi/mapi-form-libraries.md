# <a name="mapi-form-libraries"></a>Bibliotecas de formularios MAPI

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Para ayudar a organizar los servidores de formulario y hacerlos fácilmente disponibles para las aplicaciones cliente, la arquitectura de formulario MAPI incluye bibliotecas de formularios, que admiten la instalación, administración, replicación y uso de servidores de formularios.
  
Las bibliotecas de formularios son repositorios de servidores de formulario e información de configuración sobre ellos que el administrador de formularios usa al cargar servidores de formulario. 
  
> [!NOTE]
> Al principio de la historia de MAPI, las bibliotecas de formularios se denominaban "registros". Debido a esto, el término "registro" ha persistido en algunas interfaces MAPI relacionadas con formularios, propiedades, entradas de archivo de configuración, etc. Si la interfaz de la aplicación requiere el uso de uno de estos términos, debe usar el término "biblioteca" exclusivamente.

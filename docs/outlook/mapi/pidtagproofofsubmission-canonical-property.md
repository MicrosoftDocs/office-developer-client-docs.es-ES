# <a name="pidtagproofofsubmission-canonical-property"></a>Propiedad canónica PidTagProofOfSubmission

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Esta propiedad contiene TRUE si el remitente de un mensaje solicita la prueba de que el sistema de transferencia de mensajes ha enviado un mensaje para su entrega al destinatario previsto originalmente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PROOF_OF_SUBMISSION  <br/> |
|Identificador:  <br/> |0x0E26  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |MAPI no transmitible  <br/> |

# <a name="pidtagweddinganniversary-canonical-property"></a>Propiedad canónica PidTagWeddingAnniversary

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la fecha del aniversario de un usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_WEDDING_ANNIVERSARY  <br/> |
|Identificador:  <br/> |0x3A41  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad la define el usuario o la organización del usuario.

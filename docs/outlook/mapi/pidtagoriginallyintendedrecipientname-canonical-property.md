# <a name="pidtagoriginallyintendedrecipientname-canonical-property"></a>Propiedad canónica PidTagOriginallyIntendedRecipientName

  
  
**Hace referencia a**: Outlook 
  
Contiene el nombre codificado del destinatario de un mensaje de autoforwarded originalmente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIPIENT_NAME  <br/> |
|Identificador:  <br/> |0 x 0020  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentarios

La propiedad **PR_ORIGINALLY_INTENDED_RECIPIENT_NAME** se debe establecer el agente automática que reenvió el mensaje.

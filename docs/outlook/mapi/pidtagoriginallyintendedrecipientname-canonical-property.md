# <a name="pidtagoriginallyintendedrecipientname-canonical-property"></a>Propiedad canónica PidTagOriginallyIntendedRecipientName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre codificado del destinatario previsto originalmente de un mensaje reenviado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIPIENT_NAME  <br/> |
|Identificador:  <br/> |0x0020  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentarios

La propiedad **PR_ORIGINALLY_INTENDED_RECIPIENT_NAME** debe establecerse mediante el agente automático que ha reenviado el mensaje.

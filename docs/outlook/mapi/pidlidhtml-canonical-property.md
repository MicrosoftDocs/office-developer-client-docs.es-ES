# <a name="pidlidhtml-canonical-property"></a>Propiedad canónica PidLidHtml

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la dirección URL de página Web de negocios del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidHTML  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000802B  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad, si está presente, debe ser el mismo que el valor de la propiedad **PR_BUSINESS_HOME_PAGE** ([PidTagBusinessHomePage](pidtagbusinesshomepage-canonical-property.md)).

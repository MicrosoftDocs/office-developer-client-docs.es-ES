# <a name="pidlidworkaddresspostalcode-canonical-property"></a>Propiedad canónica PidLidWorkAddressPostalCode

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte del código postal (código postal) de la dirección de trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressPostalCode  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008048  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

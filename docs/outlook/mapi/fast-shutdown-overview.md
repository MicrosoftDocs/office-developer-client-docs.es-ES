# <a name="fast-shutdown-overview"></a><span data-ttu-id="ba48d-103">Introducción al apagado rápido</span><span class="sxs-lookup"><span data-stu-id="ba48d-103">Fast Shutdown Overview</span></span>

<span data-ttu-id="ba48d-104">**Se aplica a**: Outlook 2013 | Outlook 2016</span><span class="sxs-lookup"><span data-stu-id="ba48d-104">**Applies to**: Outlook 2013 | Outlook 2016</span></span> 
  
<span data-ttu-id="ba48d-105">El apagado rápido es un mecanismo para que un cliente MAPI inicie un apagado rápido del proceso de cliente, notificando a todos los proveedores con los que el cliente tiene una sesión MAPI activa para guardar los datos y la configuración antes de que salga el proceso de cliente.</span><span class="sxs-lookup"><span data-stu-id="ba48d-105">Fast shutdown is a mechanism for a MAPI client to initiate a quick shutdown of the client process, notifying all providers with which the client has an active MAPI session to save data and settings before the client process exits.</span></span> <span data-ttu-id="ba48d-106">En este tema se describe el mecanismo básico del apagado rápido.</span><span class="sxs-lookup"><span data-stu-id="ba48d-106">This topic describes the basic mechanism of fast shutdown.</span></span> 

<span data-ttu-id="ba48d-107">A partir Microsoft Outlook 2010 y ahora incluyendo Microsoft Outlook 2013, el subsistema MAPI proporciona la [interfaz IMAPIClientShutdown : IUnknown.](imapiclientshutdowniunknown.md)</span><span class="sxs-lookup"><span data-stu-id="ba48d-107">Starting in Microsoft Outlook 2010 and now including Microsoft Outlook 2013, the MAPI subsystem provides the [IMAPIClientShutdown : IUnknown](imapiclientshutdowniunknown.md) interface.</span></span> <span data-ttu-id="ba48d-108">Outlook y otros clientes MAPI pueden adoptar el apagado rápido como mecanismo predeterminado para salir del proceso de cliente.</span><span class="sxs-lookup"><span data-stu-id="ba48d-108">Outlook and other MAPI clients can adopt fast shutdown as the default mechanism to exit the client process.</span></span> <span data-ttu-id="ba48d-109">Una configuración de nivel de usuario en el registro Windows del equipo cliente controla la adopción del apagado rápido para todos los clientes MAPI para ese usuario en ese equipo.</span><span class="sxs-lookup"><span data-stu-id="ba48d-109">A user-level setting in the Windows registry of the client computer controls the adoption of fast shutdown for all MAPI clients for that user on that computer.</span></span> <span data-ttu-id="ba48d-110">Para obtener más información sobre la configuración del Registro, [vea Fast Shutdown User Options](fast-shutdown-user-options.md).</span><span class="sxs-lookup"><span data-stu-id="ba48d-110">For details about the registry settings, see [Fast Shutdown User Options](fast-shutdown-user-options.md).</span></span>
  
<span data-ttu-id="ba48d-111">Si un cliente MAPI necesita adoptar el apagado rápido, debe usar la interfaz **IMAPIClientShutdown : IUnknown.**</span><span class="sxs-lookup"><span data-stu-id="ba48d-111">If a MAPI client needs to adopt fast shutdown, it must use the **IMAPIClientShutdown : IUnknown** interface.</span></span> <span data-ttu-id="ba48d-112">A continuación se muestra el curso típico de los eventos cuando el cliente intenta apagar:</span><span class="sxs-lookup"><span data-stu-id="ba48d-112">The following is the typical course of events when the client attempts to shut down:</span></span> 
  
1. <span data-ttu-id="ba48d-113">El cliente MAPI inicia el cierre llamando al método [IMAPIClientShutdown::QueryFastShutdown](imapiclientshutdown-queryfastshutdown.md) para determinar si el subsistema MAPI admite el apagado rápido.</span><span class="sxs-lookup"><span data-stu-id="ba48d-113">The MAPI client initiates the shutdown by calling the [IMAPIClientShutdown::QueryFastShutdown](imapiclientshutdown-queryfastshutdown.md) method to determine whether the MAPI subsystem supports fast shutdown.</span></span> 
    
2. <span data-ttu-id="ba48d-114">El subsistema MAPI responde con la compatibilidad de apagado rápido disponible a la llamada **IMAPIClientShutdown::QueryFastShutdown** del cliente mediante el procedimiento siguiente:</span><span class="sxs-lookup"><span data-stu-id="ba48d-114">The MAPI subsystem responds with the available fast shutdown support to the client's **IMAPIClientShutdown::QueryFastShutdown** call by using the following procedure:</span></span> 
    
    1. <span data-ttu-id="ba48d-115">El subsistema MAPI llama al método [IMAPIProviderShutdown::QueryFastShutdown](imapiprovidershutdown-queryfastshutdown.md) para cada proveedor MAPI con el que el proceso de cliente MAPI tenga una sesión MAPI activa, si el proveedor ha implementado la [interfaz IMAPIProviderShutdown : IUnknown.](imapiprovidershutdowniunknown.md)</span><span class="sxs-lookup"><span data-stu-id="ba48d-115">The MAPI subsystem calls the [IMAPIProviderShutdown::QueryFastShutdown](imapiprovidershutdown-queryfastshutdown.md) method for each MAPI provider with which the MAPI client process has an active MAPI session, if the provider has implemented the [IMAPIProviderShutdown : IUnknown](imapiprovidershutdowniunknown.md) interface.</span></span> 
        
       > [!NOTE]
       >  <span data-ttu-id="ba48d-116">El subsistema MAPI siempre consulta y notifica a los proveedores MAPI a través de la interfaz **IMAPIProviderShutdown : IUnknown** dentro de cada sesión MAPI en el siguiente orden:</span><span class="sxs-lookup"><span data-stu-id="ba48d-116">The MAPI subsystem always queries and notifies MAPI providers through the **IMAPIProviderShutdown : IUnknown** interface within each MAPI session in the following order:</span></span>
       > 1. <span data-ttu-id="ba48d-117">Proveedores de transporte</span><span class="sxs-lookup"><span data-stu-id="ba48d-117">Transport providers</span></span>
       > 2. <span data-ttu-id="ba48d-118">Proveedores de libreta de direcciones</span><span class="sxs-lookup"><span data-stu-id="ba48d-118">Address book providers</span></span>
       > 3. <span data-ttu-id="ba48d-119">Proveedores de la tienda</span><span class="sxs-lookup"><span data-stu-id="ba48d-119">Store providers</span></span> 
    
    2. <span data-ttu-id="ba48d-120">Según la configuración del Registro de cierre rápido para ese usuario en el equipo cliente, el subsistema MAPI especifica el código de devolución adecuado a **IMAPIClientShutdown::QueryFastShutdown**.</span><span class="sxs-lookup"><span data-stu-id="ba48d-120">Depending on the fast shutdown registry setting for that user on the client computer, the MAPI subsystem specifies the appropriate return code to **IMAPIClientShutdown::QueryFastShutdown**.</span></span> <span data-ttu-id="ba48d-121">El código devuelto es S_OK o MAPI_E_NO_SUPPORT.</span><span class="sxs-lookup"><span data-stu-id="ba48d-121">The return code is either S_OK or MAPI_E_NO_SUPPORT.</span></span>
        
    3. <span data-ttu-id="ba48d-122">El cliente MAPI llama al método [IMAPIClientShutdown::NotifyProcessShutdown](imapiclientshutdown-notifyprocessshutdown.md) para indicar al subsistema MAPI la intención de cerrar.</span><span class="sxs-lookup"><span data-stu-id="ba48d-122">The MAPI client calls the [IMAPIClientShutdown::NotifyProcessShutdown](imapiclientshutdown-notifyprocessshutdown.md) method to indicate to the MAPI subsystem the intention to shut down.</span></span> 
        
    4. <span data-ttu-id="ba48d-123">El subsistema MAPI indica a cada proveedor MAPI cargado que el cliente MAPI se apagará.</span><span class="sxs-lookup"><span data-stu-id="ba48d-123">The MAPI subsystem indicates to each loaded MAPI provider that the MAPI client will shut down.</span></span> <span data-ttu-id="ba48d-124">Para los proveedores que han implementado la interfaz **IMAPIProviderShutdown : IUnknown,** el subsistema MAPI llama al método [IMAPIProviderShutdown::NotifyProcessShutdown](imapiprovidershutdown-notifyprocessshutdown.md) correspondiente.</span><span class="sxs-lookup"><span data-stu-id="ba48d-124">For those providers that have implemented the **IMAPIProviderShutdown : IUnknown** interface, the MAPI subsystem calls the corresponding [IMAPIProviderShutdown::NotifyProcessShutdown](imapiprovidershutdown-notifyprocessshutdown.md) method.</span></span> 
        
    5. <span data-ttu-id="ba48d-125">El cliente MAPI llama al método [IMAPIClientShutdown::D oFastShutdown](imapiclientshutdown-dofastshutdown.md) para indicar al subsistema MAPI que el proceso de cliente está saliendo inmediatamente.</span><span class="sxs-lookup"><span data-stu-id="ba48d-125">The MAPI client calls the [IMAPIClientShutdown::DoFastShutdown](imapiclientshutdown-dofastshutdown.md) method to indicate to the MAPI subsystem that the client process is exiting immediately.</span></span> 
        
    6. <span data-ttu-id="ba48d-126">El subsistema MAPI indica a cada proveedor MAPI cargado que el proceso de cliente MAPI está saliendo.</span><span class="sxs-lookup"><span data-stu-id="ba48d-126">The MAPI subsystem indicates to each loaded MAPI provider that the MAPI client process is exiting.</span></span> <span data-ttu-id="ba48d-127">Para los proveedores que han implementado la interfaz **IMAPIProviderShutdown : IUnknown,** el subsistema MAPI llama al método [IMAPIProviderShutdown::D oFastShutdown](imapiprovidershutdown-dofastshutdown.md) correspondiente.</span><span class="sxs-lookup"><span data-stu-id="ba48d-127">For those providers that have implemented the **IMAPIProviderShutdown : IUnknown** interface, the MAPI subsystem calls the corresponding [IMAPIProviderShutdown::DoFastShutdown](imapiprovidershutdown-dofastshutdown.md) method.</span></span> <span data-ttu-id="ba48d-128">En este punto, estos proveedores MAPI deben comprobar que todas las acciones necesarias, como guardar datos y opciones de configuración, se completen para que el cliente MAPI desconecte inmediatamente todas las referencias y salga.</span><span class="sxs-lookup"><span data-stu-id="ba48d-128">At this point, these MAPI providers should verify that all necessary actions, such as saving data and settings, are complete in preparation for the MAPI client to immediately disconnect all references and exit.</span></span>

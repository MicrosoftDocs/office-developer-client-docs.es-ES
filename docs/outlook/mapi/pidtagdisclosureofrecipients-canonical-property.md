# <a name="pidtagdisclosureofrecipients-canonical-property"></a>Propiedad canónica PidTagDisclosureOfRecipients

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si se permite la revelación de destinatarios.

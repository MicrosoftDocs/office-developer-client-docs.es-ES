# <a name="pidlidcontactlinkentry-canonical-property"></a>Propiedad canónica PidLidContactLinkEntry

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la lista de **identificadores** de la libreta de direcciones vinculado a este objeto de mensaje. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactLinkEntry  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008585  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contacto  <br/> |

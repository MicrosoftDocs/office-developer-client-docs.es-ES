# <a name="pidnamexsharingcapabilities-canonical-property"></a>Propiedad canónica PidNameXSharingCapabilities

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un valor que indica las capacidades de uso compartido del elemento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |X-Sharing-Capabilities  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Compartir  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene un valor que indica las capacidades de uso compartido del objeto message.

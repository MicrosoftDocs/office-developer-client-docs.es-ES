# <a name="pidnamexsharingcapabilities-canonical-property"></a>Propiedad canónica PidNameXSharingCapabilities

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene un valor que indica las capacidades de uso compartidas del elemento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |Capacidades de uso compartido de X  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Uso compartido  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene un valor que indica las capacidades de uso compartidas del objeto de mensaje.

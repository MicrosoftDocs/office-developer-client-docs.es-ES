# <a name="pidtagx400contenttype-canonical-property"></a>Propiedad canónico PidTagX400ContentType

  
  
**Se aplica a**: Outlook 
  
Esta propiedad contiene el tipo de contenido de un mensaje enviado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_X400_CONTENT_TYPE  <br/> |
|Identificador:  <br/> |0x003C  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |X400  <br/> |

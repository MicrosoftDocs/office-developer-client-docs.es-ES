# <a name="pidtagrelatedmessageids-canonical-property"></a>Propiedad canónica PidTagRelatedMessageIds

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de identificadores de los mensajes a la que está relacionada con un mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RELATED_IPMS  <br/> |
|Identificador:  <br/> |0x002D  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Sobres MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Los identificadores de usar las mismas reglas de construcción específico tal como se utiliza para la propiedad **PR_SEARCH_KEY** ([PidTagSearchKey](pidtagsearchkey-canonical-property.md)).

# <a name="pidtagrelatedmessageids-canonical-property"></a>Propiedad canónica PidTagRelatedMessageIds

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de identificadores para los mensajes con los que está relacionado un mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RELATED_IPMS  <br/> |
|Identificador:  <br/> |0x002D  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Los identificadores usan las mismas reglas de construcción específicas que se usan para la propiedad **PR_SEARCH_KEY** ([PidTagSearchKey](pidtagsearchkey-canonical-property.md)).

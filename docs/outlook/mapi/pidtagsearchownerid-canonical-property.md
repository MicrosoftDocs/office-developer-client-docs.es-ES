# <a name="pidtagsearchownerid-canonical-property"></a>Propiedad canónica PidTagSearchOwnerId

  
  
**Hace referencia a**: Outlook 
  
Contiene el identificador de proceso del proceso que envía las notificaciones para el indizador acerca de los objetos listo para indexación.

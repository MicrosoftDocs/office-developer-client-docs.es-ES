# <a name="pidtagsearchownerid-canonical-property"></a>Propiedad canónica PidTagSearchOwnerId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador de proceso del proceso que envía las notificaciones para el indizador acerca de los objetos listo para indexación.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SEARCH_OWNER_ID  <br/> |
|Expuestos por:  <br/> |Objeto de almacén de mensajes  <br/> |
|Identificador:  <br/> |0x3419  <br/> |
|Tipo de propiedad:  <br/> |PT_LONG  <br/> |
|Etiqueta de la propiedad:  <br/> |0x34190003  <br/> |
|Access:  <br/> |Solo lectura  <br/> |

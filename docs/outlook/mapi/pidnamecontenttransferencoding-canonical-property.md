# <a name="pidnamecontenttransferencoding-canonical-property"></a>Propiedad canónico PidNameContentTransferEncoding

  
  
**Se aplica a**: Outlook 
  
Contiene un valor de campo de encabezado de codificación de transferencia de contenido de [RFC3282].
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |None  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |Codificación de transferencia de contenido  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Email  <br/> |

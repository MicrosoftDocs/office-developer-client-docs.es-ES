# <a name="pidtagtransportproviders-canonical-property"></a>Propiedad canónica PidTagTransportProviders

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de identificadores de proveedores de transporte en el perfil actual.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|Identificador:  <br/> |0x3D02  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Perfil MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

No utilice esta propiedad. Está reservado para su uso por MAPI.

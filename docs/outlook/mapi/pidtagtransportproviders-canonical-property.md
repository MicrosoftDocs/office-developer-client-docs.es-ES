# <a name="pidtagtransportproviders-canonical-property"></a>Propiedad canónico PidTagTransportProviders

  
  
**Se aplica a**: Outlook 
  
Contiene una lista de identificadores de los proveedores de transporte en el perfil actual.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|Identificador:  <br/> |0x3D02  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Perfil MAPI  <br/> |
   
## <a name="remarks"></a>Notas

No utilice esta propiedad. Está reservado para su uso por MAPI.

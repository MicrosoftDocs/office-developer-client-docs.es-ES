# <a name="pidlidworkaddresscity-canonical-property"></a>Propiedad canónica PidLidWorkAddressCity

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte de ciudad o localidad de la dirección de trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressCity  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008046  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

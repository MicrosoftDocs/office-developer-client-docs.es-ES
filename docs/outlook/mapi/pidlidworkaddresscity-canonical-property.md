# <a name="pidlidworkaddresscity-canonical-property"></a>Propiedad canónica PidLidWorkAddressCity

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte de ciudad o localidad de dirección del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressCity  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008046  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

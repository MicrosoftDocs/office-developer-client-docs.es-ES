# <a name="mapi-application-development"></a>Desarrollo de aplicaciones de MAPI

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Los temas de esta sección se aplican a los desarrolladores de aplicaciones cliente y proveedores de servicios. Estos temas se organizan en orden de importancia. Es decir, los temas que son críticos para todos los programadores aparecen primero y los temas que tienen una importancia mínima o que no se aplican a todos los programadores aparecen en último lugar.

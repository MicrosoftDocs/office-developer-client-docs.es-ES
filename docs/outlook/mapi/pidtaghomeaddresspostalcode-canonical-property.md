# <a name="pidtaghomeaddresspostalcode-canonical-property"></a>Propiedad canónica PidTagHomeAddressPostalCode

  
  
**Hace referencia a**: Outlook 
  
Contiene el código postal de dirección particular del usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_ADDRESS_POSTAL_CODE, PR_HOME_ADDRESS_POSTAL_CODE_A, PR_HOME_ADDRESS_POSTAL_CODE_W  <br/> |
|Identificador:  <br/> |0x3A5B  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario o la organización del usuario.

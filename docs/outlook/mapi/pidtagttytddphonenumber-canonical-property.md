# <a name="pidtagttytddphonenumber-canonical-property"></a>Propiedad canónica PidTagTtyTddPhoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono del teléfono de texto (TTY) del contacto o del dispositivo de telecomunicaciones para personas sordas (TDD).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A4B  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

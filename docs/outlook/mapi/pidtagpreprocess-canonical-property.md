# <a name="pidtagpreprocess-canonical-property"></a>Propiedad canónica PidTagPreprocess

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si el mensaje requiere preprocesamiento.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PREPROCESS  <br/> |
|Identificador:  <br/> |0x0E22  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |MAPI no transmitible  <br/> |
   
## <a name="remarks"></a>Comentarios

> [!IMPORTANT]
> No utilice esta propiedad. Está reservado para su uso por parte de MAPI.

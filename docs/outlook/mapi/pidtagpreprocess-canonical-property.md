# <a name="pidtagpreprocess-canonical-property"></a>Propiedad canónico PidTagPreprocess

  
  
**Se aplica a**: Outlook 
  
Contiene TRUE si el mensaje requiere preprocesamiento.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PREPROCESS  <br/> |
|Identificador:  <br/> |0x0E22  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |MAPI no transmisible  <br/> |
   
## <a name="remarks"></a>Notas

> [!IMPORTANT]
> No utilice esta propiedad. Está reservado para su uso por MAPI.

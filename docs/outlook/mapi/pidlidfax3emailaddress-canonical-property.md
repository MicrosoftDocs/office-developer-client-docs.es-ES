# <a name="pidlidfax3emailaddress-canonical-property"></a>Propiedad canónica PidLidFax3EmailAddress

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la dirección de correo electrónico de la otra dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3EmailAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080D3  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si está presente, debe contener un nombre para mostrar legible por el usuario, seguido del carácter "@", seguido de un número de fax.

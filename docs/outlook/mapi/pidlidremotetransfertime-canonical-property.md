# <a name="pidlidremotetransfertime-canonical-property"></a>Propiedad canónica PidLidRemoteTransferTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica el tiempo de transferencia, en segundos, de un elemento remoto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidRemoteXferTime  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Remote  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008F04  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Mensaje remoto  <br/> |

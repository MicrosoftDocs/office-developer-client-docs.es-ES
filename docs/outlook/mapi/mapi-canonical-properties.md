<span data-ttu-id="9fe71-124">Propiedades con nombre se expresarán además por un conjunto de propiedades y un long identificador (tapa), que es un valor de 32 bits sin signo, o bien una cadena.</span><span class="sxs-lookup"><span data-stu-id="9fe71-124">Named properties are additionally attributed by a property set, and either a long ID (LID), which is an unsigned 32-bit value, or a string.</span></span> <span data-ttu-id="9fe71-125">Un conjunto de propiedades es un GUID que identifica un grupo de propiedades con nombre con un propósito similar.</span><span class="sxs-lookup"><span data-stu-id="9fe71-125">A property set is a GUID that identifies a group of named properties with a similar purpose.</span></span> <span data-ttu-id="9fe71-126">El conjunto de propiedades y el nombre de cubierta o cadena se usan para obtener o establecer la propiedad con nombre.</span><span class="sxs-lookup"><span data-stu-id="9fe71-126">The property set and LID or string name are used to get or set the named property.</span></span>

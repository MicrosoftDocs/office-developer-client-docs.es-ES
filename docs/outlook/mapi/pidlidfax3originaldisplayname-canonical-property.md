# <a name="pidlidfax3originaldisplayname-canonical-property"></a>Propiedad canónica PidLidFax3OriginalDisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre para mostrar original de la otra dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3OriginalDisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080D4  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si está presente, debe establecerse en el mismo valor que la **propiedad PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).

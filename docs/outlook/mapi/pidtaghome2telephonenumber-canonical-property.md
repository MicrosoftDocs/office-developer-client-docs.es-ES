# <a name="pidtaghome2telephonenumber-canonical-property"></a>Propiedad canónica PidTagHome2TelephoneNumber

  
  
**Hace referencia a**: Outlook 
  
Contiene un número de teléfono secundario en la página principal del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME2_TELEPHONE_NUMBER, PR_HOME2_TELEPHONE_NUMBER_A, PR_HOME2_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A2F  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y su organización.

# <a name="pidlidpostrsschannellink-canonical-property"></a>Propiedad canónico PidLidPostRssChannelLink

  
  
**Se aplica a**: Outlook 
  
Contiene la dirección URL de la fuente RSS o Atom de que el archivo XML procede.

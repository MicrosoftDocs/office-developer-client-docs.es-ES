# <a name="pidlidpostrsschannellink-canonical-property"></a>Propiedad canónica PidLidPostRssChannelLink

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene la dirección URL de la fuente RSS o Atom de que el archivo XML procede.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidPostRssChannelLink  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_PostRss  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008900  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |RSS  <br/> |

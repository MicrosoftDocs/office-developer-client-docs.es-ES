# <a name="pidtagotheraddresscity-canonical-property"></a>Propiedad canónica PidTagOtherAddressCity

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre de la otra localidad del usuario de correo, como la ciudad o la ciudad.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_CITY, PR_OTHER_ADDRESS_CITY_A, PR_OTHER_ADDRESS_CITY_W  <br/> |
|Identificador:  <br/> |0x3A5F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

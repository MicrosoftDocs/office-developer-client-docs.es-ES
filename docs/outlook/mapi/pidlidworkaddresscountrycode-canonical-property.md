# <a name="pidlidworkaddresscountrycode-canonical-property"></a>Propiedad canónica PidLidWorkAddressCountryCode

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte del código de país de la dirección del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressCountryCode  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080DB  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

# <a name="pidlidappointmentproposalnumber-canonical-property"></a>Propiedad canónica PidLidAppointmentProposalNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el número de asistentes que han enviado propuestas de contador que no han sido aceptadas o rechazadas por el organizador.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptProposalNum  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008259  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniones  <br/> |

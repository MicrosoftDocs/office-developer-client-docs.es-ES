# <a name="pidlidfax1originalentryid-canonical-property"></a>Propiedad canónica PidLidFax1OriginalEntryId

  
  
**Hace referencia a**: Outlook 
  
Especifica la propiedad EntryID original de dirección de fax del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1OriginalEntryID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080B5  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si está presente, debe especificar la propiedad EntryId de uso único que corresponde a esta dirección de fax.

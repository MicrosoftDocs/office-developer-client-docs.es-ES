# <a name="pidlidworkaddresspostofficebox-canonical-property"></a>Propiedad canónica PidLidWorkAddressPostOfficeBox

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte del cuadro de correo del trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressPostOfficeBox  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x0000804A  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

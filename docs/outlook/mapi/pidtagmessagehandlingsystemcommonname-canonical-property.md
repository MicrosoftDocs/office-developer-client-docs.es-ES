# <a name="pidtagmessagehandlingsystemcommonname-canonical-property"></a>Propiedad canónica PidTagMessageHandlingSystemCommonName

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre común del sistema de tratamiento de mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MHS_COMMON_NAME, PR_MHS_COMMON_NAME_A, PR_MHS_COMMON_NAME_W  <br/> |
|Identificador:  <br/> |0x3A0F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades contienen el nombre común del sistema de tratamiento de mensaje.

# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>Propiedad canónica PidTagOtherAddressStateOrProvince

  
  
**Hace referencia a**: Outlook 
  
Contiene el nombre del estado o provincia de la otra dirección.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Identificador:  <br/> |0x3A62  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario o la organización del usuario.

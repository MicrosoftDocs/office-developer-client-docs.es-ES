# <a name="pidlidglobalobjectid-canonical-property"></a>Propiedad canónica PidLidGlobalObjectId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el identificador único del objeto de calendario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |LID_GLOBAL_OBJID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Meeting  <br/> |
|Identificador de tipo Long (LID):  <br/> |0 x 00000003  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Una vez establecida para un objeto calendar, no debe cambiar el valor de esta propiedad. Una descripción detallada del formato puede encontrarse en [[MS-OXOCAL]](http://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).

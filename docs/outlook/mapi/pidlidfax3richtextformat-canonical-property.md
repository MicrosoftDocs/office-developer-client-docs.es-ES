# <a name="pidlidfax3richtextformat-canonical-property"></a>Propiedad canónica PidLidFax3RichTextFormat

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica que el formato de texto enriquecido del contacto del otra dirección de fax.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3RTF  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080D6  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contacto  <br/> |

# <a name="pidlidformpropstream-canonical-property"></a>Propiedad canónica PidLidFormPropStream

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Representa los datos de las propiedades personalizadas que se guardan con el mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFormPropStream  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x0000851B  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuración en tiempo de ejecución  <br/> |

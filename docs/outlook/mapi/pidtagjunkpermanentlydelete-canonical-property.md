# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>Propiedad canónica PidTagJunkPermanentlyDelete

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Significa, si se establece en 1, que los mensajes identificados como correo no deseado pueden eliminarse de forma permanente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|Identificador:  <br/> |0x6102  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Correo no deseado  <br/> |

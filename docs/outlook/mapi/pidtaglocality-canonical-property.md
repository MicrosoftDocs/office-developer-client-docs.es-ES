# <a name="pidtaglocality-canonical-property"></a>Propiedad canónica PidTagLocality

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre de la localidad del destinatario, como la ciudad o la ciudad. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_LOCALITY, PR_LOCALITY_A, PR_LOCALITY_W, PR_BUSINESS_ADDRESS_LOCALITY, PR_BUSINESS_ADDRESS_LOCALITY_A, PR_BUSINESS_ADDRESS_LOCALITY_W  <br/> |
|Identificador:  <br/> |0x3A27  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Son definidos por el destinatario y su organización.

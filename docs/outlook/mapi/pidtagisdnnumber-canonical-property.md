# <a name="pidtagisdnnumber-canonical-property"></a>Propiedad canónica PidTagIsdnNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono compatible con ISDN del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ISDN_NUMBER, PR_ISDN_NUMBER_A, PR_ISDN_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A2D  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Los define el destinatario y su organización.

# <a name="pidtagprofession-canonical-property"></a>Propiedad canónica PidTagProfession

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el ámbito del usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PROFESSION, PR_PROFESSION_A, PR_PROFESSION_W  <br/> |
|Identificador:  <br/> |0x3A46  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario o la organización del usuario.

# <a name="pidtaginitials-canonical-property"></a>Propiedad canónica PidTagInitials

  
  
**Hace referencia a**: Outlook 
  
Contiene las iniciales de los elementos del nombre completo del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Identificador:  <br/> |0x3A0A  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y su organización.

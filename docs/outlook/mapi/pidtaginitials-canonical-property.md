# <a name="pidtaginitials-canonical-property"></a>Propiedad canónica PidTagInitials

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene las iniciales de partes del nombre completo del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Identificador:  <br/> |0x3A0A  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Los define el destinatario y su organización.

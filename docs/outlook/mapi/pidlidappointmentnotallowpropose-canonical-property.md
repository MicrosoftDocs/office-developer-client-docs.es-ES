# <a name="pidlidappointmentnotallowpropose-canonical-property"></a>Propiedad canónica PidLidAppointmentNotAllowPropose

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si los asistentes no pueden proponer una fecha y hora nuevas para la reunión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptNotAllowPropose  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x0000825A  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentarios

Un valor de FALSE o la ausencia de esta propiedad indica que los asistentes pueden proponer una nueva fecha y hora.

# <a name="pidtagcontactaddressbookfolderentryids-canonical-property"></a>Propiedad canónica PidTagContactAddressBookFolderEntryIds

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene identificadores de las carpetas contactos de los almacenes de mensajes donde se mantienen los usuarios de correo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_FOLDER_ENTRYIDS  <br/> |
|Identificador:  <br/> |0x6620  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones de contactos  <br/> |
   
## <a name="remarks"></a>Comentarios

Se trata de una propiedad de una sección de perfil de libreta de direcciones de contactos.

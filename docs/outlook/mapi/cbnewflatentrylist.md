# <a name="cbnewflatentrylist"></a>CbNewFLATENTRYLIST

  
  
**Hace referencia a**: Outlook 
  
Calcula el número de bytes que se debe asignar para una nueva estructura [FLATENTRYLIST](flatentrylist.md) que contiene varias estructuras [FLATENTRY](flatentry.md) de un tamaño especificado. 
  
|||
|:-----|:-----|
|Archivo de encabezado:  <br/> |Mapidefs.h  <br/> |
|Estructura relacionado:  <br/> |**FLATENTRYLIST** <br/> |
   
```cpp
CbNewFLATENTRY (_cb)
```

## <a name="parameters"></a>Parámetros

 __cb_
  
> Recuento de bytes en las estructuras **FLATENTRY** que se deben incluir en la nueva estructura **FLATENTRYLIST** .

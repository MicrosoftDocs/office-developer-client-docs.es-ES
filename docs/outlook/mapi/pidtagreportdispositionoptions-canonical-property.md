# <a name="pidtagreportdispositionoptions-canonical-property"></a>Propiedad canónico PidTagReportDispositionOptions

  
  
**Se aplica a**: Outlook 
  
Se asigna al encabezado de opciones de eliminación de MIME para los mensajes que solicitar confirmación. El valor de esta propiedad debe ser una cadena vacía.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_DISPOSITION_OPTIONS, PR_REPORT_DISPOSITION_OPTIONS_A, PR_REPORT_DISPOSITION_OPTIONS_W  <br/> |
|Identificador:  <br/> |0x0085  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobres MAPI  <br/> |

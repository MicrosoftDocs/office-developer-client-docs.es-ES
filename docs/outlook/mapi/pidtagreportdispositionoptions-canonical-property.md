# <a name="pidtagreportdispositionoptions-canonical-property"></a>Propiedad canónica PidTagReportDispositionOptions

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Se asigna al encabezado de disposición MIME-opciones para los mensajes que solicitan confirmaciones. El valor de esta propiedad debe ser una cadena vacía.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REPORT_DISPOSITION_OPTIONS, PR_REPORT_DISPOSITION_OPTIONS_A, PR_REPORT_DISPOSITION_OPTIONS_W  <br/> |
|Identificador:  <br/> |0x0085  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |

# <a name="pidlidownercriticalchange-canonical-property"></a>Propiedad canónica PidLidOwnerCriticalChange

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la fecha y hora cuando se ha enviado una convocatoria de reunión mediante el organizador.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |LID_OWNER_CRITICAL_CHANGE  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Meeting  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000001a  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor debe especificarse en hora Universal coordinada (UTC).

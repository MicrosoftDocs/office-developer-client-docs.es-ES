# <a name="pidtagsurname-canonical-property"></a>Propiedad canónica PidTagSurname

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el último o el apellido del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SURNAME, PR_SURNAME_A, PR_SURNAME_W  <br/> |
|Identificador:  <br/> |0x3A11  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y la organización del destinatario.

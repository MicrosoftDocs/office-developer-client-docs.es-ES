# <a name="pidtagfreebusyrangetimestamp-canonical-property"></a>Propiedad canónica PidTagFreeBusyRangeTimestamp

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la hora en que se publicaron los datos.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FREEBUSY_RANGE_TIMESTAMP  <br/> |
|Identificador:  <br/> |0x6868  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Disponibilidad  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es el número de minutos desde la medianoche del 1 de enero de 1601 en la hora universal coordinada (UTC).

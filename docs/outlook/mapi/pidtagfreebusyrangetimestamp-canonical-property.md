# <a name="pidtagfreebusyrangetimestamp-canonical-property"></a>Propiedad canónica PidTagFreeBusyRangeTimestamp

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene la hora cuando se publican los datos.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FREEBUSY_RANGE_TIMESTAMP  <br/> |
|Identificador:  <br/> |0x6868  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Libre/ocupado  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es el número de minutos transcurridos desde la medianoche del 1 de enero de 1601 en hora Universal coordinada (UTC).

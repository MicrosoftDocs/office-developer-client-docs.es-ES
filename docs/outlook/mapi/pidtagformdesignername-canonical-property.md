# <a name="pidtagformdesignername-canonical-property"></a>Propiedad canónica PidTagFormDesignerName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre para mostrar para el objeto que se utiliza para diseñar el formulario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FORM_DESIGNER_NAME, PR_FORM_DESIGNER_NAME_A, PR_FORM_DESIGNER_NAME_W  <br/> |
|Identificador:  <br/> |0x3308  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MAPI comunes  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades contienen el identificador único para el objeto de diseñador del formulario.

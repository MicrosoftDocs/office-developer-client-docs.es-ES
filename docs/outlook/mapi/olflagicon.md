# <a name="olflagicon"></a>OlFlagIcon

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica si un elemento está marcado y, si es así, el color asociado con el elemento marcado.
  
## <a name="quick-info"></a>Información rápida

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Miembros

 _olNoFlagIcon_
  
> No se ha asociado ninguna marca con el elemento.
    
 _olPurpleFlagIcon_
  
> El color púrpura se ha asociado con el elemento marcado.
    
 _olOrangeFlagIcon_
  
> Naranja se ha asociado con el elemento marcado.
    
 _olGreenFlagIcon_
  
> El color verde se ha asociado al elemento marcado.
    
 _olYellowFlagIcon_
  
> El color amarillo se ha asociado al elemento marcado.
    
 _olBlueFlagIcon_
  
> El color azul se ha asociado al elemento marcado.
    
 _olRedFlagIcon_
  
> El color rojo se ha asociado al elemento marcado.

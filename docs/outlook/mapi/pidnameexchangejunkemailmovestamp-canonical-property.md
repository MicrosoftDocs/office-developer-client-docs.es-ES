# <a name="pidnameexchangejunkemailmovestamp-canonical-property"></a>Propiedad canónica PidNameExchangeJunkEmailMoveStamp

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el valor del mensaje persistente que indica que el mensaje no debe ser procesado por un filtro de correo no deseado porque el mensaje ya se ha procesado o es seguro.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |http://schemas.microsoft.com/exchange/junkemailmovestamp  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Mensajería segura  <br/> |
   
## <a name="remarks"></a>Observaciones

Esta propiedad se marca en todos los mensajes movidos por la regla de correo electrónico no deseado o en contenido de confianza.

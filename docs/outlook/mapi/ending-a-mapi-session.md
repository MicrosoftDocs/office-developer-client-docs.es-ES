<span data-ttu-id="6eb5a-105">Los clientes pueden finalizar sus sesiones en respuesta a la solicitud de un usuario, ya sea inmediatamente o después de que se hayan procesado todos los mensajes salientes y cuando se produzca un error crítico.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-105">Clients can end their sessions in response to a user's request, either immediately or after all outbound messages have been processed, and when a critical error occurs.</span></span> <span data-ttu-id="6eb5a-106">Algunos clientes deben permanecer conectados para que los mensajes salientes pendientes puedan llegar al proveedor de transporte y al sistema de mensajería de destino.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-106">Some clients need to stay logged on so that pending outbound messages can reach the transport provider and the destination messaging system.</span></span> <span data-ttu-id="6eb5a-107">Si un cliente envía un mensaje y cierra sesión inmediatamente, el mensaje puede permanecer en la cola saliente hasta que un usuario vuelva a iniciar sesión y permanezca conectado lo suficiente como para que el mensaje se transmita.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-107">If such a client sends a message and immediately logs off, the message may remain in the outgoing queue until a user logs back on and stays logged on long enough for the message to be transmitted.</span></span>

6. <span data-ttu-id="6eb5a-116">Si llamó [a OleInitialize](https://msdn.microsoft.com/library/ms690134%28v=VS.85%29.aspx) durante el inicio de la sesión para inicializar las bibliotecas OLE, desinicializarlas ahora llamando [a OleUninitialize](https://msdn.microsoft.com/library/ms691326%28VS.85%29.aspx).</span><span class="sxs-lookup"><span data-stu-id="6eb5a-116">If you called [OleInitialize](https://msdn.microsoft.com/library/ms690134%28v=VS.85%29.aspx) during session startup to initialize the OLE libraries, uninitialize them now by calling [OleUninitialize](https://msdn.microsoft.com/library/ms691326%28VS.85%29.aspx).</span></span> <span data-ttu-id="6eb5a-117">Solo los clientes que han llamado **a OleInitialize** deben llamar **a OleUninitialize**.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-117">Only clients that have called **OleInitialize** must call **OleUninitialize**.</span></span> 
    
7. <span data-ttu-id="6eb5a-118">Uninitialize las bibliotecas MAPI llamando [a MAPIUninitialize](mapiuninitialize.md).</span><span class="sxs-lookup"><span data-stu-id="6eb5a-118">Uninitialize the MAPI libraries by calling [MAPIUninitialize](mapiuninitialize.md).</span></span> <span data-ttu-id="6eb5a-119">Si llamó a **OleInitialize** en algún momento, asegúrese de que se produce una llamada a **OleUninitialize** antes de esta llamada a **MAPIUninitialize**.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-119">If you called **OleInitialize** at some point, make sure that a call to **OleUninitialize** occurs before this call to **MAPIUninitialize**.</span></span> <span data-ttu-id="6eb5a-120">El tiempo es crucial.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-120">The timing is crucial.</span></span> <span data-ttu-id="6eb5a-121">Si la llamada a **OleUninitialize** sigue la llamada a **MAPIUninitialize**, el cliente podría finalizar de forma no correcta.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-121">If the call to **OleUninitialize** follows the call to **MAPIUninitialize**, your client might terminate ungracefully.</span></span> 
    
8. <span data-ttu-id="6eb5a-122">Si llamó [a ScInitMapiUtil](scinitmapiutil.md) durante el inicio de la sesión para inicializar la biblioteca de utilidades MAPI, desinicializarla ahora llamando [a DeinitMapiUtil](deinitmapiutil.md).</span><span class="sxs-lookup"><span data-stu-id="6eb5a-122">If you called [ScInitMapiUtil](scinitmapiutil.md) during session startup to initialize the MAPI utility library, uninitialize it now by calling [DeinitMapiUtil](deinitmapiutil.md).</span></span> <span data-ttu-id="6eb5a-123">Solo los clientes que han llamado **a ScInitMapiUtil** deben llamar **a DeinitMapiUtil**.</span><span class="sxs-lookup"><span data-stu-id="6eb5a-123">Only clients that have called **ScInitMapiUtil** must call **DeinitMapiUtil**.</span></span>

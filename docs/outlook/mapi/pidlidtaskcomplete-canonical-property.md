# <a name="pidlidtaskcomplete-canonical-property"></a>Propiedad canónica PidLidTaskComplete

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Indica que la tarea está terminada.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskComplete  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000811C  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Task  <br/> |

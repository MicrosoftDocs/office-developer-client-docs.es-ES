# <a name="pidtagchildrensnames-canonical-property"></a>Propiedad canónica PidTagChildrensNames

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de nombres de elementos secundarios.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CHILDRENS_NAMES, PR_CHILDRENS_NAMES_A, PR_CHILDRENS_NAMES_W  <br/> |
|Identificador:  <br/> |0x3A58  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

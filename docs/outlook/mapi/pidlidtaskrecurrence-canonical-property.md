# <a name="pidlidtaskrecurrence-canonical-property"></a>Propiedad canónica PidLidTaskRecurrence

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una estructura RecurrencePattern que proporciona información sobre las tareas periódicas.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidTaskRecur  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Task  <br/> |
|Id. largo (LID):  <br/> |0x00008116  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Tarea  <br/> |
   
## <a name="remarks"></a>Comentarios

Para obtener información sobre cómo crear y especificar una estructura RecurrencePattern, vea [Create a Simple Recurrent Task Item](how-to-create-a-simple-recurrent-task-item.md).

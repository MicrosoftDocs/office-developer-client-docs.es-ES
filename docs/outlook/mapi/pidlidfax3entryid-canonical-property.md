# <a name="pidlidfax3entryid-canonical-property"></a>Propiedad canónica PidLidFax3EntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el EntryID de la otra dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3EntryID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080D1  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contacto  <br/> |

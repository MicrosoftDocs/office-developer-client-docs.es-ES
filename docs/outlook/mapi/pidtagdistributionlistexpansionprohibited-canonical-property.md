# <a name="pidtagdistributionlistexpansionprohibited-canonical-property"></a>Propiedad canónico PidTagDistributionListExpansionProhibited

  
  
**Se aplica a**: Outlook 
  
Contiene TRUE si un agente de transferencia de mensajes (MTA) está prohibido de expandir listas de distribución.

# <a name="pidtagdistributionlistexpansionprohibited-canonical-property"></a>Propiedad canónica PidTagDistributionListExpansionProhibited

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si un agente de transferencia de mensajes (MTA) está prohibido a expandir las listas de distribución.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_DL_EXPANSION_PROHIBITED  <br/> |
|Identificador:  <br/> |0x0014  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |

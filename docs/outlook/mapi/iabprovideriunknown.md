# <a name="iabprovider--iunknown"></a>IABProvider : IUnknown

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Proporciona un método para iniciar la sesión a un objeto de proveedor de la libreta de direcciones y un método para invalidar un objeto de proveedor de la libreta de direcciones.
  
|||
|:-----|:-----|
|Archivo de encabezado:  <br/> |Mapispi.h  <br/> |
|Expuestos por:  <br/> |Objetos de proveedor de la libreta de direcciones  <br/> |
|Se implementa mediante:  <br/> |Proveedores de la libreta de direcciones  <br/> |
|Llamado por:  <br/> |MAPI  <br/> |
|Identificador de interfaz:  <br/> |IID_IABProvider  <br/> |
|Tipo de puntero:  <br/> |LPABPROVIDER  <br/> |

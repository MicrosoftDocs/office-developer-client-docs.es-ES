# <a name="mapi-client-application-overview"></a>Información general sobre la aplicación de cliente MAPI

  
  
**Se aplica a**: Outlook 
  
Una aplicación de cliente MAPI es cualquier aplicación que usa la interfaz de programación de MAPI. Las aplicaciones cliente implementan mensajería tareas como su enfoque principal o secundario. Aplicaciones de cliente de mensajería, como las aplicaciones que envían y reciben correo electrónico, implementar la mensajería como su enfoque principal. Para aplicaciones cliente que no sean de mensajería, como el inventario o aplicaciones de configuración, la mensajería es una característica secundaria.
  
Las actividades de mensajería puede incluir, por ejemplo, una aplicación de procesamiento de texto que tiene un comando **Enviar** en su menú **archivo** para permitir que los documentos que se envíen, correo electrónico de Microsoft Office Outlook, programas de automatización de flujo de trabajo y servicios de boletines electrónicos. 
  
Las aplicaciones cliente pueden incluir ya sea el usuario para crear un entorno interactivo o funcionar sin un usuario en un entorno automatizado. Aunque MAPI proporciona un conjunto de cuadros de diálogo comunes con su interfaz de usuario estándar, las aplicaciones cliente no son necesarios para presentar una interfaz de usuario. De hecho, puede controlarse todo el procesamiento en la aplicación, si lo desea. Un ejemplo de una aplicación cliente automatizada sería una aplicación de administración de inventario que está programada para enrutar elementos de un tipo determinado a destinatarios estándares de forma regular.

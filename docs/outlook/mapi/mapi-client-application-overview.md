# <a name="mapi-client-application-overview"></a>Información general sobre la aplicación cliente MAPI

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Una aplicación de cliente MAPI es cualquier aplicación que utiliza la interfaz de programación de MAPI. Las aplicaciones cliente implementan tareas de mensajería como un enfoque principal o secundario. Las aplicaciones cliente de mensajería, como las aplicaciones que envían y reciben correo electrónico, implementan la mensajería como enfoque principal. Para las aplicaciones cliente que no sean de mensajería, como las aplicaciones de inventario o configuración, la mensajería es una característica secundaria.
  
Las actividades de mensajería pueden incluir, por ejemplo, una aplicación de procesamiento de texto que tiene un comando **Enviar** en su menú **archivo** para permitir el envío de documentos, el correo electrónico de Microsoft Office Outlook, los programas de automatización de flujo de trabajo y los servicios de tablón de anuncios. 
  
Las aplicaciones cliente pueden incluir al usuario para crear un entorno interactivo o operar sin un usuario en un entorno automatizado. Aunque MAPI proporciona un conjunto de cuadros de diálogo comunes con su interfaz de usuario estándar, las aplicaciones cliente no necesitan presentar una interfaz de usuario. De hecho, todo el procesamiento puede controlarse en la aplicación, si lo desea. Un ejemplo de una aplicación de cliente automatizada sería una aplicación de administración de inventario programada para enrutar los elementos de un tipo determinado a los destinatarios estándar de forma regular.

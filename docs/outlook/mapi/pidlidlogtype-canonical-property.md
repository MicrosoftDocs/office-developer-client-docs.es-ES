# <a name="pidlidlogtype-canonical-property"></a>Propiedad canónica PidLidLogType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre para mostrar de la aplicación de registro en diario (por ejemplo, "MSWord") y suele ser un atributo de forma libre de un mensaje de diario, normalmente una cadena.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLogType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x00008700  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Diario  <br/> |

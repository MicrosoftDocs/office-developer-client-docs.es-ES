# <a name="pidtagscheduleinforesourcetype-canonical-property"></a>Propiedad canónica PidTagScheduleInfoResourceType

  
  
**Hace referencia a**: Outlook 
  
Contiene un valor que se debe establecer en cero (0).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SCHDINFO_RESOURCE_TYPE  <br/> |
|Identificador:  <br/> |0x6841  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Libre/ocupado  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe establecerse en cero (0) cuando envían y omite tras la recepción.

# <a name="pidtagscheduleinforesourcetype-canonical-property"></a>Propiedad canónica PidTagScheduleInfoResourceType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un valor que debe establecerse en cero (0).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SCHDINFO_RESOURCE_TYPE  <br/> |
|Identificador:  <br/> |0x6841  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Disponibilidad  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe establecerse en cero (0) cuando se envía y se omite al recibirla.

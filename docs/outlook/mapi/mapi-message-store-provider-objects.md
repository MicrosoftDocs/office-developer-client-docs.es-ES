# <a name="mapi-message-store-provider-objects"></a>Objetos de proveedor de almacén de mensajes MAPI
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Los proveedores de almacén de mensajes implementan proveedor y objetos de inicio de sesión, como hacen todos los proveedores de servicio. También implementan un objeto de almacén de mensajes, carpetas, mensajes, datos adjuntos y las tablas. Como una opción, algunos proveedores de almacenes de mensaje implementan objetos de estado.
  
En la siguiente ilustración se muestra cada objeto de almacén de mensajes con su interfaz correspondiente y el componente MAPI que lo utiliza.

# <a name="pidnamecreatedatetimereadonly-canonical-property"></a>Propiedad canónica PidNameCreateDateTimeReadOnly

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la hora en que se creó el archivo.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |CreateDtmRo  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Común  <br/> |

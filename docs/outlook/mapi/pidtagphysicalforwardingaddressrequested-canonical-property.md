# <a name="pidtagphysicalforwardingaddressrequested-canonical-property"></a>Propiedad canónica PidTagPhysicalForwardingAddressRequested

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si un remitente de mensaje solicita al agente de transferencia de mensajes que adjunte una dirección de reenvío física para un destinatario del mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PHYSICAL_FORWARDING_ADDRESS_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C0E  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

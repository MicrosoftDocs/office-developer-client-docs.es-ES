<span data-ttu-id="79d45-109">Las propiedades de contenido del mensaje describen el texto de un mensaje.</span><span class="sxs-lookup"><span data-stu-id="79d45-109">Message content properties describe the text of a message.</span></span> <span data-ttu-id="79d45-110">Cada clase de mensaje tiene su propio conjunto de propiedades de contenido.</span><span class="sxs-lookup"><span data-stu-id="79d45-110">Every message class has its own set of content properties.</span></span> <span data-ttu-id="79d45-111">MAPI define las propiedades de contenido para los mensajes de nota e informe; <span data-ttu-id="79d45-115">Para las nuevas clases de mensaje, los clientes pueden definir propiedades específicas del contenido de dos maneras:</span><span class="sxs-lookup"><span data-stu-id="79d45-115">For new message classes, clients can define content-specific properties in one of two ways:</span></span>

<span data-ttu-id="79d45-124">Las propiedades de sobre proporcionan información que se usa para transmitir un mensaje de un destinatario a otro.</span><span class="sxs-lookup"><span <span data-ttu-id="79d45-129">Los proveedores de transporte suelen implementar una página de propiedades especiales para mostrar estas propiedades y permitir que los clientes las cambien.</span><span <span data-ttu-id="79d45-132">Las propiedades del destinatario describen el destino de un mensaje enviado.</span><span class="sxs-lookup"><span data-stu-id="79d45-132">Recipient properties describe the destination for a sent message.</span></span> <span data-ttu-id="79d45-133">Un destinatario puede ser un usuario de mensajería, una lista de distribución o un equipo.</span><span class="sxs-lookup"><span data-stu-id="79d45-133">A recipient can be a messaging user, distribution list, or a computer.</span></span> <span data-ttu-id="79d45-134">Mapi define las propiedades de los destinatarios y las establecen los proveedores de servicios.</span><span class="sxs-lookup"><span data-stu-id="79d45-134">Recipient properties are defined by MAPI and set by service providers.</span></span> <span data-ttu-id="79d45-135">Algunos proveedores de libretas de direcciones admiten algunas propiedades de destinatarios para sus objetos de lista de distribución y usuario de mensajería;

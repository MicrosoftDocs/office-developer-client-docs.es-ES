# <a name="forms-and-mapi-client-applications"></a>Formularios y las aplicaciones cliente MAPI

**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Dado que todos los mensajes MAPI son instancias de formularios, las aplicaciones cliente mostrarán e interactúan con objetos de formulario al igual que con otros objetos de mensaje.
  
Las aplicaciones de cliente que usan las interfaces de formulario MAPI pueden proporcionar interacción adicional con objetos de formulario. Las aplicaciones cliente pueden usar cualquiera de los verbos especial definido por un formulario para proporcionar a los usuarios con toda la gama de interacciones previsto por el diseñador del formulario. Los clientes también pueden usar las propiedades personalizadas que expone el servidor de formulario para ayudar a los usuarios ordenar, obtener una vista previa o, en caso contrario, administrar formularios en la interfaz del cliente.
  
Para obtener más información, vea [Controlar formularios de MAPI](handling-mapi-forms.md).

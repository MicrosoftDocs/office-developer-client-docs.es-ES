# <a name="about-the-attachment-security-api"></a>Información sobre la API de seguridad de datos adjuntos

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
La API de seguridad de datos adjuntos permite soluciones de Outlook detectar si un archivo adjunto se considera seguro por Outlook y está bloqueado para su visualización e indización. Proporciona la interfaz y las definiciones siguientes.

# <a name="pidtagspamtrustedrecipients-canonical-property"></a>Propiedad canónica PidTagSpamTrustedRecipients
 
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista delimitada por punto y coma de direcciones de correo electrónico y dominios que representan a los destinatarios de confianza.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SPAM_TRUSTED_RECIPIENTS_W  <br/> |
|Id. largo (LID):  <br/> |0x0419  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Correo no deseado  <br/> |

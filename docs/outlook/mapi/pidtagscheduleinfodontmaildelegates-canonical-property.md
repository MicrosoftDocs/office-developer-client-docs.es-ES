# <a name="pidtagscheduleinfodontmaildelegates-canonical-property"></a>Propiedad canónica PidTagScheduleInfoDontMailDelegates

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si el delegado no desea recibir actualizaciones.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SCHDINFO_DONT_MAIL_DELEGATES  <br/> |
|Identificador:  <br/> |0x6843  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Libre/ocupado  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se debe establecer en el objeto de información de delegado.

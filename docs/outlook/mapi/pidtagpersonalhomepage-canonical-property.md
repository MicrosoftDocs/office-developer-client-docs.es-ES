# <a name="pidtagpersonalhomepage-canonical-property"></a>Propiedad canónica PidTagPersonalHomePage

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la dirección URL de la página principal personal de un usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PERSONAL_HOME_PAGE, PR_PERSONAL_HOME_PAGE_A, PR_PERSONAL_HOME_PAGE_W  <br/> |
|Identificador:  <br/> |0x3A50  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

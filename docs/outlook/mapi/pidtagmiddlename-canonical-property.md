# <a name="pidtagmiddlename-canonical-property"></a>Propiedad canónica PidTagMiddleName

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el segundo nombre de un contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MIDDLE_NAME, PR_MIDDLE_NAME_A, PR_MIDDLE_NAME_W  <br/> |
|Identificador:  <br/> |0x3A44  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario.

# <a name="pidtagreturnedmessageid-canonical-property"></a>Propiedad canónica PidTagReturnedMessageid

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene TRUE si el mensaje original se devuelve con un informe no leído.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RETURNED_IPM  <br/> |
|Identificador:  <br/> |0x0033  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Sobre MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Un proveedor de transporte X. 400 establece esta propiedad en el informe de no leído.

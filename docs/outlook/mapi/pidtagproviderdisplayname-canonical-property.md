# <a name="pidtagproviderdisplayname-canonical-property"></a>Propiedad canónica PidTagProviderDisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre para mostrar de un proveedor.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PROVIDER_DISPLAY_NAME, PR_PROVIDER_DISPLAY_NAME_A, PR_PROVIDER_DISPLAY_NAME_W  <br/> |
|Identificador:  <br/> |0x3418  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Almacén de mensajes MAPI  <br/> |

# <a name="optional-report-message-properties"></a>Propiedades opcionales para los mensajes de informes

  
  
**Hace referencia a**: Outlook 
  
En ocasiones, aparecen las siguientes propiedades con mensajes de informe:
  
**Título de la tabla**

|**Propiedad**|**Descripción**|
|:-----|:-----|
|**PR_ORIGINAL_AUTHOR_ADDRTYPE** ([PidTagOriginalAuthorAddressType](pidtagoriginalauthoraddresstype-canonical-property.md))  <br/> |Contiene el tipo de dirección del autor de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_AUTHOR_EMAIL_ADDRESS** ([PidTagOriginalAuthorEmailAddress](pidtagoriginalauthoremailaddress-canonical-property.md))  <br/> |Contiene la dirección de correo electrónico del autor de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_AUTHOR_ENTRYID** ([PidTagOriginalAuthorEntryId](pidtagoriginalauthorentryid-canonical-property.md))  <br/> |Contiene el identificador de entrada del autor de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_AUTHOR_NAME** ([PidTagOriginalAuthorName](pidtagoriginalauthorname-canonical-property.md))  <br/> |Contiene el nombre para mostrar del autor de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_AUTHOR_SEARCH_KEY** ([PidTagOriginalAuthorSearchKey](pidtagoriginalauthorsearchkey-canonical-property.md))  <br/> |Contiene la clave de búsqueda del autor de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_ADDRTYPE** ([PidTagOriginalSentRepresentingAddressType](pidtagoriginalsentrepresentingaddresstype-canonical-property.md))  <br/> |Contiene el tipo de dirección del usuario de mensajería en cuyo nombre se ha enviado el mensaje original.  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_EMAIL_ADDRESS** ([PidTagOriginalSentRepresentingEmailAddress](pidtagoriginalsentrepresentingemailaddress-canonical-property.md))  <br/> |Contiene la dirección de correo electrónico del usuario de mensajería en cuyo nombre se ha enviado el mensaje original.  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_ENTRYID** ([PidTagOriginalSentRepresentingEntryId](pidtagoriginalsentrepresentingentryid-canonical-property.md))  <br/> |Contiene el identificador de entrada del usuario de mensajería en cuyo nombre se ha enviado el mensaje original.  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_NAME** ([PidTagOriginalSentRepresentingName](pidtagoriginalsentrepresentingname-canonical-property.md))  <br/> |Contiene el nombre para mostrar del usuario de mensajería en cuyo nombre se ha enviado el mensaje original.  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_SEARCH_KEY** ([PidTagOriginalSentRepresentingSearchKey](pidtagoriginalsentrepresentingsearchkey-canonical-property.md))  <br/> |Contiene la clave de búsqueda del usuario de mensajería en cuyo nombre se ha enviado el mensaje original.  <br/> |
|**PR_ORIGINAL_SENDER_ADDRTYPE** ([PidTagOriginalSenderAddressType](pidtagoriginalsenderaddresstype-canonical-property.md))  <br/> |Contiene el tipo de dirección del remitente de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_SENDER_EMAIL_ADDRESS** ([PidTagOriginalSenderEmailAddress](pidtagoriginalsenderemailaddress-canonical-property.md))  <br/> |Contiene la dirección de correo electrónico del remitente de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_SENDER_ENTRYID** ([PidTagOriginalSenderEntryId](pidtagoriginalsenderentryid-canonical-property.md))  <br/> |Contiene el identificador de entrada del remitente de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_SENDER_NAME** ([PidTagOriginalSenderName](pidtagoriginalsendername-canonical-property.md))  <br/> |Contiene el nombre para mostrar del remitente de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |
|**PR_ORIGINAL_SENDER_SEARCH_KEY** ([PidTagOriginalSenderSearchKey](pidtagoriginalsendersearchkey-canonical-property.md))  <br/> |Contiene la clave de búsqueda para el remitente de la primera versión de un mensaje, es decir, el mensaje antes de se reenvía o se responde a.  <br/> |

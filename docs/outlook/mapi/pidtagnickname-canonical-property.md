# <a name="pidtagnickname-canonical-property"></a>Propiedad canónica PidTagNickname

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el alias del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_NICKNAME, PR_NICKNAME_A, PR_NICKNAME_W  <br/> |
|Identificador:  <br/> |0x3A4F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son opcionales. El usuario define su contenido.

# <a name="pidlidagingdontageme-canonical-property"></a>Propiedad canónica PidLidAgingDontAgeMe

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica un valor TRUE si el elemento no se archivará automáticamente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidAgingDontAgeMe  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x0000850E  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Común  <br/> |

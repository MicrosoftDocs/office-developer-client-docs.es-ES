# <a name="pidlidagingdontageme-canonical-property"></a>Propiedad canónica PidLidAgingDontAgeMe

  
  
**Hace referencia a**: Outlook 
  
Especifica un valor de TRUE si el elemento no se guardarán automáticamente.

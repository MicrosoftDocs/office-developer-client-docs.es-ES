# <a name="pidlidclipstart-canonical-property"></a>Propiedad canónica PidLidClipStart

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la fecha de inicio y la hora del evento en hora Universal coordinada (UTC) para los objetos de calendario de una sola instancia y medianoche en la fecha de la primera instancia en UTC para una serie periódica.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidClipStart  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008235  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Calendario  <br/> |

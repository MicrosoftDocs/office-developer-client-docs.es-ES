<span data-ttu-id="2551f-112">Un informe de lectura indica que se ha establecido el indicador de lectura de un mensaje, que puede producirse cuando se abre, imprimir, mover o copiar el mensaje.</span><span class="sxs-lookup"><span data-stu-id="2551f-112">A read report indicates that the read flag of a message has been set, which can occur when the message is opened, printed, moved, or copied.</span></span> <span data-ttu-id="2551f-113">Si un proveedor de almacén de mensajes genera un informe de lectura en respuesta a un movimiento u operación de copia depende de dónde se dirige el mensaje.</span><span class="sxs-lookup"><span data-stu-id="2551f-113">Whether or not a message store provider generates a read report in response to a move or copy operation depends on where the message is going.</span></span> <span data-ttu-id="2551f-114">Si es que se mueven o se copian a otro almacén de mensajes, un informe de lectura seguramente siempre se envíen.</span><span class="sxs-lookup"><span data-stu-id="2551f-114">If it is being moved or copied to another message store, a read report will most likely always be sent.</span></span> <span data-ttu-id="2551f-115">Si es que se va a mover o copiar en el almacén de mensajes actual, un informe de lectura es posible que o es posible que no se enviarán.</span><span class="sxs-lookup"><span data-stu-id="2551f-115">If it is being moved or copied in the current message store, a read report might or might not be sent.</span></span>

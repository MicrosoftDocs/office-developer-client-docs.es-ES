# <a name="pidlidemail3addresstype-canonical-property"></a>Propiedad canónica PidLidEmail3AddressType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el tipo de dirección de la tercera dirección de correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail3AddrType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080A2  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Si está presente, el valor de esta propiedad debe ser un tipo de dirección válida.

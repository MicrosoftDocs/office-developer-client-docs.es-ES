# <a name="pidtagread-canonical-property"></a>Propiedad canónica PidTagRead

  
  
**Hace referencia a**: Outlook 
  
Indica si se ha leído una fila de tabla.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_READ  <br/> |
|Identificador:  <br/> |0x0E69  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |MAPI no transmisible  <br/> |

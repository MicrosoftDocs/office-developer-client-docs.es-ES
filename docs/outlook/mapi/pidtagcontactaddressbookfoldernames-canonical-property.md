# <a name="pidtagcontactaddressbookfoldernames-canonical-property"></a>Propiedad canónica PidTagContactAddressBookFolderNames

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene los nombres de las carpetas contactos donde se mantienen los usuarios de correo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_FOLDER_NAMES  <br/> |
|Identificador:  <br/> |0x6623  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Libreta de direcciones de contactos  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas son propiedades de una sección de perfil de libreta de direcciones de contactos.

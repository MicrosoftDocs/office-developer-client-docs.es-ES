# <a name="pidlidfax1addresstype-canonical-property"></a>Propiedad canónico PidLidFax1AddressType

  
  
**Se aplica a**: Outlook 
  
Especifica el tipo de dirección para la dirección de fax del trabajo de un contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax1AddrType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080B2  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

Esta propiedad, si está presente, debe establecerse en "FAX".

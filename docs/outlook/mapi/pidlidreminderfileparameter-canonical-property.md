# <a name="pidlidreminderfileparameter-canonical-property"></a>Propiedad canónica PidLidReminderFileParameter

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre de archivo del archivo de sonido que se debe reproducir un cliente cuando el aviso para ese objeto se convierte en vencidas.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidReminderFileParam  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000851F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reminder  <br/> |
   
## <a name="remarks"></a>Comentarios

Si esta propiedad no está presente, el cliente puede utilizar un valor predeterminado.

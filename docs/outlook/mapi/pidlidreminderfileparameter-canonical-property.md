# <a name="pidlidreminderfileparameter-canonical-property"></a>Propiedad canónica PidLidReminderFileParameter

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre de archivo del sonido que un cliente debe reproducir cuando el aviso de ese objeto se ha vencido.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidReminderFileParam  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Id. largo (LID):  <br/> |0x0000851F  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reminder  <br/> |
   
## <a name="remarks"></a>Comentarios

Si esta propiedad no está presente, el cliente puede usar un valor predeterminado.

# <a name="pidtagpostofficebox-canonical-property"></a>Propiedad canónica PidTagPostOfficeBox

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número o identificador del apartado de correos del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_POST_OFFICE_BOX, PR_POST_OFFICE_BOX_A, PR_POST_OFFICE_BOX_W, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_A, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Identificador:  <br/> |0x3A2B  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. Los define el destinatario y su organización.

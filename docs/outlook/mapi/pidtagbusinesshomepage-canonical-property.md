# <a name="pidtagbusinesshomepage-canonical-property"></a>Propiedad canónica PidTagBusinessHomePage

  
  
**Hace referencia a**: Outlook 
  
Contiene la dirección URL de la página principal de la empresa.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_BUSINESS_HOME_PAGE, PR_BUSINESS_HOME_PAGE_A, PR_BUSINESS_HOME_PAGE_W  <br/> |
|Identificador:  <br/> |0x3A51  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades se utilizan como un vínculo a la página del contacto empresarial principal.

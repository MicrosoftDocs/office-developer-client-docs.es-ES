# <a name="pidlidappointmentcounterproposal-canonical-property"></a>Propiedad canónica PidLidAppointmentCounterProposal

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica que esta respuesta de reunión es una propuesta de contador.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptCounterProposal  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008257  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniones  <br/> |

# <a name="pidtaghomeaddressstateorprovince-canonical-property"></a>Propiedad canónica PidTagHomeAddressStateOrProvince

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la parte del estado o la provincia de la dirección de un usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_ADDRESS_STATE_OR_PROVINCE, PR_HOME_ADDRESS_STATE_OR_PROVINCE_A, PR_HOME_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Identificador:  <br/> |0x3A5C  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

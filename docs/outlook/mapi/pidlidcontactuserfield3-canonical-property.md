# <a name="pidlidcontactuserfield3-canonical-property"></a>Propiedad canónica PidLidContactUserField3

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el tercer campo del contacto que está destinado a uso varios para el contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactUserField3  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x00008051  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene texto que no está relacionado con ninguna otra propiedad específica del contacto.

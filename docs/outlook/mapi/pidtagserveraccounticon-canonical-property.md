# <a name="pidtagserveraccounticon-canonical-property"></a>Propiedad canónica PidTagServerAccountIcon

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el icono que se muestra en el **selector de cuentas**.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SERVER_ACCOUNT_ICON  <br/> |
|Identificador:  <br/> |0x341F  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Común  <br/> |

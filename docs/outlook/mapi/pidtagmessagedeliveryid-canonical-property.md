# <a name="pidtagmessagedeliveryid-canonical-property"></a>Propiedad canónica PidTagMessageDeliveryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador de sistema (MTS) de transferencia de mensaje de un mensaje enviado a una aplicación cliente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MESSAGE_DELIVERY_ID  <br/> |
|Identificador:  <br/> |0x001B  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |

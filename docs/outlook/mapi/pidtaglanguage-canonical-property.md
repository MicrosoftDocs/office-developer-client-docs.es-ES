# <a name="pidtaglanguage-canonical-property"></a>Propiedad canónica PidTagLanguage

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un valor que indica el idioma en el que el usuario de mensajería está escribiendo mensajes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_LANGUAGE, PR_LANGUAGE_A, PR_LANGUAGE_W  <br/> |
|Identificador:  <br/> |0x3A0C  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

La cadena contiene un solo código de país o región de dos caracteres.

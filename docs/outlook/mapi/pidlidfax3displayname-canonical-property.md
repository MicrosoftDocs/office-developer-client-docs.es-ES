# <a name="pidlidfax3displayname-canonical-property"></a>Propiedad canónica PidLidFax3DisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre para mostrar de la otra dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3DisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080D0  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

# <a name="pidtagattachmenthidden-canonical-property"></a>Propiedad canónica PidTagAttachmentHidden

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si hay datos adjuntos ocultos para el usuario final.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ATTACHMENT_HIDDEN  <br/> |
|Identificador:  <br/> |0x7FFE  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Datos adjuntos del mensaje  <br/> |

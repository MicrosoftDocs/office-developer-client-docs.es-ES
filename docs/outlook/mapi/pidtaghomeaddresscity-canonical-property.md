# <a name="pidtaghomeaddresscity-canonical-property"></a>Propiedad canónica PidTagHomeAddressCity

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la ciudad de la dirección del destinatario particular.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_ADDRESS_CITY, PR_HOME_ADDRESS_CITY_A, PR_HOME_ADDRESS_CITY_W  <br/> |
|Identificador:  <br/> |0x3A59  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |

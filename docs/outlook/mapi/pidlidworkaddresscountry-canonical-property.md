# <a name="pidlidworkaddresscountry-canonical-property"></a>Propiedad canónica PidLidWorkAddressCountry

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte país o región de la dirección de trabajo del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddressCountry  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x00008049  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

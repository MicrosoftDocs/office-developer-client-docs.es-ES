# <a name="pidlidemail3emailaddress-canonical-property"></a>Propiedad canónica PidLidEmail3EmailAddress

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la tercera dirección de correo electrónico del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail3EmailAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080A3  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad debe ser adecuado para el tipo de dirección especificado para esta dirección de correo electrónico.

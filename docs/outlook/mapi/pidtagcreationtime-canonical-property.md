# <a name="pidtagcreationtime-canonical-property"></a>Propiedad canónica PidTagCreationTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la fecha y hora de creación de un mensaje. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CREATION_TIME  <br/> |
|Identificador:  <br/> |0x3007  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Hora del mensaje  <br/> |
   
## <a name="remarks"></a>Comentarios

Un almacén de mensajes establece esta propiedad para cada mensaje que crea.

**Se aplica a**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|Archivo de encabezado:  <br/> |Mapidefs.h  <br/> |
|Estructura relacionado:  <br/> |**FLATENTRY** <br/> |
   
```cpp

```

## <a name="parameters"></a>Parámetros

> Puntero a la estructura **FLATENTRY** existente.

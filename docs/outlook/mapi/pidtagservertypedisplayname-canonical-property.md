# <a name="pidtagservertypedisplayname-canonical-property"></a>Propiedad canónica PidTagServerTypeDisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la cadena Unicode que se va a mostrar en la barra de estado con esta propiedad.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SERVER_TYPE_DISPLAY_NAME  <br/> |
|Identificador:  <br/> |0x341D  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Común  <br/> |

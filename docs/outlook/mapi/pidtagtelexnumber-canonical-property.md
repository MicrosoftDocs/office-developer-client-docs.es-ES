# <a name="pidtagtelexnumber-canonical-property"></a>Propiedad canónica PidTagTelexNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de telex del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TELEX_NUMBER, PR_TELEX_NUMBER_A, PR_TELEX_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A2C  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso para un destinatario. El destinatario y la organización del destinatario definen estas propiedades.

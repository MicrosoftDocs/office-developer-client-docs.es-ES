# <a name="pidnamexsharingprovidername-canonical-property"></a>Propiedad canónica PidNameXSharingProviderName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Representa un nombre para mostrar del usuario del proveedor de uso compartido identificado por la propiedad **dispidSharingProviderGuid** ([PidLidSharingProviderGuid](pidlidsharingproviderguid-canonical-property.md)).
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |X-Sharing-Provider-Name  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Uso compartido  <br/> |

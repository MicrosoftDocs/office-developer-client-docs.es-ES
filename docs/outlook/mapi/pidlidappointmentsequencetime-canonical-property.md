# <a name="pidlidappointmentsequencetime-canonical-property"></a>Propiedad canónica PidLidAppointmentSequenceTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la fecha y hora en que se modificó por última vez esta propiedad.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptSeqTime  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Id. largo (LID):  <br/> |0x00008202  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor debe especificarse en hora universal coordinada (UTC).

# <a name="pidlidremoteattachment-canonical-property"></a>Propiedad canónica PidLidRemoteAttachment

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si un elemento remoto tiene datos adjuntos asociados a él.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidRemoteAttachment  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Remote  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008F07  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Mensaje remoto  <br/> |

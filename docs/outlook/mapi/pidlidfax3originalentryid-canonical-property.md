# <a name="pidlidfax3originalentryid-canonical-property"></a>Propiedad canónica PidLidFax3OriginalEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el EntryID original de la otra dirección de fax del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFax3OriginalEntryID  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Id. largo (LID):  <br/> |0x000080D5  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si está presente, debe especificar el EntryId único que corresponde a esta dirección de fax.

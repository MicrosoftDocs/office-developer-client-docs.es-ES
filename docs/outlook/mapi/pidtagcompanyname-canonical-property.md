# <a name="pidtagcompanyname-canonical-property"></a>Propiedad canónica PidTagCompanyName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre de la empresa del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_COMPANY_NAME, PR_COMPANY_NAME_A, PR_COMPANY_NAME_W  <br/> |
|Identificador:  <br/> |0x3A16  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan información de identificación y acceso sobre un destinatario. Los define el destinatario y la organización del destinatario.

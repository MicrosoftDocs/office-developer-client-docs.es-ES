# <a name="pidtagrecipienttrackstatustime-canonical-property"></a>Propiedad canónica PidTagRecipientTrackStatusTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la fecha y la hora cuando el asistente ha respondido.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RECIPIENT_TRACKSTATUS_TIME  <br/> |
|Identificador:  <br/> |0x5FFB  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Destinatario del transporte  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor debe especificarse en hora Universal coordinada (UTC).

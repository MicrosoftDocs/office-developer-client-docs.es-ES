# <a name="pidtagbusinesstelephonenumber-canonical-property"></a>Propiedad canónica PidTagBusinessTelephoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono principal del lugar del destinatario de negocio. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_BUSINESS_TELEPHONE_NUMBER, PR_BUSINESS_TELEPHONE_NUMBER_A, PR_BUSINESS_TELEPHONE_NUMBER_W, PR_OFFICE_TELEPHONE_NUMBER, PR_OFFICE_TELEPHONE_NUMBER_A, PR_OFFICE_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A08  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son ejemplos de las propiedades que proporciona la identificación y el acceso a información acerca de un destinatario. Estas propiedades son definidas por el destinatario y la organización del destinatario.

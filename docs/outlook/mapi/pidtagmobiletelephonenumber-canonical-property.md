# <a name="pidtagmobiletelephonenumber-canonical-property"></a>Propiedad canónica PidTagMobileTelephoneNumber

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono móvil del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MOBILE_TELEPHONE_NUMBER, PR_MOBILE_TELEPHONE_NUMBER_A, PR_MOBILE_TELEPHONE_NUMBER_W, PR_CELLULAR_TELEPHONE_NUMBER, PR_CELLULAR_TELEPHONE_NUMBER_A, PR_CELLULAR_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A1C  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Libreta de direcciones  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y su organización.

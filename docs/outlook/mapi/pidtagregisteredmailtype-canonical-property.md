# <a name="pidtagregisteredmailtype-canonical-property"></a>Propiedad canónica PidTagRegisteredMailType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Esta propiedad contiene el tipo de registro usado para la entrega física de un mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_REGISTERED_MAIL_TYPE  <br/> |
|Identificador:  <br/> |0x0C16  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

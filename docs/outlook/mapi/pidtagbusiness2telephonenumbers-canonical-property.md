# <a name="pidtagbusiness2telephonenumbers-canonical-property"></a>Propiedad canónica PidTagBusiness2TelephoneNumbers

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de números de teléfono para una empresa.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_BUSINESS2_TELEPHONE_NUMBER_A_MV  <br/> |
|Identificador:  <br/> |0x3A1B  <br/> |
|Tipo de datos:  <br/> |PT_MV_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se puede usar para enumerar números de teléfono alternativos o una lista de extensiones para una empresa.

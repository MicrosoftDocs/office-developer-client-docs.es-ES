# <a name="pidlidfreebusylocation-canonical-property"></a>Propiedad canónica PidLidFreeBusyLocation

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la ruta de acceso de la dirección URL desde la que un cliente puede recuperar la información de disponibilidad para el contacto como archivo iCal, tal como se especifica en [[ms-OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFreeBusyLocation  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x000080D8  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |

# <a name="pidtaglastverbexecutiontime-canonical-property"></a>Propiedad canónica PidTagLastVerbExecutionTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la hora en que se ejecutó el último verbo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_LAST_VERB_EXECUTION_TIME  <br/> |
|Identificador:  <br/> |0x1082  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Historial  <br/> |

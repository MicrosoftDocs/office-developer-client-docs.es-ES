# <a name="creating-a-profile"></a>Crear un perfil

  
  
**Hace referencia a**: Outlook 
  
Un cliente puede crear un perfil en una de estas dos formas, mediante el Asistente para perfiles, o mediante la escritura de código personalizado.
  
## <a name="in-this-section"></a>En esta sección

[Crear un perfil con el Asistente de perfiles](creating-a-profile-by-using-the-profile-wizard.md)
  
> Describe cómo invocar al Asistente para perfiles, una aplicación que crea un perfil al permitir al usuario elegir servicios de mensajes y proveedores de servicios de forma interactiva.
    
[Crear un perfil con código personalizado](creating-a-profile-by-using-custom-code.md)
  
> Describe cómo escribir código de C o C++, a menudo la mejor opción para las aplicaciones de cliente no interactivo que requieren un conjunto específico de servicios de mensajes.

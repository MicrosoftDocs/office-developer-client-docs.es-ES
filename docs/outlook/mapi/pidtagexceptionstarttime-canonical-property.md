# <a name="pidtagexceptionstarttime-canonical-property"></a>Propiedad canónica PidTagExceptionStartTime

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica la fecha y hora de inicio de la excepción en la zona horaria local del equipo cuando se crea la excepción.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_EXCEPTION_STARTTIME  <br/> |
|Identificador:  <br/> |0x7FFB  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Clase de mensaje no transmitible definida  <br/> |
   
## <a name="remarks"></a>Comentarios

> [!NOTE]
> Esta propiedad es informativa y no debe confiarse en la información importante.

# <a name="pidlidintendedbusystatus-canonical-property"></a>Propiedad canónica PidLidIntendedBusyStatus

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el valor de la propiedad **dispidBusyStatus** ([PidLidBusyStatus](pidlidbusystatus-canonical-property.md)) en la reunión en el calendario del organizador cuando se envió la solicitud de reunión o la actualización de la reunión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidIntendedBusyStatus  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008224  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Comentarios

Los valores permitidos de esta propiedad son los mismos que los de la propiedad **dispidBusyStatus**.

**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la **propiedad EntryID** del destinatario 
  
|||

|Propiedades asociadas:  <br/> |PR_RECIPIENT_ENTRYID  <br/> |

|Tipo de datos:  <br/> |PT_BINARY  <br/> |

# <a name="pidtagrulename-canonical-property"></a>Propiedad canónica PidTagRuleName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre legible para el usuario de la regla.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RULE_NAME  <br/> |
|Identificador:  <br/> |0x6682  <br/> |
|Tipo de datos:  <br/> |PT_STRING8  <br/> |
|Área:  <br/> |Reglas del lado servidor  <br/> |

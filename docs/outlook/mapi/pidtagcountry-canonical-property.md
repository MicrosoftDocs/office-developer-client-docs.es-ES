# <a name="pidtagcountry-canonical-property"></a>Propiedad canónica PidTagCountry

  
  
**Hace referencia a**: Outlook 
  
Contiene el nombre del país o la región del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_COUNTRY, PR_COUNTRY_A, PR_COUNTRY_W, PR_BUSINESS_ADDRESS_COUNTRY, PR_BUSINESS_ADDRESS_COUNTRY_A, PR_BUSINESS_ADDRESS_COUNTRY_W  <br/> |
|Identificador:  <br/> |0x3A26  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son ejemplos de las propiedades que proporciona la identificación y el acceso a información acerca de un destinatario. Estas propiedades son definidas por el destinatario y la organización del destinatario.

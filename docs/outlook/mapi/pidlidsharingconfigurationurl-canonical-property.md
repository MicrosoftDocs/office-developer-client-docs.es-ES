# <a name="pidlidsharingconfigurationurl-canonical-property"></a>Propiedad canónica PidLidSharingConfigurationUrl

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Designa como propiedad de un mensaje para compartir.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidSharingConfigUrl  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A24  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Compartir  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad debe ser una cadena de longitud cero.

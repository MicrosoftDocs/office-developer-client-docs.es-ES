# <a name="pidlidformstorage-canonical-property"></a>Propiedad canónica PidLidFormStorage

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Representa las propiedades del formulario que está asociada con el mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFormStorage  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000850F  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuración de tiempo de ejecución  <br/> |

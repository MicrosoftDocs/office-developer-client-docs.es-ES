# <a name="pidtaginternetcodepage-canonical-property"></a>Propiedad canónica PidTagInternetCodepage

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica la página de código usada para **PR_BODY** ([PidTagBody](pidtagbody-canonical-property.md)) o **PR_BODY_HTML** ([PidTagBodyHtml](pidtagbodyhtml-canonical-property.md)) propiedades.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INTERNET_CPID  <br/> |
|Identificador:  <br/> |0x3FDE  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Varios  <br/> |

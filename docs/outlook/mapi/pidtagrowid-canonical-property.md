# <a name="pidtagrowid-canonical-property"></a>Propiedad canónica PidTagRowid

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador único para un destinatario en una tabla de destinatarios o tabla de estado.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ROWID  <br/> |
|Identificador:  <br/> |0x3000  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MAPI común  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad es un valor temporal que solo es válido para la vida del objeto propietario de la tabla. Aparece como una columna de la tabla, pero no se almacena.

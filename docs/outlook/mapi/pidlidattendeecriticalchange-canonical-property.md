# <a name="pidlidattendeecriticalchange-canonical-property"></a>Propiedad canónica PidLidAttendeeCriticalChange

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la fecha y la hora en que se envió el objeto relacionado con la reunión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |LID_ATTENDEE_CRITICAL_CHANGE  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Meeting  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00000001  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor debe especificarse en el formato de hora universal coordinada (UTC).

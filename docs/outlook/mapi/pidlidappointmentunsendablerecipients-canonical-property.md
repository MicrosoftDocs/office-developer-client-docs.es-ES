# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Propiedad canónica PidLidAppointmentUnsendableRecipients

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de asistentes no enviados.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptUnsendableRecips  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x0000823D  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad no es necesaria, pero se debe establecer. Su formato se detalla en [[ms-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).

# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Propiedad canónico PidLidAppointmentUnsendableRecipients

  
  
**Se aplica a**: Outlook 
  
Contiene una lista de asistentes compuestas.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidApptUnsendableRecips  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000823D  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniones  <br/> |
   
## <a name="remarks"></a>Notas

Esta propiedad no es obligatoria, pero se debe establecer. Su formato se detalla en [[MS-OXOCAL]](http://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).

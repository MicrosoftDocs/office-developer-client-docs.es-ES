# <a name="pidnamemanager-canonical-property"></a>Propiedad canónica PidNameManager

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Especifica el administrador del archivo que se adjunta al documento.
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguno  <br/> |
|Conjunto de propiedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nombre de la propiedad:  <br/> |Manager  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Common  <br/> |

# <a name="pidtagrecipientdisplayname-canonical-property"></a>Propiedad canónica PidTagRecipientDisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el valor de la **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_RECIPIENT_DISPLAY_NAME, PR_RECIPIENT_DISPLAY_NAME_A, PR_RECIPIENT_DISPLAY_NAME_W  <br/> |
|Identificador:  <br/> |0x5FF6  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Destinatario de transporte  <br/> |

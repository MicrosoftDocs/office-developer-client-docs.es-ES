# <a name="pidtaghome2telephonenumbers-canonical-property"></a>Propiedad canónica PidTagHome2TelephoneNumbers

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una lista de números de teléfono secundarios en la casa del destinatario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME2_TELEPHONE_NUMBER_A_MV  <br/> |
|Identificador:  <br/> |0x3A2F  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE8  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |

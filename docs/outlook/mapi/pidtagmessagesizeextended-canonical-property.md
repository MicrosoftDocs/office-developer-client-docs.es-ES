# <a name="pidtagmessagesizeextended-canonical-property"></a>Propiedad canónica PidTagMessageSizeExtended

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una versión entera de 64 bits de la propiedad **PR_MESSAGE_SIZE** ([PidTagMessageSize](pidtagmessagesize-canonical-property.md)).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_MESSAGE_SIZE_EXTENDED  <br/> |
|Identificador:  <br/> |0x0E08  <br/> |
|Tipo de datos:  <br/> |PT_I8  <br/> |
|Área:  <br/> |Mensajes generales  <br/> |

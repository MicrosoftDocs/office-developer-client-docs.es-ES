# <a name="pidtaghomeaddresscountry-canonical-property"></a>Propiedad canónica PidTagHomeAddressCountry

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el condado en la dirección de un contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_ADDRESS_COUNTRY, PR_HOME_ADDRESS_COUNTRY_A, PR_HOME_ADDRESS_COUNTRY_W  <br/> |
|Identificador:  <br/> |0x3A5A  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades las define el usuario o la organización del usuario.

# <a name="pidtagpreferredbyname-canonical-property"></a>Propiedad canónica PidTagPreferredByName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el nombre que el destinatario prefiere que se llame.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PREFERRED_BY_NAME, PR_PREFERRED_BY_NAME_A, PR_PREFERRED_BY_NAME_W  <br/> |
|Identificador:  <br/> |0x3A47  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |

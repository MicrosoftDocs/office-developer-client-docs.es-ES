# <a name="pidtagscheduleinfodelegatorwantsinfo-canonical-property"></a>Propiedad canónica PidTagScheduleInfoDelegatorWantsInfo

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el delegador que desea recibir actualizaciones informativos.

|Propiedades asociadas:  <br/> |PR_SCHDINFO_BOSS_WANTS_INFO  <br/> |

|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |

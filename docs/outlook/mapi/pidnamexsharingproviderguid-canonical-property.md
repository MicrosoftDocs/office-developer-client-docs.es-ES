# <a name="pidnamexsharingproviderguid-canonical-property"></a>Propiedad canónica PidNameXSharingProviderGuid

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Representa el valor de la propiedad **dispidSharingProviderGuid** ([PidLidSharingProviderGuid](pidlidsharingproviderguid-canonical-property.md)).
  
|||
|:-----|:-----|
|Nombres descriptivos:  <br/> |Ninguna  <br/> |
|Conjunto de propiedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nombre de la propiedad:  <br/> |X-uso compartido-proveedor-Guid  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Uso compartido  <br/> |

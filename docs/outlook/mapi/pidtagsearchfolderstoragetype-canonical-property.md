# <a name="pidtagsearchfolderstoragetype-canonical-property"></a>Propiedad canónica PidTagSearchFolderStorageType

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene marcas que especifican los datos de objeto binario grande (BLOB) que aparecen en **la propiedad PR_WB_SF_DEFINITION** ([PidTagSearchFolderDefinition](pidtagsearchfolderdefinition-canonical-property.md)).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_WB_SF_STORAGE_TYPE  <br/> |
|Identificador:  <br/> |0x6846  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Búsqueda  <br/> |
   
## <a name="remarks"></a>Comentarios

Las definiciones de las marcas se especifican en [[MS-OXOSRCH]](https://msdn.microsoft.com/library/c72e49b8-78c7-4483-ad65-e46e9133673b%28Office.15%29.aspx). Busque **PR_WB_SF_STORAGE_TYPE**.

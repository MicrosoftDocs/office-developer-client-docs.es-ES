# <a name="pidtagroamingxmlstream-canonical-property"></a>Propiedad canónica PidTagRoamingXmlStream

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una secuencia XML arbitraria.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ROAMING_XMLSTREAM  <br/> |
|Identificador:  <br/> |0x7C08  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuración  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene una secuencia de datos XML arbitraria. Otras propiedades en el mensaje pueden implicar esquemas específicos que se utilizan en esta propiedad.

# <a name="pidtagabdefaultpab-canonical-property"></a>Propiedad canónica PidTagAbDefaultPab

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el identificador de entrada del contenedor de la libreta de direcciones que se usará como libreta de direcciones personal (PAB). 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_AB_DEFAULT_PAB  <br/> |
|Identificador:  <br/> |0x3D07  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones  <br/> |
   
## <a name="remarks"></a>Comentarios

No utilice esta propiedad. Está reservado para su uso por MAPI.

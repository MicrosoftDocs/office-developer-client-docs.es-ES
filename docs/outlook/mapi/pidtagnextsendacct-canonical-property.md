# <a name="pidtagnextsendacct-canonical-property"></a>Propiedad canónica PidTagNextSendAcct

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el servidor que un cliente está intentando usar actualmente para enviar correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_NEXT_SEND_ACCT  <br/> |
|Identificador:  <br/> |0x0E29  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Aplicación de Outlook  <br/> |
   
## <a name="remarks"></a>Comentarios

El formato de esta propiedad depende de la implementación. El cliente puede usar esta propiedad para determinar a qué servidor dirigir el correo electrónico, pero es opcional y el valor no tiene ningún significado para el servidor.

# <a name="saving-frequently-used-properties"></a>Guardar las propiedades de uso frecuente

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Mejorar el rendimiento mediante el almacenamiento de datos que se requiere tiempo y recursos para recuperar y se tiene acceso con frecuencia. Uso de memoria adicional a veces puede ser una mejor opción que se repite recuperaciones. Usar precaución y atención al crear una caché para almacenar estos datos. Tenga en cuenta que una memoria caché mal diseñada puede afectar negativamente al rendimiento.
  
Por ejemplo, los clientes más interpersonales de mensaje (IPM) necesitan mostrar y abrir el subárbol IPM de carpetas cuántas veces durante una sesión típica. Puede mejorar el rendimiento mediante el almacenamiento de los identificadores de entrada para cada una de estas carpetas.

# <a name="pidtagotheraddresscountry-canonical-property"></a>Propiedad canónica PidTagOtherAddressCountry

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el correo otro país o región del usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_OTHER_ADDRESS_COUNTRY, PR_OTHER_ADDRESS_COUNTRY_A, PR_OTHER_ADDRESS_COUNTRY_W  <br/> |
|Identificador:  <br/> |0x3A60  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Libreta de direcciones  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario o la organización del usuario.

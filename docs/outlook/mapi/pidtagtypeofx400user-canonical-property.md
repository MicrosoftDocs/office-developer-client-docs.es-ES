# <a name="pidtagtypeofx400user-canonical-property"></a>Propiedad canónica PidTagTypeOfX400User

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Esta propiedad contiene el tipo de destinatario de un mensaje para usarlo en un informe.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_TYPE_OF_MTS_USER  <br/> |
|Identificador:  <br/> |0x0C1C  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Destinatario MAPI  <br/> |

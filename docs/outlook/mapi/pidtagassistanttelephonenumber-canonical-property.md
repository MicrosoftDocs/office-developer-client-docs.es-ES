# <a name="pidtagassistanttelephonenumber-canonical-property"></a>Propiedad canónica PidTagAssistantTelephoneNumber

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene el número de teléfono del auxiliar administrativo del destinatario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ASSISTANT_TELEPHONE_NUMBER, PR_ASSISTANT_TELEPHONE_NUMBER_A, PR_ASSISTANT_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A2E  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades proporcionan identificación y obtener acceso a información de un destinatario. Se definen por el destinatario y su organización. 
  
El número de teléfono es para el Ayudante especificado en la propiedad **PR_ASSISTANT** ([pidtagassistant de MAPI](pidtagassistant-canonical-property.md)).

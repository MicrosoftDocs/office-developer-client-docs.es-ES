# <a name="pidlidfileunder-canonical-property"></a>Propiedad canónica PidLidFileUnder

  
  
**Hace referencia a**: Outlook 
  
Especifica el nombre en la que se archiva el contacto al mostrar una lista de contactos.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidFileUnder  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008005  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

La aplicación debe tratar esta propiedad como un PT_UNICODE vacía si se encuentra el contacto.

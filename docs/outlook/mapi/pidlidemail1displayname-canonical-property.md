# <a name="pidlidemail1displayname-canonical-property"></a>Propiedad canónico PidLidEmail1DisplayName

  
  
**Se aplica a**: Outlook 
  
Especifica el nombre para mostrar legible por el usuario para la primera dirección de correo electrónico.

|Tipo de datos:  <br/> |PT_UNICODE  <br/> |

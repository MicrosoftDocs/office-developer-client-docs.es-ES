# <a name="pidlidemail1displayname-canonical-property"></a>Propiedad canónica PidLidEmail1DisplayName

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el nombre para mostrar legible por el usuario para la primera dirección de correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail1DisplayName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008080  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

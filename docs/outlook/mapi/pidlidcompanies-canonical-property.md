# <a name="pidlidcompanies-canonical-property"></a>Propiedad canónica PidLidCompanies

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene los nombres de las empresas que están asociados con un elemento.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidCompanies  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008539  <br/> |
|Tipo de datos:  <br/> |PT_MV_UNICODE  <br/> |
|Área:  <br/> |General de mensajería  <br/> |

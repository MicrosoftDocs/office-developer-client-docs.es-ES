# <a name="pidtagconvertedeits-canonical-property"></a>Propiedad canónica PidTagConvertedEits

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador para los tipos de texto en un mensaje después de la conversión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONVERTED_EITS  <br/> |
|Identificador:  <br/> |0x000E  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad indica qué tipos de información codificada (EITs) se usaron para convertir los componentes de texto del mensaje.

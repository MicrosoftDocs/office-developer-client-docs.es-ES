# <a name="pidtagcontactaddressbookfolderentryid-canonical-property"></a>Propiedad canónica PidTagContactAddressBookFolderEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un identificador para la carpeta contactos en el almacén de mensajes donde se conservan los usuarios de correo. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_FOLDER_ENTRYID  <br/> |
|Identificador:  <br/> |0x6610  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Se trata de una propiedad de un contenedor de libreta de direcciones de contacto y de una columna en la tabla de contenedores de la libreta de direcciones de contacto.

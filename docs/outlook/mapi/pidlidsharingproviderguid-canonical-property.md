# <a name="pidlidsharingproviderguid-canonical-property"></a>Propiedad canónica PidLidSharingProviderGuid

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el identificador único global uso compartido de proveedor (GUID). Esto es una propiedad de un mensaje para compartir.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidSharingProviderGuid  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Sharing  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008A01  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Uso compartido  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad debe establecerse en "% xAE.F0.06.00.00.00.00.00.C0.00.00.00.00.00.00.46".

# <a name="pidtagcontactaddressbookstoresupportmask-canonical-property"></a>Propiedad canónica PidTagContactAddressBookStoreSupportMask

  
  
**Hace referencia a**: Outlook 
  
Contiene la propiedad **PR_STORE_SUPPORT_MASK** ([PidTagStoreSupportMask](pidtagcontactaddressbookstoresupportmask-canonical-property.md)) obtenida en el almacén que contiene la carpeta Contactos.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_STORE_SUPPORT_MASK  <br/> |
|Identificador:  <br/> |0x6611  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

El proveedor de la libreta de direcciones de contacto utiliza esta propiedad para evaluar la adecuación de características compatibles de la tienda. Se trata de una propiedad en un contenedor de la libreta de direcciones de contacto y una columna en la tabla de contenedores de la libreta de direcciones de contacto.

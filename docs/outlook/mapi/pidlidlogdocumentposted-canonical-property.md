# <a name="pidlidlogdocumentposted-canonical-property"></a>Propiedad canónica PidLidLogDocumentPosted

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica si el documento se envió por correo electrónico o se contabilizó en una carpeta del servidor durante el registro en diario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLogDocPosted  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Log  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008711  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Diario  <br/> |

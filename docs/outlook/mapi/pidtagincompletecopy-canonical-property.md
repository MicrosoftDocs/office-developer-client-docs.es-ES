# <a name="pidtagincompletecopy-canonical-property"></a>Propiedad canónico PidTagIncompleteCopy

  
  
**Se aplica a**: Outlook 
  
Contiene TRUE si este mensaje es una copia incompleta de otro mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INCOMPLETE_COPY  <br/> |
|Identificador:  <br/> |0x0035  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Propiedades del mensaje general  <br/> |

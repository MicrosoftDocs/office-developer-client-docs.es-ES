# <a name="pidtagsearchfoldertag-canonical-property"></a>Propiedad canónica PidTagSearchFolderTag

  
  
**Hace referencia a**: Outlook 
  
Contiene el valor utilizado para sincronizar este mensaje definición con el contenedor de carpeta de búsqueda coincidentes.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_WB_SF_TAG  <br/> |
|Identificador:  <br/> |0x6847  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Búsqueda  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se cambia cuando se modifica el mensaje de definición. Debe cambiar cada iteración, pero puede que no sea único.

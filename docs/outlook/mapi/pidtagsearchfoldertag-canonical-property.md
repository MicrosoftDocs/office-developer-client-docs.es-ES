# <a name="pidtagsearchfoldertag-canonical-property"></a>Propiedad canónica PidTagSearchFolderTag

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el valor que se usa para sincronizar este mensaje de definición con el contenedor de la carpeta de búsqueda correspondiente.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_WB_SF_TAG  <br/> |
|Identificador:  <br/> |0x6847  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Buscar   <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad se cambia cuando se modifica el mensaje de definición. Debe cambiar cada iteración, pero es posible que no sea única.

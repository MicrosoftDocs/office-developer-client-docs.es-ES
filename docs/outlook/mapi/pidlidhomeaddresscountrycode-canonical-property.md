# <a name="pidlidhomeaddresscountrycode-canonical-property"></a>Propiedad canónica PidLidHomeAddressCountryCode

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la parte de código de país de la dirección del contacto principal.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidHomeAddressCountryCode  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x000080DA  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |

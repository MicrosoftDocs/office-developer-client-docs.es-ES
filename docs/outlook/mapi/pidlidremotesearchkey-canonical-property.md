# <a name="pidlidremotesearchkey-canonical-property"></a>Propiedad canónica PidLidRemoteSearchKey

  
  
**Hace referencia a**: Outlook 
  
Indica la clave de búsqueda de un elemento remoto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidRemoteSearchKey  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Remote  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008F06  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Mensaje remoto  <br/> |

# <a name="pidlidcontactuserfield2-canonical-property"></a>Propiedad canónica PidLidContactUserField2

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica el segundo campo del contacto que está destinado a los diversos usos del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactUserField2  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|IDENTIFICADOR largo (LID):  <br/> |0x00008050  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene texto que no está relacionado con ninguna otra propiedad específica del contacto.

# <a name="pidlidcontactuserfield2-canonical-property"></a>Propiedad canónica PidLidContactUserField2

  
  
**Hace referencia a**: Outlook 
  
Especifica el segundo campo en el contacto al que está destinado a diversos usos del contacto.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactUserField2  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008050  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad contiene texto que no está relacionado con cualquier otra propiedad específicos de los contactos.

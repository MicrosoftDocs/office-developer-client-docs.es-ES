# <a name="pidtagcontactaddressbooksortflag-canonical-property"></a>Propiedad canónica PidTagContactAddressBookSortFlag

  
  
**Hace referencia a**: Outlook 
  
Contiene un indicador que es TRUE cuando contactos deben ordenarse por archivo bajo y FALSE cuando se debe ordenar por nombre para mostrar. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_SORT_FLAG  <br/> |
|Identificador:  <br/> |0x6602  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Se trata de una propiedad en una sección de perfil de la libreta de direcciones de contacto.

# <a name="pidtagcontactaddressbooksortflag-canonical-property"></a>Propiedad canónica PidTagContactAddressBookSortFlag

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una marca que es TRUE cuando los contactos deben ordenarse por archivo y FALSE cuando deben ordenarse por nombre para mostrar. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONTAB_SORT_FLAG  <br/> |
|Identificador:  <br/> |0x6602  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Libreta de direcciones de contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Se trata de una propiedad de una sección de Perfil de la libreta de direcciones de contacto.

# <a name="pidtagconflictentryid-canonical-property"></a>Propiedad canónica PidTagConflictEntryId

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene el EntryID del mensaje de resolución de conflictos.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_CONFLICT_ENTRYID  <br/> |
|Identificador:  <br/> |0x3FF0  <br/> |
|Tipo de datos:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Sincronización de cambio incremental (ICS)  <br/> |

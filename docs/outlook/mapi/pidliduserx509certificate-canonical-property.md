# <a name="pidliduserx509certificate-canonical-property"></a>Propiedad canónica PidLidUserX509Certificate

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene certificados de seguridad X.509 versión 3 para un usuario de mensajería.

|Id. largo (LID):  <br/> |0x000080D9  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |

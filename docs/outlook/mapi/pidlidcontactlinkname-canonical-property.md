# <a name="pidlidcontactlinkname-canonical-property"></a>Propiedad canónico PidLidContactLinkName

  
  
**Se aplica a**: Outlook 
  
Contiene los elementos de la propiedad **dispidContacts** ([PidLidContacts](pidlidcontacts-canonical-property.md)).
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidContactLinkName  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Common  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008586  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

Los elementos de la propiedad **dispidContactLinkName** están separados por un punto y coma y un espacio (";").

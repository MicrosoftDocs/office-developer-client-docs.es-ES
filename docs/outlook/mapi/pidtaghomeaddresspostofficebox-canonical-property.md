# <a name="pidtaghomeaddresspostofficebox-canonical-property"></a>Propiedad canónica PidTagHomeAddressPostOfficeBox

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la información de apartado de correos de la dirección de un usuario.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_HOME_ADDRESS_POST_OFFICE_BOX, PR_HOME_ADDRESS_POST_OFFICE_BOX_A, PR_HOME_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Identificador:  <br/> |0x3A5E  <br/> |
|Tipo de datos:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuario de correo MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Estas propiedades son definidas por el usuario o la organización del usuario.

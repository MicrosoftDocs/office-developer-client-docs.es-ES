# <a name="pidtagprimarysendaccount-canonical-property"></a>Propiedad canónica PidTagPrimarySendAccount

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene una cadena que nombra al primer servidor que se usa para enviar el mensaje.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_PRIMARY_SEND_ACCOUNT  <br/> |
|Identificador:  <br/> |0x0E28  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Cuenta  <br/> |
   
## <a name="remarks"></a>Comentarios

Especifica el primer servidor que un cliente debe usar para enviar el correo. El formato de estas propiedades depende de la implementación. El cliente puede usar estas propiedades para determinar a qué servidor dirigir el correo, pero es opcional y el valor no tiene ningún significado para el servidor.

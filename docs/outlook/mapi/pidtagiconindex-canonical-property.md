# <a name="pidtagiconindex-canonical-property"></a>Propiedad canónica PidTagIconIndex

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene un número que indica qué icono usar al mostrar un grupo de objetos de correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_ICON_INDEX  <br/> |
|Identificador:  <br/> |0x1080  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Mensajería general  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad, si existe, es una sugerencia para el cliente. El cliente puede omitir el valor de esta propiedad.

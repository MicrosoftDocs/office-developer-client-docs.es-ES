# <a name="pidlidemail1addresstype-canonical-property"></a>Propiedad canónico PidLidEmail1AddressType

  
  
**Se aplica a**: Outlook 
  
Especifica el tipo de dirección de la primera dirección de correo electrónico.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidEmail1AddrType  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008082  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Notas

Si está presente, el valor de esta propiedad debe ser un tipo de dirección válida.

# <a name="pidlidissilent-canonical-property"></a>Propiedad canónica PidLidIsSilent

  
  
**Hace referencia a**: Outlook 
  
Indica que el usuario no incluía ningún texto en el cuerpo de la respuesta a la reunión.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |LID_IS_SILENT  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Meeting  <br/> |
|Identificador de tipo Long (LID):  <br/> |0 x 00000004  <br/> |
|Tipo de datos:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniones  <br/> |

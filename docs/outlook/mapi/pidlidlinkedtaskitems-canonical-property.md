# <a name="pidlidlinkedtaskitems-canonical-property"></a>Propiedad canónica PidLidLinkedTaskItems

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica una lista de la propiedad **PR_ENTRYID** ([PidTagEntryId](pidtagentryid-canonical-property.md)) de las tareas relacionadas con el calendario. 
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidLinkedTaskItems  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Appointment  <br/> |
|Id. largo (LID):  <br/> |0x0000820C  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Tarea  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad no es necesaria.

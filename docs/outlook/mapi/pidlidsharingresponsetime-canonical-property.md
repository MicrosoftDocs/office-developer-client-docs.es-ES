# <a name="pidlidsharingresponsetime-canonical-property"></a>Propiedad canónico PidLidSharingResponseTime

  
  
**Se aplica a**: Outlook 
  
Especifica la hora en que el destinatario de la solicitud para compartir enviado una respuesta para compartir. Esto es una propiedad de un mensaje para compartir.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidSharingResponseTime  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Sharing  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x00008A28  <br/> |
|Tipo de datos:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Uso compartido  <br/> |

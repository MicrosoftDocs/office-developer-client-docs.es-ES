# <a name="pidlidworkaddress-canonical-property"></a>Propiedad canónica PidLidWorkAddress

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Especifica la dirección del contacto completar el trabajo.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |dispidWorkAddress  <br/> |
|Conjunto de propiedades:  <br/> |PSETID_Address  <br/> |
|Identificador de tipo Long (LID):  <br/> |0x0000801B  <br/> |
|Tipo de datos:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contacto  <br/> |
   
## <a name="remarks"></a>Comentarios

Esta propiedad debe ser una combinación de las demás propiedades de la dirección física y se basa en la configuración regional del cliente.

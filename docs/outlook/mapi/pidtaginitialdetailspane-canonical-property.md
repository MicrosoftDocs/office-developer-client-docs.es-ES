# <a name="pidtaginitialdetailspane-canonical-property"></a>Propiedad canónica PidTagInitialDetailsPane

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Indica la página de una plantilla para mostrar que se va a mostrar en primer lugar.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_INITIAL_DETAILS_PANE  <br/> |
|Identificador:  <br/> |0x3F08  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Tablas de presentación MAPI  <br/> |
   
## <a name="remarks"></a>Comentarios

Debe estar presente en todos los objetos de la libreta de direcciones en un servidor de interfaz del proveedor de servicios de nombres (NSPI) y debe tener el valor cero (0). No se debe definir para ningún objeto de una libreta de direcciones sin conexión.

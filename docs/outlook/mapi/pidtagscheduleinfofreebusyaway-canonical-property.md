# <a name="pidtagscheduleinfofreebusyaway-canonical-property"></a>Propiedad canónica PidTagScheduleInfoFreeBusyAway

  
  
**Hace referencia a**: Outlook 2013 | Outlook 2016 
  
Contiene las horas para la que se establece el estado de disponibilidad para OOF.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SCHDINFO_FREEBUSY_OOF  <br/> |
|Identificador:  <br/> |0x6856  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Libre/ocupado  <br/> |
   
## <a name="remarks"></a>Comentarios

El formato, el cálculo y las restricciones de esta propiedad son los mismos que los de **PR_SCHDINFO_FREEBUSY_TENTATIVE** ([PidTagScheduleInfoFreeBusyTentative](pidtagscheduleinfofreebusytentative-canonical-property.md)) pero hacer referencia a las citas que están marcadas OOF en el calendario asociado.

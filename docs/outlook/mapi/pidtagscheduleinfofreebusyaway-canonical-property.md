# <a name="pidtagscheduleinfofreebusyaway-canonical-property"></a>Propiedad canónica PidTagScheduleInfoFreeBusyAway

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene las horas en las que se establece el estado de disponibilidad en OOF.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_SCHDINFO_FREEBUSY_OOF  <br/> |
|Identificador:  <br/> |0x6856  <br/> |
|Tipo de datos:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Disponibilidad  <br/> |
   
## <a name="remarks"></a>Comentarios

El formato, las computaciones y las restricciones de esta propiedad son los mismos que los de **PR_SCHDINFO_FREEBUSY_TENTATIVE** ([PidTagScheduleInfoFreeBusyTentative](pidtagscheduleinfofreebusytentative-canonical-property.md)), pero hacen referencia a las citas que están marcadas como OOF en el calendario asociado.

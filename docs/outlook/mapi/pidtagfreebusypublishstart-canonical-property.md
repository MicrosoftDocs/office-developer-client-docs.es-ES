# <a name="pidtagfreebusypublishstart-canonical-property"></a>Propiedad canónica PidTagFreeBusyPublishStart

  
  
**Se aplica a**: Outlook 2013 | Outlook 2016 
  
Contiene la hora de inicio del intervalo de publicación.
  
|||
|:-----|:-----|
|Propiedades asociadas:  <br/> |PR_FREEBUSY_PUBLISH_START  <br/> |
|Identificador:  <br/> |0x6847  <br/> |
|Tipo de datos:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Libre/ocupado  <br/> |
   
## <a name="remarks"></a>Comentarios

El valor de esta propiedad es el número de minutos transcurridos desde la medianoche del 1 de enero de 1601 en hora Universal coordinada (UTC).

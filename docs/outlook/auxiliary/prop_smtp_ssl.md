# <a name="prop_smtp_ssl"></a>PROP_SMTP_SSL

Especifica si se va a usar el protocolo capa de sockets seguros (SSL) para la cuenta SMTP.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Identificador:  <br/> |0x0202  <br/> |
|Tipo de propiedad:  <br/> |PT_DWORD  <br/> |
|Etiqueta de propiedad:  <br/> |0x02020003  <br/> |
|Acceso:  <br/> |Solo lectura  <br/> |
   
## <a name="remarks"></a>Comentarios

Un valor cero significa no usar el cifrado SSL; de lo contrario, use el cifrado SSL.

# <a name="propsmtpssl"></a>PROP_SMTP_SSL

Especifica si se debe usar el protocolo de capa de sockets seguros (SSL) para la cuenta de SMTP.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Identificador:  <br/> |valor 0 x 0202  <br/> |
|Tipo de propiedad:  <br/> |PT_DWORD  <br/> |
|Etiqueta de la propiedad:  <br/> |0x02020003  <br/> |
|Access:  <br/> |Solo lectura  <br/> |
   
## <a name="remarks"></a>Comentarios

Un medio de valor cero no utilizar el cifrado SSL, en caso contrario, use el cifrado SSL.

# <a name="propsmtpssl"></a>PROP_SMTP_SSL

Especifica si se debe usar el protocolo de capa de sockets seguros (SSL) para la cuenta SMTP.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Identificador:  <br/> |0x0202  <br/> |
|Tipo de propiedad:  <br/> |PT_DWORD  <br/> |
|Etiqueta de propiedad:  <br/> |0x02020003  <br/> |
|Al  <br/> |Solo lectura  <br/> |
   
## <a name="remarks"></a>Comentarios

Un valor de cero significa que no se usa el cifrado SSL; de lo contrario, se usa el cifrado SSL.

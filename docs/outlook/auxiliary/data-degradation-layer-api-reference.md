# <a name="data-degradation-layer-api-reference"></a>Referencia de api de capa de degradación de datos

Esta sección proporciona contenido de referencia para la API de capa de degradación de datos, que proporciona la capacidad de ocultar el formato de carácter nativo subyacente de un objeto Outlook, lo que permite que un cliente acceda al objeto en un formato de carácter preferido en lugar de en el formato de carácter nativo del objeto.

# <a name="interfaces-account-management-api"></a>Interfaces (API de administración de cuenta)

En esta sección se describe las interfaces de la API de administración de cuenta.
  
|**Interfaz**|**Descripción**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |Admite la obtención y definición de propiedades y otra información acerca de una cuenta.  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |Proporciona funcionalidad auxiliar en la sesión MAPI actual para administrar las cuentas.  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |Administra el acceso a las cuentas y configura las notificaciones sobre cambios de cuenta.  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |Proporciona una devolución de llamada al cliente para que los cambios a una cuenta.  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |Admite enumerar cuentas como objetos de [IUnknown](https://docs.microsoft.com/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) .  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |Proporciona información adicional sobre el último error.  <br/> |

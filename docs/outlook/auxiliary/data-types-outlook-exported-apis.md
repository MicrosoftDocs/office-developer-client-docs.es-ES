# <a name="data-types-outlook-exported-apis"></a>Tipos de datos (API de exportadas de Outlook)

En esta sección se describe los tipos de datos que se exporta de Outlook.
  
|**Tipo de datos**|**Descripción**|
|:-----|:-----|
|[TZDEFINITION](tzdefinition.md) <br/> |Representa una zona horaria toda incluidas todas las reglas de MAYÚS históricas, actuales y futuros zona horaria para el horario de verano.  <br/> |
|[TZREG](tzreg.md) <br/> |Define cuándo se inicia el tiempo de horario de verano, cuando se produce la devolución a la hora estándar y es el turno de verano de cuántas horas.  <br/> |
|[TZRULE](tzrule.md) <br/> |Especifica la información de una regla de zona horaria acerca de cuándo se inicia el horario de verano y el año en el que dicha regla de zona horaria en primer lugar en vigor.  <br/> |

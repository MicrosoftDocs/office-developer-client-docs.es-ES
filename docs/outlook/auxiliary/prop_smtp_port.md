# <a name="propsmtpport"></a>PROP_SMTP_PORT

Representa el número de puerto de la cuenta de SMTP.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Identificador:  <br/> |espera 0 x 0201  <br/> |
|Tipo de propiedad:  <br/> |PT_DWORD  <br/> |
|Etiqueta de la propiedad:  <br/> |0x02010003  <br/> |
|Access:  <br/> |Solo lectura  <br/> |

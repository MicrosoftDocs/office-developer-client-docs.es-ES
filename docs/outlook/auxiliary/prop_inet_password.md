Representa la contraseña de usuario para un buzón de correo de Internet general.

|Tipo de propiedad:  <br/> |PT_UNICODE|SECURE_FLAG  <br/> |
|Etiqueta de la propiedad:  <br/> |0x0102801F  <br/> |

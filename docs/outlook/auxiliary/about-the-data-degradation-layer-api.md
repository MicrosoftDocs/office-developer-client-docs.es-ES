# <a name="about-the-data-degradation-layer-api"></a>Información sobre la API de la capa de degradación de datos

La API de capa de degradación de datos proporciona la capacidad de ocultar el formato de carácter nativo subyacente de un objeto de Outlook, lo que permite a un cliente tener acceso al objeto en un formato de carácter preferido en lugar de en el formato de carácter nativo del objeto. Proporciona las definiciones de constantes que se muestran en constantes [(API](constants-data-degradation-layer-api.md)de capa de degradación de datos) y la función [HrCreateNewWrappedObject](hrcreatenewwrappedobject.md) que devuelve un objeto de Outlook ajustado en un formato especificado.

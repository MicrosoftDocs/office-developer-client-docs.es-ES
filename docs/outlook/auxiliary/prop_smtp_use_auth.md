# <a name="propsmtpuseauth"></a>PROP_SMTP_USE_AUTH

Especifica si se debe usar la autenticación para la cuenta de SMTP.
  
## <a name="quick-info"></a>Información rápida

|||
|:-----|:-----|
|Identificador:  <br/> |0x0203  <br/> |
|Tipo de propiedad:  <br/> |PT_DWORD  <br/> |
|Etiqueta de la propiedad:  <br/> |0x02030003  <br/> |
|Access:  <br/> |Solo lectura  <br/> |
   
## <a name="remarks"></a>Comentarios

Un medio de valor cero no usar la autenticación, en caso contrario, usar la autenticación.

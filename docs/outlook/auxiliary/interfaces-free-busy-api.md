# <a name="interfaces-freebusy-api"></a>Interfaces (API de libre/ocupado)

En esta sección se describe las interfaces de la API de la información de disponibilidad.
  
|**Interfaz**|**Descripción**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |Admite acceso y enumerar los bloques de disponibilidad de datos para un usuario dentro de un intervalo de tiempo.  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |Para un usuario determinado, obtiene y establece un intervalo de tiempo y devuelve una interfaz para enumerar los bloques de libre/ocupado de datos dentro de este intervalo de tiempo.  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |Admite la especificación de las interfaces que tienen acceso a datos de disponibilidad para los usuarios especificados.  <br/> |

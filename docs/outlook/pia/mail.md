# <a name="mail"></a>Correo

Esta sección proporciona tareas de ejemplo que implican elementos de correo. Un elemento de correo representa un mensaje de correo en una carpeta.

## <a name="in-this-section"></a>En esta sección

|Tema|Descripción|
|:----|:----------|
|[Crear un elemento de correo con una plantilla de mensaje](how-to-create-a-mail-item-by-using-a-message-template.md)  |Crea un elemento de correo mediante el método [CreateItemFromTemplate](https://msdn.microsoft.com/library/bb611329\(v=office.15\)).|
|[Create a mail item, attach a report, and send the mail item to the user's manager](how-to-create-a-mail-item-attach-a-report-and-send-the-mail-item-to-the-user-s-manager.md) (Crear un elemento de correo, adjuntar un informe y enviar el elemento de correo al jefe del usuario)  |Crea un elemento de correo que tiene datos adjuntos y se lo envía al administrador del usuario.|
|[Send an email given the SMTP address of an account](how-to-send-an-e-mail-given-the-smtp-address-of-an-account.md) (Enviar un correo electrónico según la dirección SMTP de una cuenta) |Crea un mensaje de correo electrónico y lo envía desde una cuenta de Microsoft Outlook, dada la dirección de Protocolo simple de transferencia de correo (SMTP) de esa cuenta. |
|[Agregar opciones de votación a un elemento de correo](how-to-add-voting-options-to-a-mail-item.md) |Usa la propiedad [VotingOptions](https://msdn.microsoft.com/library/bb652695\(v=office.15\)) del objeto [MailItem](https://msdn.microsoft.com/library/bb643865\(v=office.15\)) para agregar opciones de votación a un mensaje de correo electrónico.|
|[Add a custom action as a response to a mail item](how-to-add-a-custom-action-as-a-response-to-a-mail-item.md) (Agregar una acción personalizada como respuesta a un elemento de correo)  |Agrega acciones personalizadas como respuesta a un elemento de correo electrónico mediante el método [Add()](https://msdn.microsoft.com/library/bb612077\(v=office.15\)) de la colección [Actions](https://msdn.microsoft.com/library/bb611963\(v=office.15\)).|
|[Get the SMTP address of the sender of a mail item](how-to-get-the-smtp-address-of-the-sender-of-a-mail-item.md) (Obtener la dirección SMTP del remitente de un elemento de correo)  |Obtiene la dirección del Protocolo simple de transferencia de correo (SMTP) de un elemento de correo.|
|[Especificar diferentes tipos de destinatarios de un elemento de correo](how-to-specify-different-recipient-types-for-a-mail-item.md) |Establece mediante programación diferentes tipos de destinatarios (Para, CC o CCO) para un elemento de correo.|

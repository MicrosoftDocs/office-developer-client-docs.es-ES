# <a name="sessions"></a>Sesiones

Esta sección incluye una tarea de ejemplo que hace referencia a las sesiones de Microsoft Outlook. Una sesión es un período de tiempo durante el cual un usuario inicia sesión en Outlook. Una sesión se representa en Outlook mediante la clase [NameSpace](https://msdn.microsoft.com/library/bb645857\(v=office.15\)). Esta clase admite el inicio y el cierre de sesión, tener acceso a los objetos de almacenamiento directamente mediante su id., tener acceso directamente a algunas carpetas predeterminadas especiales y tener acceso a los orígenes de datos que son propiedad de otros usuarios. El único origen de datos admitido es MAPI, que permite el acceso a todos los datos de Outlook almacenados en los almacenes de correo del usuario.

# <a name="conversations"></a>Conversaciones

En esta sección se proporcionan tareas de ejemplo relacionadas con las conversaciones. Las conversaciones en Outlook se representan mediante el objeto [Conversation](https://msdn.microsoft.com/library/ff184711\(v=office.15\)) e incluyen uno o más elementos en una carpeta que comparten el mismo título o conversación del asunto.

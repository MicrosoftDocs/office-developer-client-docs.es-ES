# <a name="views"></a>Vistas

Esta sección proporciona tareas de ejemplo que implican vistas. Las vistas se usan para ordenar, agrupar y ver datos de todo tipo.

## <a name="in-this-section"></a>En esta sección

|Tema|Descripción|
|:----|:----------|
|[Create a view](how-to-create-a-view.md) (Crear una vista)  |Usa el método [Add(String, OlViewType, OlViewSaveOption)](https://msdn.microsoft.com/library/bb643986\(v=office.15\)) de la colección [Views](https://msdn.microsoft.com/library/bb644226\(v=office.15\)) para crear una vista para un objeto [Folder](https://msdn.microsoft.com/library/bb645774\(v=office.15\)).|
|[Add fields to a view](how-to-add-fields-to-a-view.md) (Agregar campos a una vista)  |Personaliza una vista mediante el método [Add(String)](https://msdn.microsoft.com/library/bb646040\(v=office.15\)) de la colección [ViewFields](https://msdn.microsoft.com/library/bb645950\(v=office.15\)) para agregar campos a una vista.|
|[Enumerar elementos de una vista de tabla](how-to-enumerate-items-in-a-table-view.md)  |Enumera los elementos en una vista de tabla utilizando el método [GetTable()](https://msdn.microsoft.com/library/ff184699\(v=office.15\)).|

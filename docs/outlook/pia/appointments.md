# <a name="appointments"></a>Citas

En esta sección se proporcionan tareas de ejemplo relacionadas con las citas. Las citas pueden ser, por ejemplo, eventos, reuniones y citas periódicas. Outlook usa el objeto [AppointmentItem](https://msdn.microsoft.com/library/bb645611\(v=office.15\)) para representar las citas en la carpeta Calendario.

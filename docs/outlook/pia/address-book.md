# <a name="address-book"></a>Libreta de direcciones

En esta sección, se proporcionan tareas de ejemplo relacionadas con las libretas de direcciones. Las libretas de direcciones se usan para proporcionar un sistema de direccionamiento para que los usuarios se comuniquen electrónicamente. La lista global de direcciones (LGD) de Exchange, la libreta de direcciones de Outlook, los contenedores de Exchange y la libreta de direcciones sin conexión (OAB) son ejemplos de proveedores de libretas de direcciones.

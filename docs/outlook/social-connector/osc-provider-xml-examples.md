# <a name="osc-provider-xml-examples"></a>Ejemplos xml del proveedor de OSC

Esta sección contiene ejemplos XML que usan las capacidades, amigos y elementos XML **activityFeed,** cumpliendo con el esquema XML del proveedor de Outlook Social Connector (OSC). 
  
- [En el ejemplo XML](capabilities-xml-example.md) de **funcionalidades** se muestran las capacidades XML de un proveedor de OSC, especificando sus capacidades y requisitos para el OSC. 
    
- [En el ejemplo XML](friends-xml-example.md) de amigos se **muestra** el XML de amigos de dos personas en la red social. 
    
- [En el ejemplo XML de](activity-feed-xml-example.md) la fuente de actividades se muestra el XML de **activityFeed** de cuatro actividades: dos actualizaciones de imagen de perfil, una actualización de estado y una entrada de blog. El XML también especifica tres plantillas para mostrar actividades para mostrar las actividades correspondientes.

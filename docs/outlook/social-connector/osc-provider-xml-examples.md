# <a name="osc-provider-xml-examples"></a>Ejemplos xml del proveedor de OSC

Esta sección contiene ejemplos XML que usan las funcionalidades, los amigos y los elementos XML **activityFeed,** que cumplen con el esquema XML del proveedor Outlook Social Connector (OSC). 
  
- [Ejemplo XML de funcionalidades](capabilities-xml-example.md) muestra **las capacidades** XML de un proveedor de OSC, especificando sus capacidades y requisitos para el OSC. 
    
- [Ejemplo xml de amigos](friends-xml-example.md) muestra el XML **de amigos** para dos personas en la red social. 
    
- [Ejemplo XML de](activity-feed-xml-example.md) fuente de actividad muestra **el xml activityFeed** de cuatro actividades: dos actualizaciones de imágenes de perfil, una actualización de estado y una entrada de blog. El XML también especifica tres plantillas de presentación de actividad para mostrar las actividades correspondientes.

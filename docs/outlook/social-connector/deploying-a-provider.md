# <a name="deploying-a-provider"></a>Implementar un proveedor

El método recomendado para implementar un proveedor Outlook Social Connector (OSC) es usar un paquete Windows Installer (.msi). Después de descargar el paquete de instalación en un disco duro local, Office usuario final instala el proveedor de OSC. Una vez completada la instalación del proveedor, el usuario puede conectarse al proveedor de OSC y proporcionar credenciales de inicio de sesión a la cuenta de red social. Si el usuario decide desinstalar el proveedor de OSC, el usuario puede usar Programas **y características** en el Panel de control para desinstalar.

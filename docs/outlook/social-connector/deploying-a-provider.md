# <a name="deploying-a-provider"></a>Implementar un proveedor

El método recomendado para implementar un proveedor de Outlook Social Connector (OSC) es usar un paquete de Windows Installer (. msi). Después de descargar el paquete de instalación en un disco duro local, el usuario final de Office instala el proveedor de OSC. Una vez completada la instalación del proveedor, el usuario puede conectarse al proveedor OSC y proporcionar las credenciales de inicio de sesión a la cuenta de red social. Si el usuario decide desinstalar el proveedor OSC, el usuario puede usar **programas y características** del panel de control para desinstalar.

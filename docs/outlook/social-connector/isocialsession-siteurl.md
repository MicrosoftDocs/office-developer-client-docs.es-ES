# <a name="isocialsessionsiteurl"></a>ISocialSession::SiteUrl

Establece la dirección URL del sitio de red social. 
  
```cpp
[propput] HRESULT _stdcall siteUrl([in] BSTR rhs);
```

## <a name="property-value"></a>Valor de propiedad

Una cadena que representa una dirección URL de un sitio de red social.

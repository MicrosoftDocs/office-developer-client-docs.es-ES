# <a name="c-api-functions-that-can-be-called-only-from-a-dll-or-xll"></a>Funciones de la API de C que se pueden llamar solo desde una DLL o XLL

**Se aplica a**: Excel 2013 | Office 2013 | Visual Studio 
  
La API de C proporciona funciones de devolución de llamada de Microsoft Excel 15 que sólo se pueden llamar mediante la **Excel4**, **Excel4v**, **Excel12**o funciones **Excel12v** (o por una de estas funciones indirectamente utilizando las funciones de Framework **Excel **o **Excel12f**). Esto significa que sólo se pueden llamar desde una DLL o XLL.

# <a name="welcome-to-the-excel-software-development-kit"></a>Kit de desarrollo de software Excel

 **Hace referencia a**: Excel 2013 | Office 2013 | Visual Studio 
  
Bienvenido a la documentación del Kit de desarrollo de software de XLL (SDK) de Excel 2013. Esta referencia contiene información general conceptual, tareas de programación y ejemplos para ayudarle a desarrollar los XLL de Microsoft Excel 2013.
  
Revisado: Noviembre de 2012
  
Descargue el [SDK de XLL de Excel 2013](https://go.microsoft.com/fwlink/?LinkID=251082&amp;clcid=0x409).
  
El SDK de XLL de Excel 2013 incluye lo siguiente:
  
- **Interfaz de programación de aplicaciones C (API)**: incluye archivos de origen y de encabezado que permiten que los DLL accedan a la funcionalidad de Excel 2013 y una descripción de la interfaz que debe exponer un DLL para trabajar con el Administrador de complementos de Excel.
    
- **Proyectos de Microsoft Visual Studio**: incluye el código fuente de C/C++ y muestra cómo usar la API de C. Estos proyectos de ejemplo proporcionan ejemplo y sirven como punto de inicio para el desarrollo de complementos.
    
La documentación del SDK contiene las siguientes secciones:
  
- [Introducción al SDK de XLL de Excel 2013](getting-started-with-the-excel-xll-sdk.md)
    
- [Desarrollo de XLL de Excel de 2013](developing-excel-xlls.md)
    
- [Desarrollar conectores clúster de Excel](developing-excel-cluster-connectors.md)
    
- [Referencia de funciones de API de SDK de XLL de Excel 2013](excel-xll-sdk-api-function-reference.md)
    
## <a name="functionality-not-covered"></a>Funcionalidades no cubiertas

No se tratan los siguientes temas:
  
- Desarrollo de funciones definidas por el usuario y comandos en hojas de macros de Excel (XLM).
    
- Creación de funciones definidas por el usuario en DLL que controlan el flujo de ejecución de una macro XLM.
    
    Estas funciones funcionan devolviendo un tipo de datos de control de flujo especial, que tampoco se describe en esta documentación.

# <a name="getting-started-with-the-excel-xll-sdk"></a>Introducción al SDK de XLL de Excel

**Hace referencia a**: Excel 2013 | Office 2013 | Visual Studio 
  
El kit de desarrollo de software (SDK) de Microsoft Excel 2013 XLL proporciona información para ayudarle a empezar a crear complementos DLL para Excel 2013 que usan la API de C. En esta sección se proporciona información sobre la API de C y también vínculos a documentación de referencia para ayudarle a empezar a trabajar con el desarrollo de XLL de Excel 2013.
  
## <a name="in-this-section"></a>En esta sección

- [Programar con la API de c en Excel](programming-with-the-c-api-in-excel.md): presenta la API de c y explica cómo se compara con otras opciones de programación en Excel 2013.
    
- Novedades [de la API de c para Excel](what-s-new-in-the-c-api-for-excel.md): describe las actualizaciones de la API de c para Excel 2013.
    
- [Problemas conocidos en el desarrollo de XLL de Excel](known-issues-in-excel-xll-development.md): describe los problemas encontrados en la programación de XLL de Excel.
    
## <a name="related-sections"></a>Secciones relacionadas

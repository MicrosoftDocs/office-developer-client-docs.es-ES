# <a name="getting-started-with-the-excel-xll-sdk"></a>Introducción al SDK de XLL de Excel

**Hace referencia a**: Excel 2013 | Office 2013 | Visual Studio 
  
El Kit de desarrollo de software (SDK) de XLL de Microsoft Excel 2013 proporciona información para ayudarle a empezar a crear complementos DLL para Excel 2013 que usan la API de C. En esta sección se proporciona información sobre la API de C y también vínculos a documentación de referencia para ayudarle a empezar a desarrollar XLL de Excel 2013.
  
## <a name="in-this-section"></a>En esta sección

- [Programación con la API de C](programming-with-the-c-api-in-excel.md)en Excel: presenta la API de C y describe cómo se compara con otras opciones de programación en Excel 2013.
    
- [Novedades de la API](what-s-new-in-the-c-api-for-excel.md)de C para Excel: describe las actualizaciones de la API de C para Excel 2013.
    
- [Problemas conocidos en el desarrollo de XLL](known-issues-in-excel-xll-development.md)de Excel: analiza los problemas detectados en la programación XLL de Excel.

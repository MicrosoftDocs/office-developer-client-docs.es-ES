# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Instalar la Ayuda local de .NET Framework para Visual Studio Tools for Applications

Cuando el equipo en el que ejecuta Microsoft InfoPath y Visual Studio 2012 está conectado a Internet, el visor de Ayuda de Visual Studio 2012 tiene acceso a la documentación de .NET Framework y a la referencia de lenguaje de Visual Basic y Visual C# desde orígenes con conexión. Sin embargo, cuando el equipo en el que está trabajando no está conectado a Internet, no se tiene acceso a estas fuentes de documentación hasta que haya descargado e instalado una copia local de la documentación del SDK de.NET Framework 2.0 y llevado a cabo los pasos necesarios para integrar esa documentación en el entorno de desarrollo de Visual Studio 2012.
  
Para ver un tema de Ayuda que describa esta situación, abra el panel **Contenido**, expanda **Visual Studio para aplicaciones** y **Introducción al Entorno de desarrollo integrado** y, a continuación, haga clic en **Cómo: Integrar la documentación del SDK de .NET Framework 2.0 con el entorno de desarrollo integrado**.
  
> [!IMPORTANT]
> [!IMPORTANTE] Si trabaja en Windows Vista con Control de cuentas de usuario, debe iniciar InfoPath como administrador para realizar el procedimiento mencionado anteriormente. Para hacerlo, diríjase a C:\Archivos de programa\Microsoft Office\Office14, haga clic con el botón secundario del mouse en INFOPATH.EXE y a continuación haga clic en **Ejecutar como administrador**. Si no ejecuta InfoPath como administrador, no podrá realizar el procedimiento.

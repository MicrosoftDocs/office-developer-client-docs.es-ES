<span data-ttu-id="a997c-105">Cuando el equipo en el que ejecuta Microsoft InfoPath y Visual Studio 2012 está conectado a Internet, el visor de ayuda de Visual Studio 2012 puede obtener acceso a la documentación de .NET Framework y a las referencias de idioma de Visual Basic y Visual C# desde orígenes en línea.</span><span class="sxs-lookup"><span data-stu-id="a997c-105">When the computer on which you are running Microsoft InfoPath and Visual Studio 2012 is connected to the Internet, the Visual Studio 2012 help viewer can access .NET Framework documentation and the Visual Basic and Visual C# language references from online sources.</span></span> <span data-ttu-id="a997c-106">Sin embargo, cuando el equipo con el que está trabajando no está conectado a Internet, no podrá obtener acceso a estos orígenes de documentación hasta que descargue e instale una copia local de la documentación del SDK de .NET Framework 2.0 y realice los pasos necesarios para integrar esa documentación con el entorno de desarrollo de Visual Studio 2012.</span><span class="sxs-lookup"><span data-stu-id="a997c-106">However, when the computer you are working with is not connected to the Internet, you cannot access these sources of documentation until you download and install a local copy of .NET Framework 2.0 SDK documentation and perform the steps required to integrate that documentation with the Visual Studio 2012 development environment.</span></span>

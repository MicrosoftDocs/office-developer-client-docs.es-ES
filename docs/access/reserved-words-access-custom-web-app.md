# <a name="reserved-keywords-access-custom-web-app"></a>Palabras clave reservadas (aplicación web personalizado de Access)

Microsoft Access utiliza las palabras clave reservadas para definir, manipular y obtener acceso a las bases de datos. Palabras clave reservadas forman parte de la gramática del idioma que se usa Access para analizar y comprender las instrucciones SQL. 
  
> [!IMPORTANT]
> Microsoft ya no recomienda crear ni usar aplicaciones web de Access en SharePoint. Como alternativa, considere la posibilidad de usar [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para crear soluciones empresariales sin código para la Web y dispositivos móviles.

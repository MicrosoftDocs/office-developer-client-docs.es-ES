# <a name="replace-function-access-custom-web-app"></a>Reemplace la función (aplicación web personalizado de Access)

Reemplaza todas las apariciones de un valor de cadena especificado con otro valor de cadena.
  
> [!IMPORTANT]
> [!IMPORTANTE] Microsoft ya no recomienda crear ni usar aplicaciones web de Access en SharePoint. Como alternativa, considere la posibilidad de usar [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para crear soluciones empresariales sin código para la Web y dispositivos móviles. 
  
## <a name="syntax"></a>Syntax

 **Reemplazar** (*TextExpression*, de *trama*, *reemplazo*) 
  
La función **Replace** contiene los siguientes argumentos. 
  
|**Nombre del argumento**|**Descripción**|
|:-----|:-----|
| *TextExpression*  <br/> |La expresión de cadena que se desea buscar.  <br/> |
| *Pattern*  <br/> |Subcadena que se encuentra.  *Trama* no puede ser una cadena vacía ("").  <br/> |
| *Replacement*  <br/> |Cadena de reemplazo.  <br/> |

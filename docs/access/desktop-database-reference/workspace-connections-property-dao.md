# <a name="workspaceconnections-property-dao"></a>Propiedad Workspace. Connections (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve una colección **Connections** que representa las conexiones actuales del objeto **Workspace** especificado. Solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Entre

*expresión* Variable que representa un objeto **Workspace** .

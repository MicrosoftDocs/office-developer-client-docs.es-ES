# <a name="workspaceconnections-property-dao"></a>Workspace.Connections Property (DAO)


**Se aplica a**: Access 2013 | Office 2013

Devuelve una colección **Connections** que representa las conexiones actuales del objeto **Workspace** especificado. Es de solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Conexiones

*expresión* Variable que representa un objeto **Workspace** .

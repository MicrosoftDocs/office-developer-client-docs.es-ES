# <a name="recordset2name-property-dao"></a>Propiedad Recordset2.Name (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve el nombre del objeto especificado. Sólo lectura de la **cadena**.

## <a name="syntax"></a>Sintaxis

*expresión* . Denomina

*expresión* Variable que representa un objeto **Recordset2** .

## <a name="remarks"></a>Comentarios

La propiedad **Name** de un objeto **Recordset** abierto mediante una instrucción SQL es los primeros 256 caracteres de la instrucción SQL.

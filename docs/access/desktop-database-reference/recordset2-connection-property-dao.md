# <a name="recordset2connection-property-dao"></a>Propiedad Recordset2.Connection (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve el objeto **[Connection](connection-object-dao.md)** que corresponde a la base de datos.

## <a name="syntax"></a>Sintaxis

*expresión* . Conexión

*expresión* Variable que representa un objeto **Recordset2.**

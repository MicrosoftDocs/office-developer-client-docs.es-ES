# <a name="programming-with-ado-md"></a>Programación con ADO MD


**Se aplica a:** Access 2013, Office 2013

Para utilizar ADO MD con su herramienta de desarrollo, debe establecer una referencia a la biblioteca de tipos ADO MD. La descripción de la biblioteca ADO MD es Biblioteca de Microsoft ActiveX Data Objects (MultiDimensional). El nombre de archivo de biblioteca ADO MD es msadomd.dll y el identificador de programa (ProgID) es "ADOMD". Para obtener más información acerca del establecimiento de referencias a bibliotecas, vea la documentación de su herramienta de desarrollo.

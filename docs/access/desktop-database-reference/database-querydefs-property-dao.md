# <a name="databasequerydefs-property-dao"></a>Propiedad Database.QueryDefs (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve una colección **QueryDefs** que contiene todos los objetos **QueryDef** de la base de datos especificada. Sólo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . QueryDefs

*expression* Variable que representa un objeto **Database**.

# <a name="caption-property-ado-md"></a>Caption (propiedad, ADO MD)


**Se aplica a**: Access 2013, Office 2013

Indica el texto de título que se utilizará para mostrar un objeto [Level](level-object-ado-md.md) o [Member](member-object-ado-md.md).

## <a name="return-values"></a>Valores devueltos

Devuelve un valor **String** y es de sólo lectura.

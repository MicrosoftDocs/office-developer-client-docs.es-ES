## <a name="saving-to-the-xml-dom-object"></a>Guardar en el objeto XML DOM

Un **conjunto de registros** se puede guardar en formato XML en una instancia de un objeto DOM de MSXML, como se muestra en el c√≥digo siguiente de Visual Basic:

```vb

Dim rsXML As New ADODB.Recordset

        "\Common Files\System\msadc\samples\NWind.mdb"
```

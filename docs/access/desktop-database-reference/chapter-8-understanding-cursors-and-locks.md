# <a name="chapter-8-understanding-cursors-and-locks"></a>Capítulo 8: Descripción de cursores y bloqueos


**Se aplica a**: Access 2013 | Office 2013

Es importante comprender cómo funcionan los cursores, para poder seleccionar el tipo de cursor mejor y más eficaz para los requisitos de acceso a datos de una aplicación. Una configuración de cursor que no sea la óptima puede provocar que las operaciones de acceso a datos sean extremadamente lentas.

Numerosas funcionalidades del objeto **Recordset** de ADO están determinadas por el tipo y la ubicación del cursor, así como el tipo de bloqueo.

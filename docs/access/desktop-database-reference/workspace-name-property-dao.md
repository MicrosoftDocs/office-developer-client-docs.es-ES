# <a name="workspacename-property-dao"></a>Workspace.Name Property (DAO)


**Se aplica a**: Access 2013 | Office 2013

Devuelve o establece el nombre del objeto especificado. **String** de lectura y escritura si el objeto no se anexó a una colección. **String** de solo lectura si el objeto se anexó a una colección.

## <a name="syntax"></a>Sintaxis

*expresión* . Nombre

*expresión* Variable que representa un objeto **Workspace** .

## <a name="remarks"></a>Observaciones

La longitud máxima del nombre de un objeto **Workspace** es de 20 caracteres.

# <a name="exitforeachrecord-macro-action"></a>SalirDeCadaRegistro (acción de macro)


**Se aplica a:** Access 2013, Office 2013

Utilice la acción **SalirDeCadaRegistro** para salir inmediatamente de un bloque de datos **[ParaCadaRegistro](foreachrecord-data-block.md)**.


> [!NOTE]
> [!NOTA] La acción **ParaCadaRegistro** solo está disponible en macros de datos.

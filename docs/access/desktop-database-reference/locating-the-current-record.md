# <a name="locating-the-current-record"></a>Localización del registro actual

**Se aplica a**: Access 2013, Office 2013

La posición actual del cursor en el **conjunto de registros** delinea la posición del registro activo. Suponiendo que el comando emitido devuelve resultados, el cursor se coloca automáticamente en el primer registro cuando se llama al método **Open** del **conjunto de registros**. Así, con el **conjunto de registros** de ejemplo, el cursor estaría en el primer registro: "Uncle Bob's Organic Dried Pears".

# <a name="ado-error-codes"></a>Códigos de error ADO

**Se aplica a**: Access 2013, Office 2013

Además de los errores de proveedor devueltos en los objetos [Error](error-object-ado.md) de la colección [Errors](errors-collection-ado.md), el propio ADO puede devolver errores al mecanismo de control de excepciones del entorno en tiempo de ejecución. Use el mecanismo de intercepción de errores del lenguaje de programación, como la instrucción **On Error** de Microsoft Visual Basic o el bloqueo **try-catch** de Microsoft Visual C++ o Microsoft Visual J++ para capturar errores de ADO.

Para obtener la lista de los códigos de error ADO, vea [ErrorValueEnum](errorvalueenum.md).

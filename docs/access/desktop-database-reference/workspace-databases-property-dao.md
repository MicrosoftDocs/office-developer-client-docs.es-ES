# <a name="workspacedatabases-property-dao"></a>Workspace.Databases Property (DAO)


**Se aplica a**: Access 2013 | Office 2013

Devuelve una colección **Databases** que representa las bases de datos abiertas del objeto **Workspace** especificado. Es de solo lectura

## <a name="syntax"></a>Sintaxis

*expresión* . Bases de datos

*expresión* Variable que representa un objeto **Workspace** .

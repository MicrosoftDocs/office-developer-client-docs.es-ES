# <a name="navigating-through-the-data"></a>Desplazarse por los datos


**Se aplica a**: Access 2013 | Office 2013

Ahora que ha ejecutado un comando en el origen de datos y determinado que el conjunto de resultados contiene datos, puede moverse por los resultados mediante los métodos de desplazamiento y las propiedades disponibles con el objeto **Recordset**. En los siguientes temas, se describe cómo usar estos métodos y propiedades con el **conjunto de registros** de ejemplo:

  - [Saltar a un registro](jumping-to-a-record.md)

  - [Más formas de moverse en un conjunto de registros](more-ways-to-move-in-a-recordset.md)

  - [Utilizar marcadores](using-bookmarks.md)

  - [Utilizar páginas](using-pages.md)

  - [Colocación de un conjunto de registros](recordset-positioning.md)

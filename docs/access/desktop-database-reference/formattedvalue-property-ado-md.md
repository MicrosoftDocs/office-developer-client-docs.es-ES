# <a name="formattedvalue-property-ado-md"></a>Propiedad FormattedValue (ADO MD)


**Se aplica a:** Access 2013, Office 2013

Indica la presentación con formato de un valor de celda.

## <a name="return-values"></a>Valores devueltos

Devuelve un valor **String** y es de sólo lectura.

## <a name="remarks"></a>Comentarios

Utilice la propiedad **FormattedValue** para obtener el valor de presentación con formato de la propiedad [Valor](value-property-ado-md.md) de un objeto [Cell](cell-object-ado-md.md). Por ejemplo, si el valor de una celda era 1056,87 y este valor representaba una cantidad en dólares, **FormattedValue** será 1.056,87 $.

# <a name="tabledefname-property-dao"></a>TableDef.Name Property (DAO)


**Se aplica a**: Access 2013 | Office 2013

Devuelve o establece el nombre del objeto especificado. **String** de lectura y escritura.

## <a name="syntax"></a>Sintaxis

*expresión* . Nombre

*expresión* Variable que representa un objeto **TableDef** .

## <a name="remarks"></a>Observaciones

La longitud máxima del nombre de un objeto **TableDef** es de 64 caracteres.

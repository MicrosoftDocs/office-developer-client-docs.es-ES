# <a name="position-object-ado-md"></a>Position (objeto, ADO MD)


**Se aplica a**: Access 2013, Office 2013

Representa un conjunto de uno o varios miembros de dimensiones diferentes, que define un punto a lo largo de un eje.

## <a name="remarks"></a>Comentarios

Con las propiedades y las colecciones de un objeto **Position**, puede hacer lo siguiente:

  - Utilizar la propiedad **Ordinal** para devolver la posición ordinal del objeto **Position** a lo largo del [eje](axis-object-ado-md.md).

  - Utilizar la colección [Members](members-collection-ado-md.md) para devolver los miembros que conforman la posición a lo largo del **eje**.

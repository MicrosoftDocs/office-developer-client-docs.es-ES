# <a name="leveldepth-property-ado-md"></a>LevelDepth (propiedad, ADO MD)


**Se aplica a**: Access 2013 | Office 2013

Indica el número de niveles entre la raíz de la jerarquía y un componente.

## <a name="return-values"></a>Valores devueltos

Devuelve un entero **Long** y es de sólo lectura.

## <a name="remarks"></a>Comentarios

Utilice la propiedad **LevelDepth** para determinar la distancia del objeto [Member](member-object-ado-md.md) desde el nivel raíz de la jerarquía. El valor **LevelDepth** de un elemento en el nivel raíz es 0. Corresponde a la propiedad [Depth](depth-property-ado-md.md) de un objeto [Level](level-object-ado-md.md).

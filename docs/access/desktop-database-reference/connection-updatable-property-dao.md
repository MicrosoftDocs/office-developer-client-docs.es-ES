# <a name="connectionupdatable-property-dao"></a>Connection.Updatable Property (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve un valor que indica si puede cambiar un objeto DAO. **Boolean** de solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Actualizable

*expresión* Expresión que devuelve un objeto **Connection** .

# <a name="dbengineworkspaces-property-dao"></a>DBEngine.Workspaces Property (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve una colección **Workspaces** que contiene todos los objetos **Workspace** activos que no están ocultos. Es de solo lectura

## <a name="syntax"></a>Sintaxis

*expresión* . Áreas de trabajo

*expresión* Variable que representa un objeto **DBEngine** .

# <a name="dbengineworkspaces-property-dao"></a>Propiedad DBEngine. Workspaces (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve una colección **Workspaces** que contiene todos los objetos **Workspace** activos que no están ocultos. Solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Áreas

*expresión* Variable que representa un objeto **DBEngine** .

# <a name="type-property-ado-md"></a>Tipo (propiedad, ADO MD)


**Se aplica a**: Access 2013, Office 2013

Indica el tipo de elemento activo.

## <a name="return-values"></a>Valores devueltos

Devuelve un valor [MemberTypeEnum](membertypeenum.md) y es de sólo lectura.

## <a name="remarks"></a>Comentarios

Esta propiedad sólo es compatible con objetos [Member](member-object-ado-md.md) pertenecientes a un objeto [Level](level-object-ado-md.md). Si se hace referencia a esta propiedad desde objetos **Member** pertenecientes a un objeto [Position](position-object-ado-md.md), se produce un error.

# <a name="connectionquerydefs-property-dao"></a>Propiedad Connection.QueryDefs (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve una colección **QueryDefs** que contiene todos los objetos **QueryDef** de la conexión especificada. Sólo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Definiciones de consulta

*expresión* Variable que representa un objeto **Connection** .

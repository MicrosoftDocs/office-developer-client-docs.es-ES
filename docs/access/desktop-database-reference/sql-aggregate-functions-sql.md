# <a name="sql-aggregate-functions-sql"></a>Funciones de agregado de SQL (SQL)


**Se aplica a**: Access 2013, Office 2013

Mediante las funciones de agregado de SQL, puede determinar diversas estadísticas basadas en conjuntos de valores. Puede usar estas funciones en una consulta y en expresiones de agregado en la propiedad **SQL** de un objeto **QueryDef** o al crear un objeto **Recordset** basado en una consulta SQL.

- [Función AVG](https://msdn.microsoft.com/library/ff822755\(v=office.15\))

- [Función Count](https://msdn.microsoft.com/library/ff844748\(v=office.15\))

- [Funciones First y Last](https://msdn.microsoft.com/library/ff197381\(v=office.15\))

- [Funciones Min y Max](https://msdn.microsoft.com/library/ff194490\(v=office.15\))

- [Funciones StDev y StDevP](https://msdn.microsoft.com/library/ff197043\(v=office.15\))

- [Función Sum](https://msdn.microsoft.com/library/ff844764\(v=office.15\))

- [Funciones Var y VarP](https://msdn.microsoft.com/library/ff192105\(v=office.15\))

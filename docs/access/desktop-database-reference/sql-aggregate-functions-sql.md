# <a name="sql-aggregate-functions-sql"></a>Funciones de agregado de SQL (SQL)

**Se aplica a**: Access 2013, Office 2013

Mediante las funciones de agregado de SQL, puede determinar diversas estadísticas basadas en conjuntos de valores. Puede usar estas funciones en una consulta y en expresiones de agregado en la propiedad **SQL** de un objeto **QueryDef** o al crear un objeto **Recordset** basado en una consulta SQL.

- [Función Avg](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/avg-function-microsoft-access-sql)
- [Función Count](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/count-function-microsoft-access-sql)
- [Funciones First y Last](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/first-last-functions-microsoft-access-sql)
- [Funciones Min y Max](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/min-max-functions-microsoft-access-sql)
- [Funciones StDev y StDevP](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/stdev-stdevp-functions-microsoft-access-sql)
- [Función Sum](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/sum-function-microsoft-access-sql)
- [Funciones Var y VarP](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/var-varp-functions-microsoft-access-sql)

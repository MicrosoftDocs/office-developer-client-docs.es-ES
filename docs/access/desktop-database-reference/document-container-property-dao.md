# <a name="documentcontainer-property-dao"></a>Propiedad Document. Container (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve el nombre del objeto **[Container](container-object-dao.md)** al que pertenece un objeto **Document** (s칩lo 치reas de trabajo de Microsoft Access). .

## <a name="syntax"></a>Sintaxis

*expresi칩n* . Recipiente

## <a name="example"></a>Ejemplo

En este ejemplo, se muestra la propiedad **Container** para diferentes objetos **Document**.

```vb 
Sub ContainerPropertyX() 
 
 Dim dbsNorthwind As Database 
 Dim ctrLoop As Container 
 
 Set dbsNorthwind = OpenDatabase("Northwind.mdb") 
 
 ' Display the container name for the first Document 
 ' object in each Container object's Documents collection. 
 For Each ctrLoop In dbsNorthwind.Containers 
 Debug.Print "Document: " & ctrLoop.Documents(0).Name 
 Debug.Print " Container = " & _ 
 ctrLoop.Documents(0).Container 
 Next ctrLoop 
 
 dbsNorthwind.Close 
 
End Sub 
 
```

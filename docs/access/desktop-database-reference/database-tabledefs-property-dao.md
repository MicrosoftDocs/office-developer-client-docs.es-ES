# <a name="databasetabledefs-property-dao"></a>Propiedad Database.TableDefs (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve una colección **TableDefs** que contiene todos los objetos **TableDef** almacenados en la base de datos especificada. Sólo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . TableDefs

*expression* Variable que representa un objeto **Database**.

# <a name="activeconnection-property-adox"></a>ActiveConnection (propiedad, ADOX)


**Se aplica a**: Access 2013 | Office 2013

Indica el objeto [Connection](connection-object-ado.md) de ADO al que pertenece el [catálogo](catalog-object-adox.md).

## <a name="settings-and-return-values"></a>Configuraciones y valores devueltos

Establece un objeto **Connection** o una **cadena** que contiene la definición correspondiente a una conexión. Devuelve el objeto **Connection** activo.

## <a name="remarks"></a>Comentarios

El valor predeterminado es una referencia a objeto nulo.

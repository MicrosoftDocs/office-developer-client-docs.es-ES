# <a name="clustered-property-adox"></a>Clustered (propiedad, ADOX)


**Se aplica a**: Access 2013 | Office 2013

Indica si el índice es agrupado.

## <a name="settings-and-return-values"></a>Configuraciones y valores devueltos

Establece y devuelve un valor **Boolean**.

## <a name="remarks"></a>Comentarios

El valor predeterminado es **False**.

Esta propiedad es de sólo lectura en objetos [Index](index-object-adox.md) ya anexados a una colección.

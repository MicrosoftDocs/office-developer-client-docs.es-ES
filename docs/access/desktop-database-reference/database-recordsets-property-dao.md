# <a name="databaserecordsets-property-dao"></a>Propiedad Database.Recordsets (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve una colección **Recordsets** que contiene todos los conjuntos de registros abiertos en la base de datos especificada. Es de solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Conjuntos de registros

*expresión* Variable que representa un objeto de **base de datos** .

# <a name="comment-macro-statement"></a>Comentario (instrucción de macro)


**Se aplica a**: Access 2013, Office 2013

La instrucción **Comentario** puede utilizarse para proporcionar comentarios explicativos en una macro.

## <a name="remarks"></a>Comentarios

Un **Comentario** puede tener hasta 1000 caracteres de longitud. Sin embargo, de forma predeterminada, se muestra solo la primera línea del comentario.

# <a name="connectiontransactions-property-dao"></a>Propiedad Connection.Transactions (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve un valor que indica si un objeto admite transacciones. **Boolean** de solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Transacciones

*expresión* Variable que representa un objeto **Connection** .

# <a name="depth-property-ado-md"></a>Depth (propiedad, ADO MD)


**Se aplica a**: Access 2013, Office 2013

Indica el número de niveles entre el [nivel](level-object-ado-md.md) y la raíz de la jerarquía.

## <a name="return-values"></a>Valores devueltos

Devuelve un entero y es de sólo lectura.

## <a name="remarks"></a>Comentarios

Un **nivel** en la raíz de una jerarquía tiene un valor **Depth** de cero (0).

# <a name="propertyname-property-dao"></a>Propiedad Property.Name (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve o establece el nombre del objeto especificado. **String** de lectura y escritura si el objeto no se anexó a una colección. **String** de solo lectura si el objeto se anexó a una colección.

## <a name="syntax"></a>Sintaxis

*expresión* . Nombre

*expresión* Variable que representa un objeto **Property** .

## <a name="remarks"></a>Observaciones

La propiedad **Name** de una propiedad integrada es siempre de sólo lectura.

La longitud máxima para el nombre de un objeto **Property** es de 64 caracteres.

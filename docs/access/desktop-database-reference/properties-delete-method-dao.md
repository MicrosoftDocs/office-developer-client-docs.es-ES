# <a name="propertiesdelete-method-dao"></a>Properties.Delete (método) (DAO)


**Se aplica a**: Access 2013, Office 2013

Elimina el objeto **Property** especificado de la colección **Properties**.

## <a name="syntax"></a>Sintaxis

*expresión* . Delete (***nombre***)

*expresión* Variable que representa un objeto **Properties** .

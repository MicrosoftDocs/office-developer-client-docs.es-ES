# <a name="querydefname-property-dao"></a>Propiedad QueryDef.Name (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve o establece el nombre del objeto especificado. **String** de lectura y escritura.

## <a name="syntax"></a>Sintaxis

*expresión* . Nombre

*expresión* Variable que representa un objeto **QueryDef** .

## <a name="remarks"></a>Observaciones

La longitud máxima para el nombre de un objeto **QueryDef** es de 64 caracteres.

# <a name="recordset2parentrecordset-property-dao"></a>Recordset2.ParentRecordset Property (DAO)


**Se aplica a**: Access 2013 | Office 2013 

Devuelve el objeto **Recordset** principal del conjunto de registros especificado. Sólo lectura.

## <a name="version-information"></a>Información de versión

Versión añadida: Access 2007

## <a name="syntax"></a>Sintaxis

*expresión* . ParentRecordset

*expresión* Variable que representa un objeto **Recordset2** .

## <a name="remarks"></a>Observaciones

La propiedad **ParentRecordset** devuelve **Null** si el conjunto de registros especificado no representa un campo multivalor.

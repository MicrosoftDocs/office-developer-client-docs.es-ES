# <a name="dbengineerrors-property-dao"></a>Propiedad DBEngine.Errors (DAO)


**Se aplica a:** Access 2013, Office 2013

Devuelve una colección **Errors** que contiene todos los objetos **Error** almacenados para el objeto especificado. Es de solo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Errores

*expression* Variable que representa un objeto **DBEngine**.

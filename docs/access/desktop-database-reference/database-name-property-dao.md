# <a name="databasename-property-dao"></a>Propiedad Database.Name (DAO)


**Se aplica a**: Access 2013, Office 2013

Devuelve el nombre del objeto especificado. **String** de sólo lectura.

## <a name="syntax"></a>Sintaxis

*expresión* . Nombre

*expresión* Variable que representa un objeto de **base de datos** .

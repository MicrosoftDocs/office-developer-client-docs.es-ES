# <a name="replicate-function-access-custom-web-app"></a>Función replicar (aplicación web personalizada de Access)

RePite un valor de texto un número especificado de veces.
  
> [!IMPORTANT]
> Microsoft ya no recomienda crear ni usar aplicaciones web de Access en SharePoint. Como alternativa, considere la posibilidad de usar [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para crear soluciones empresariales sin código para la Web y dispositivos móviles. 
  
## <a name="syntax"></a>Sintaxis

 **Replicar** (*TextExpression*, *NumberOfTimes*) 
  
La **** función replicate contiene los siguientes argumentos. 
  
|**Nombre de argumento**|**Descripción**|
|:-----|:-----|
| *TextExpression*  <br/> |Una expresión de texto.  <br/> |
| *NumberOfTimes*  <br/> |Valor de tipo Integer que especifica el número de veces que se va a repetir la *TextExpression* .  <br/> |

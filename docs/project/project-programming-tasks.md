# <a name="project-programming-tasks"></a>Tareas de programación de Project

En esta sección se incluyen artículos con procedimientos que muestran cómo usar la biblioteca de JavaScript para el modelo de objetos del lado cliente (CSOM) y realizar otras tareas de programación para Project Server 2013 y Project online. Algunos ejemplos de tareas de programación son la creación de una aplicación de Project Server hospedada en SharePoint, la creación de flujos de trabajo para la administración de propuestas; programación de aplicaciones de Project Server con Windows Communication Foundation (WCF); Personalización de la cinta de Project Web App; creación de elementos Web de Project Server; crear controladores de eventos de Project Server y receptores de eventos remotos; y la actualización masiva de campos personalizados y la creación de sitios de proyecto para Project online.
  
> [!NOTE]
> Para obtener información sobre la programación con el control de cuadrícula de JS, vea [JS Grid control](https://msdn.microsoft.com/library/ee535898%28office.14%29.aspx) en la referencia del programador de SharePoint 2010. Para la referencia del código administrado, vea el [espacio de nombres Microsoft. SharePoint. JSGrid](https://msdn.microsoft.com/library/microsoft.sharepoint.jsgrid%28Office.15%29.aspx). Para los controles Web de JS Grid control, vea [JSGrid Class](https://msdn.microsoft.com/library/microsoft.sharepoint.webcontrols.jsgrid%28Office.15%29.aspx).

# <a name="project-programming-tasks"></a>Tareas de programación de Project

En esta sección se incluyen algunos artículos de "how-to" que muestran cómo usar la biblioteca JavaScript para el modelo de objetos del lado cliente (CSOM) y realizar otras tareas de programación para Project Server 2013 y Project Online. Algunos ejemplos de tareas de programación incluyen la creación SharePoint una aplicación de servidor Project hospedada por el usuario, la creación de flujos de trabajo para la administración de la demanda; programación Project server con Windows Communication Foundation (WCF); personalizar la cinta Project Web App; crear Project server elementos web; crear Project de eventos server y receptores de eventos remotos; y actualización masiva de campos personalizados y creación de sitios de proyecto para Project Online.

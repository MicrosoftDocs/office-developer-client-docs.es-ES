# <a name="project-2013-programming-references"></a>Referencias de programación de Project 2013

Encuentre información general sobre la referencia Project Server Interface (PSI), cómo usar la interfaz ASMX y la interfaz WCF de la PSI, información sobre los códigos de error de Project Server y una referencia para el servicio **ProjectData** OData. 
  
> [!NOTE]
> Para obtener las referencias principales para PSI y el modelo de objetos del lado cliente (CSOM) en Project Server 2013, vea las siguientes secciones: Referencia de biblioteca de clases y servicio [web](https://msdn.microsoft.com/library/ef1830e0-3c9a-4f98-aa0a-5556c298e7d1%28Office.15%29.aspx) y referencia de api de [JavaScript para Project Server 2013](javascript-library-and-rest-reference-for-project-server-2013.md). > Para las referencias Office complementos de Office, que incluyen la referencia de JavaScript para aplicaciones de panel de tareas de Project 2013, vea API de [JavaScript](https://msdn.microsoft.com/library/fp142185.aspx)para Office . > Para obtener la referencia de VBA para Project Standard 2013 y Project Profesional 2013, vea Project referencia para desarrolladores vba de [2013](https://msdn.microsoft.com/library/jj235035.aspx). > La referencia de las tablas y vistas de informes en la base de datos Project Server 2013 aún no está disponible.
